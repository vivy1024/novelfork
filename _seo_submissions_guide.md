## 1. AlternativeTo

**注册地址：** https://alternativeto.net/ （右上角 Sign In → Sign up / Continue with GitHub）

**登录后操作：**
1. 点右上角菜单 → "Add application" 或访问 https://alternativeto.net/contribute/
2. 填写以下信息：

**Name:** InkOS

**Website URL:** https://github.com/Narcooo/inkos

**Description:**
```
InkOS is an open-source CLI AI agent for autonomous novel writing. It runs a multi-agent pipeline that handles planning, drafting, auditing, and revision automatically — keeping plot, characters, and world state consistent across the entire book.

Unlike chatbot-based writing tools, InkOS maintains structured state files that track every character, resource, plot hook, and emotional arc. The built-in auditor catches continuity errors, AI-tell patterns, and pacing issues before they compound.

Supports LitRPG, Progression Fantasy, Isekai, Romantasy, Sci-Fi, and more. Style cloning lets you match any author's voice. Multi-model routing means you pick which LLM handles which task. MIT licensed, no SaaS, no subscriptions.
```

**Alternative to:** NovelAI, Sudowrite, ChatGPT (for novel writing)

**Tags/Categories:** AI Writing, Novel Writing, Creative Writing, CLI Tools, Open Source, AI Agent

**License:** Open Source (MIT)

**Platform:** Linux, macOS, Windows (Node.js CLI)

## 2. SourceForge

**注册地址：** https://sourceforge.net/auth/ （注册或用 GitHub 登录）

**登录后操作：**
1. 访问 https://sourceforge.net/software/product/new/
2. 或在顶部菜单找 "Create" → "New Project"

**Project Name:** InkOS

**Project URL / Source Code:** https://github.com/Narcooo/inkos

**Short Description:**
```
Open-source CLI AI agent for autonomous novel writing — plans, drafts, audits, and revises full-length fiction with consistent plot and character continuity.
```

**Full Description:**
```
InkOS is an open-source CLI AI agent that autonomously writes, audits, and revises novels. A multi-agent pipeline handles planning, composition, writing, observation, reflection, and revision — maintaining structured state files for world state, character matrix, resources, plot hooks, emotional arcs, subplots, and chapter summaries.

Key capabilities:
- Autonomous end-to-end novel writing with human review gates
- Continuity tracking across unlimited chapters via structured JSON state
- De-AI-ification: fatigue word detection, style fingerprint injection, AI-tell removal
- Style cloning from reference text
- 10 English genre profiles: LitRPG, Progression Fantasy, Isekai, Cultivation, System Apocalypse, Dungeon Core, Romantasy, Sci-Fi, Tower Climber, Cozy Fantasy
- Multi-model routing (different LLMs for different pipeline stages)
- Fan fiction, continuation, and spinoff workflows
- Daemon mode with Telegram/webhook notifications
- EPUB export

Install: npm i -g @actalk/inkos
License: MIT
```

**Categories:** AI & Machine Learning, Text Processing, Creative Writing

**License:** MIT License

**Programming Language:** TypeScript

**Operating System:** OS Independent (Node.js)
