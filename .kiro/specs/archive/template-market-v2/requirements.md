# 模板市场 v2

## 概述

从 InkOS 遗留的 6 个固定 bundle 升级为完整的模板市场：26 个流派全部预制 + 用户自建模板 + 远程市场下载。

## 用户故事

1. 作为作者，我希望从 26 个中文网文流派中选择适合的模板一键应用
2. 作为作者，我希望基于现有模板修改或从零创建自己的 bundle
3. 作为作者，我希望从远程市场浏览和下载社区分享的模板
4. 作为作者，我希望将自己的模板分享到远程市场

- 推荐文风 tone
- 推荐节拍模板
- 推荐经纬分区（创建书籍时自动应用）
- 示例开头（可选）

### FR-2: 用户自建模板

- 用户可以从零创建 bundle（填写流派 prompt + 选择文风 + 配置节拍）
- 用户可以基于现有 bundle 复制并修改
- 用户自建模板存储在 SQLite（`user_template` 表）
- 用户模板与内置模板在 TemplateMarketPanel 中统一展示，标记来源

### FR-3: 远程市场

- 远程市场 API：`GET /api/market/templates`（列表）、`GET /api/market/templates/:id`（详情）
- 数据源：GitHub Gist 或自建 JSON 文件托管（v0.1.0 用 GitHub raw 文件即可）
- 下载后存为用户模板（同 FR-2 的存储）
- 上传/分享：将用户模板导出为 JSON，手动提交到 GitHub（v0.1.0 不做自动上传）

### FR-4: 模板应用增强

- 应用模板时自动：
  - 设置书籍 genre
  - 应用经纬分区模板
  - 设置默认文风 tone
  - 设置默认节拍模板
- 应用前预览（显示将要改变什么）
- 支持部分应用（只应用流派 prompt，不改经纬）

## 非功能需求

- 26 个内置 bundle 的总大小 < 500KB
- 远程市场请求超时 5s，失败不影响本地功能
- 用户模板数量无上限

## 验收标准

1. TemplateMarketPanel 展示 26 个内置 bundle + 用户自建模板
2. 点击"应用"后书籍配置正确更新
3. 可以创建/编辑/删除用户自建模板
4. 远程市场可以浏览和下载模板（网络不可用时优雅降级）
