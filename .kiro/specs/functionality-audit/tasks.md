## 一、对话界面问题

### 1.1 顶部工具栏按钮空壳

| 按钮 | 当前实现 | 应该做什么 |
|------|---------|-----------|
| 编辑标题 ✏️ | `prompt()` 弹窗 | inline 编辑（点击标题变为 Input） |
| 生成标题 ✨ | 写死日期字符串 | 调用 LLM 根据对话内容生成标题 |
| 文件修改 📄 | Sheet 空状态 | 显示当前会话中 Agent 修改的文件列表 |
| 会话信息 ℹ️ | 只显示基础字段 | 显示 compact 历史、token 用量、recovery 状态 |
| 归档 📦 | 只改 status | 应有确认对话框 + 归档后导航回会话列表 |

### 1.2 Slash Command 真实性验证

需要验证以下命令是否真的调用后端：
- `/compact` → 是否触发 `POST /api/sessions/:id/compact`
- `/model` → 是否打开模型选择器
- `/permission` → 是否切换权限模式
- `/fork` → 是否创建新会话
- `/help` → 是否显示命令列表

### 1.3 套路配置→对话传递

需要验证：
- 套路页的工具开关是否影响 Agent 可用工具
- 套路页的权限规则是否影响确认门行为
- 套路页的 MCP 工具是否在对话中可用
- 套路页的技能/提示词是否注入 system prompt

### 1.4 对话中模型切换

需要验证：
- 切换模型后下一条消息是否用新模型
- 切换推理强度是否生效
- 切换权限模式是否立即生效
- Fast Mode 开关是否有实际效果

## 二、叙事线/写作工作台问题

### 2.1 资源树问题

| 问题 | 说明 |
|------|------|
| Story/Truth 文件名英文 | `pending_hooks.md` 应显示"待处理伏笔" |
| Story 和 Truth 分组重复 | 应合并或重新分类 |
| Badge 视觉噪音 | "只读/可删除/可编辑" 应隐藏，hover 时显示 |
| 资源树太宽 | 应缩窄到 200-240px |
| JSON 文件直接展示 | 经纬资料应有结构化视图 |

### 2.2 章节功能缺失

| 功能 | spec 要求 | 当前状态 |
|------|----------|---------|
| 新建章节 | 创建记录 + 文件 + 自动打开编辑器 | ❌ 无 |
| 章节编辑 | 可编辑正文 + 保存状态 | ⚠️ 有 Canvas 但未验证 |
| 章节保存 | 加载中/未修改/未保存/保存中/已保存/失败 | ⚠️ 未验证 |
| 导入章节 | 文本导入 + 进入资源树 | ❌ 无 |

### 2.3 AI 候选稿管理缺失

| 功能 | spec 要求 | 当前状态 |
|------|----------|---------|
| 候选稿列表 | 显示所有 AI 生成的候选 | ❌ 无前端 |
| 合并到正式章节 | 确认步骤 + 追加/合并 | ❌ 无 |
| 替换正式章节 | 确认步骤 + 影响说明 | ❌ 无 |
| 另存草稿 | 保存到草稿区 | ❌ 无 |
| 放弃候选 | 标记 rejected/archived | ❌ 无 |

### 2.4 写作模式缺失

| 模式 | spec 要求 | 当前状态 |
|------|----------|---------|
| 续写 | 调用 LLM 生成续写正文 | ❌ 只有 prompt-preview |
| 扩写 | 基于场景生成扩写 | ❌ 只有 prompt-preview |
| 对话生成 | 基于角色生成对话 | ❌ 只有 prompt-preview |
| 多版本 | 生成 2-5 个版本对比 | ❌ 只有 prompt-preview |
| 大纲分支 | 生成 2-3 条走向建议 | ❌ 只有 prompt-preview |

### 2.5 经纬资料缺失

| 功能 | spec 要求 | 当前状态 |
|------|----------|---------|
| 人物/地点/势力/物品 CRUD | 新建/查看/编辑/删除 | ⚠️ 有后端 API，前端未验证 |
| 伏笔管理 | 与 pending_hooks.md 关联 | ❌ 无 |
| 世界规则 | 查看/编辑 | ❌ 无 |

### 2.6 驾驶舱缺失

| 功能 | spec 要求 | 当前状态 |
|------|----------|---------|
| 总览 Tab | 今日字数/焦点/摘要/风险 | ❌ 无 |
