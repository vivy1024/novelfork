# InkOS 路线图 | Roadmap

## 当前版本：v1.2.0

InkOS 已完成 Phase 4（MCP 集成 + Plugin 系统 + Pipeline 可视化），正在向 Phase 5 演进。

---

## Phase 5：交互叙事与局部干预（规划中）

### 5.1 互动小说引擎

**目标：** 支持分支叙事和读者选择。

- [ ] 分支点标记系统（`branch_points.md`）
- [ ] 选择树可视化（Studio 集成）
- [ ] 多结局管理
- [ ] 条件触发逻辑（基于角色状态/资源/伏笔）
- [ ] 回溯与存档系统

**使用场景：**
```bash
inkos branch create 吞天魔帝 15 --choices "接受师父传承,拒绝并离开"
inkos branch preview 吞天魔帝 15  # 预览分支树
```

### 5.2 局部干预与级联更新

**目标：** 允许重写章节的部分内容，自动更新后续真相文件。

- [ ] 段落级编辑 API
- [ ] 差异检测与影响分析
- [ ] 级联更新引擎（自动修正后续章节中的矛盾）
- [ ] 干预历史追踪

**使用场景：**
```bash
inkos edit 吞天魔帝 10 --range "1500-2000" --instruction "改为主角拒绝了邀请"
inkos cascade analyze 吞天魔帝 10  # 分析影响范围
inkos cascade apply 吞天魔帝 10    # 应用级联更新
```

### 5.3 自定义 Agent 插件系统（扩展）

**目标：** 允许用户编写自定义 Agent 和 Hook。

- [ ] Agent SDK（TypeScript API）
- [ ] Hook 生命周期完整文档
- [ ] 插件市场（社区贡献）
- [ ] 插件沙箱与权限管理

**示例插件：**
- 自动翻译插件（中英双语输出）
- 敏感词过滤插件
- 自定义审计维度插件
- 平台格式导出插件（起点、番茄、晋江）
