## 数据与脚本

### `data/processed/`

| 文件族 | 作用 |
|--------|------|
| `rruff_*` | 矿物/灵材筛查结果 |
| `tcm_herb_screening*` | 基础 herb 判定结果 |
| `tcmid_*` | TCMID herb / formula / component 主骨架 |
| `tcmbank_*` | TCMBank herb / ingredient / target / disease 富化结果 |
| `tcm_herb_screening_enriched*` | herb 与 TCMBank 合并后的可审计中间层，现已写回 ranked `Comp_i` 字段 |
| `tcm_herb_component_ranking_summary.json` | 成分降噪与 Top-N 主成分结果摘要 |
| `tcm_herb_rulec_final*` | 接入 ranked `Comp_i` 后的最终 Rule-C herb 结果 |
| `tcm_formula_rulec_final*` | 基于 TCMID 组件与 herb final 的方剂聚合原型结果 |
| `tcm_herb_rulec_expanded*` | 以方剂缺口为目标回填后的扩展 herb Rule-C 结果 |
| `tcm_formula_rulec_expanded*` | 使用 expanded herb 库重跑后的方剂聚合结果 |
| `tcm_herb_rulec_library*` | 从 TCMID 主索引批量下沉后的 `1018` 条 herb / materia medica 大库结果 |
| `tcm_herb_asset_classified*` | herb 大库的首轮分类、物种认证和产地信号结果 |
| `tcm_materia_medica_master*` | 合并 library / classified / TCMID / TCMBank 后的本草主库结果 |
| `tcm_materia_medica_master_*.(csv/sqlite)` | 按 plant / animal / mineral / processed 拆出的四张资产子表 |
| `tcm_materia_medica_world_enriched*` | 在本草主库基础上补齐玩法层 `产地` 和 `ingredient` 的世界化补全结果 |

| 脚本 | 作用 |
|------|------|
| `build_rruff_mineral_db.py` | 矿物灵材数据库构建 |
