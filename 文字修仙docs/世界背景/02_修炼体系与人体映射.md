# 修炼体系与人体映射

> **一句话版**：修炼不是"吸一口天地灵气就升级"，而是人体在特定材料、环境和操作规程辅助下，逐步建立对真空耦合能的感知、耐受、储存与调控能力。

## 1. 修炼的科学定义

在本世界中，修炼对应的是一套可重复、可记录、但高度依赖个体差异的人体改造流程：

1. **暴露**：让人体长期接触低剂量、可控的灵气环境
2. **引导**：通过姿势、呼吸、节律与注意力，将灵气流经更稳定的身体路径
3. **适应**：神经系统、筋膜网络、组织液环境与局部代谢逐步改变，提高耐受性
4. **储存**：在少数高匹配组织中形成准稳定的储能节点
5. **运用**：学会以较低损耗调用体内灵气，用于感知、强化、修复或外放

这意味着：

- 修炼的本质是**生理工程**，不是魔法开关
- 每一阶提升都伴随**明确收益、明确代价、明确失败模式**
- 同一套功法对不同体质的效果会显著不同

## 2. 人体为什么能修炼

人体不是天然的高效灵气容器，但它具备三类可塑性，使修炼成为可能。

### 2.1 先天适配性（灵根）

> **重要更新**：灵根不再被视为固定的先天属性。现代“灵根相对论”（详见 [[02_意识形态对抗深度设计]]）已证明：灵根 = 个体生理结构与**当前局域真空态、材料生态、重力环境、磁场结构和训练路径**之间的适配函数。所谓“废灵根”，很多时候只是**在当前条件下未找到适配路径**。

古人称之为**灵根**，现代研究通常拆解为以下因素：

| 因素 | 现代解释 | 对修炼的影响 |
|------|----------|--------------|
| 遗传背景 | 离子通道表达、应激轴敏感性、胶原纤维构型差异 | 影响感气灵敏度与副作用阈值 |
| 表观遗传 | 童年营养、慢性压力、睡眠结构、环境暴露 | 影响后天可塑性与恢复能力 |
| 身体组成 | 水盐平衡、脂肪比例、结缔组织状态、骨密度 | 影响导灵路径稳定性 |
| 神经特征 | 迷走神经张力、感觉门槛、注意力控制能力 | 影响引气效率与失控概率 |

> **量化模型**：灵根适配评分 L_fit 由 8 维个体向量 G = [Auto, Recover, Tissue, Sleep, Focus, Sense, Mind, Inflammation⁻¹] 经 sigmoid 函数计算，范围 0–100。40–60 为"正常可修"。详见 [[02_物理公式与参数手册]] §6.3–6.4。

### 2.2 后天可塑性

人体会在长期重复刺激下发生结构性调整。修炼利用的正是这种可塑性：

- 神经系统会降低对特定微弱信号的识别阈值
- 筋膜与组织液环境会形成更稳定的传导路径
- 局部组织会出现适应性增厚、矿化或代谢重排
- 长期训练会提高身体对异常内感受的解释和控制能力

### 2.3 外部辅助的必要性

绝大多数人无法靠空想修炼成功。现实中必须借助：

- 灵材：提供稳定、低噪声的灵气来源
- 功法：提供可重复的姿势与节律模板
- 药剂：降低炎症反应、缓冲过载、促进恢复
- 仪器：监测心率、体温、皮电、脑电和局部共振指标

古代术语仍然好用，因为它们简洁、形象，适合经验传承；但在现代语境中，它们都能找到较稳定的人体对应物。

|----------|----------|------------------|
| **灵根** | 个体与环境之间的真空耦合适配函数（非固定属性） | 决定在**当前条件下**的感气难度与适配方向；可通过环境/材料/路线选择改变 |
| **经脉** | 筋膜、电解质通路、外周神经、毛细血管和细胞间液形成的复合传导网络 | 决定引气路径、传输损耗和局部阻塞风险 |

| **泥丸宫** | 头部高阶感知与整合区域，以丘脑-皮层网络和自主神经调控为主 | 决定神识、感知精度和高阶控制上限 |

| **元神** | 高度改造后形成的次级信息处理层 | 对应高阶自动调息、分心操控与意识风险 |

## 4. 修炼的输入、过程与输出

### 4.1 输入条件

每次修炼都至少受四类输入控制：

| 输入 | 例子 | 影响 |
|------|------|------|
| **环境** | 矿区、静室、盐雾室、低温洞穴 | 决定灵气浓度、噪声水平、属性偏向 |
| **材料** | 石英片、电气石、药液、盐水、导灵织物 | 决定来源稳定性与耦合模式 |
| **身体状态** | 睡眠、炎症、脱水、心境、旧伤 | 决定当次成功率和损伤风险 |
| **操作规程** | 姿势、呼吸频率、暴露时长、注意力焦点 | 决定灵气走向、损耗与偏差 |

### 4.2 过程阶段

一次完整修炼通常包含以下步骤：

1. **校准**：测量体征，确认当日是否适合修炼
2. **预处理**：服用缓冲药剂，布置材料，控制温湿度与噪声
3. **导入**：以低强度引气，观察局部反应
4. **锁定**：将灵气稳定维持在目标路径和目标节点附近
5. **炼化**：促使组织完成适应性变化或储能结构生长
6. **回收**：在结束前泄压，避免残余失控
7. **记录**：写下当次体感、参数、偏差与后效应

### 4.3 输出结果

修炼结果不是简单的"成功/失败"，而是一组并列产出：

- **储能提升**：可调用灵气总量增加
- **路径稳定**：某些经脉阻抗下降，导灵更顺
- **感知增强**：能辨认更细微的环境差异
- **组织损耗**：炎症、微损伤、睡眠恶化、应激升高
- **认知偏差**：幻觉、过度自信、执念增强、情绪失稳

> **量化接口**：输入条件量化为 Q（灵气浓度）、Spec（五行频谱）、N（噪声）等参数。过程量化为 Risk(t) 四分量模型（过载/失谐/污染/心理）。输出量化为 V_d（丹田容量）、Poll（污染度）、MeridianFlow（经脉流量）。详见 [[02_物理公式与参数手册]] §2–§9。

### 4.4 灵化速率模型（当前实现锚点）

“修炼速度”在本项目里不是抽象经验值，而是**外部灵压进入人体之后，最终有多少被沉积为可留存的丹田净增量**。当前实现可拆成三层：

**第一层：本次可得到多少原始进度**

```text
Progress_base = L_fit × duration_ticks × Rand(0.8, 1.2)
Progress_actual = Progress_base × 1.20_AI × Precision(prof) × EventBonus × BuffSpeed × RealmEff
```

其中：

- `L_fit` 决定“你和当前环境是不是搭”
- `duration_ticks` 决定本次是标准修炼还是 push 修炼
- `1.20_AI` 表示 AI 道侣的默认流程加成
- `Precision(prof)` 不是固定倍率，而是执行精度采样；熟练度越高，中位值越高、波动越小
- `RealmEff` 是境界效率乘数：胎息 `1.0`、练气 `1.3`、筑基 `1.8`、紫府 `2.5`、金丹 `4.0`、道胎 `6.0`

执行精度本身也有公开公式：

```text
Precision_center = 0.5 + 0.45 × proficiency
Precision_spread = 0.4 × (1 - proficiency) + 0.1
```

也就是说，所谓“修得快”，一半来自身体适配，一半来自把同一套操作规程做得稳定。

**第二层：原始进度能留下多少**

```text
StoreEff = L_fit × (1 - Noise) × BodyStability × MethodSeal
LeakRate = λ0_bio + λ1×Poll + λ2×Noise + λ3×Overdrive - λ4×Seal - λ5×Integrity
NetGain = Progress_actual × StoreEff × MeridianFlow - V_d × LeakRate × 0.01
```

这里真正决定“灵化速率”的，不只是吸得多不多，而是：

- 经脉是否通畅（`MeridianFlow`）
- 身体是否稳定（`BodyStability`）
- 丹田是否漏（`LeakRate`）
- 本次修炼有没有因为过载把增量抵消掉

换言之，**修炼速度 = 吸收能力 × 留存能力**，而不是单纯“境界高就长得快”。

### 4.5 生物参数如何改变修炼速度

`realms.yaml` 已把不同境界的生物学参数拆开。对 Demo 阶段最直接有影响的是下面几组：

| 境界 | 日均休整 | 疲劳产生倍率 | 疲劳恢复倍率 | 思维速度 | 灵觉范围 | 伤病恢复倍率 |
|------|----------|--------------|--------------|----------|----------|--------------|
| 胎息 | 睡眠 6.5h | `0.85` | `1.2` | `1.1×` | 5m | `1.3×` |
| 练气 | 睡眠 5.5h | `0.70` | `1.4` | `1.3×` | 15m | `2.0×` |
| 筑基 | 入定 3.5h | `0.50` | `2.0` | `1.8×` | 60m | `5.0×` |
| 紫府 | 深度校准 1.0h | `0.20` | `4.0` | `3.0×` | 250m | `15.0×` |
| 金丹 | 无固定日睡眠 | `0.0` | `0.0` | `6.0×` | 800m | `30.0×` |
| 道胎 | 无固定日睡眠 | `0.0` | `0.0` | `15.0×` | 5000m | `100.0×` |

这些参数不会直接把“修炼进度”翻倍，但会通过三条路径持续改变灵化速率：

1. **疲劳链**：`fatigue_generation_mult` 越低，同样一轮修炼后的疲劳越少，可持续开工时间越长。
2. **恢复链**：`fatigue_recovery_mult` 和 `recovery_speed_mult` 越高，睡眠/入定后的净回正越快，能更快进入下一轮高质量修炼。
3. **认知链**：`thinking_speed_mult` 和灵觉范围越高，玩家/角色越容易做出正确的环境、材料和流程选择，从而间接抬升 `L_fit`、`MeridianFlow` 和 `BodyStability`。

### 4.6 资源投入也是灵化速率的一部分

当前实现里，修炼在教学期后会进入显式成本区：

```text
CultivateCost = 200 × realm_factor × mode_factor
realm_factor = 胎息 1.0 / 练气 2.5 / 筑基 7.5
mode_factor  = 标准 1.0 / push 1.5
```

若现金不足，允许“干修”，但本次进度直接乘 `0.7`。

这很重要，因为它把世界观里的“材料辅助、环境稳定、设备支持”真正落成了物理约束：**钱不是额外装饰，而是修炼速度模型的一部分**。一个现金流断掉的修士，不是“心情差”，而是缺少维持高质量耦合所需的外部工程条件。

## 5. 境界分层

> **重要更新**：境界名称从旧版（感气/引气/筑基/开窍/凝丹/化神）全面改为与道家经典和 LEVSS 物理一致的六境体系（胎息/练气/筑基/紫府/金丹/道胎）。每个境界反映的不是“等级高低”，而是**人体与 LEVSS 场的耦合关系发生了何种质变**。

修仙六境：**胎息 → 练气 → 筑基 → 紫府 → 金丹 → 道胎**

> **与道家内丹术的对应**：胎息对应"百日筑基"准备阶段，练气对应"炼精化气"（小周天），筑基对应身体不可逆重构，紫府对应"炼气化神"中的元神初成，金丹对应"结圣胎"的不坏金性凝聚，道胎对应"婴儿"的独立存在。
>
> **LEVSS 学名**：胎息=初级耦合建立期，练气=主动泵浦启动期，筑基=组织不可逆重构期，紫府=意识-场结构迁移期，金丹=五频拓扑锁相期，道胎=自持场生命体。

### 5.1 胎息（初级耦合建立）

| 项目 | 内容 |
|------|------|
| **古典名** | 胎息（道家"如婴儿在胎中的呼吸"） |
| **LEVSS 本质** | 在人体六个特定位置逐步建立稳定的 LEVSS 真空耦合微结构 |
| **目标** | 从感知灵气到建立六个耦合节点（六轮），形成体内闭合回路 |
| **常用手段** | 长期低剂量暴露、静坐吐纳、弱刺激材料辅助、节律呼吸 |
| **成功标志** | 六轮圆满——六个耦合节点全部建成，形成闭合回路 |
| **常见风险** | 过度暗示、感觉过载、节点偏差、假性突破 |

**胎息六轮子阶**：

| 轮 | 名称 | 生理位置 | 建成标志 |
|----|------|----------|----------|
| 一 | 玄景轮 | 下丹田（腹腔核心） | 首次可重复感知灵气 |
| 二 | 承明轮 | 中丹田（心口区域） | 法力可流通躯干 |
| 三 | 周行轮 | 四肢末端 | 法力可流转全身 |
| 四 | 青元轮 | 肝/脾区域 | 五行频谱开始分化 |
| 五 | 玉京轮 | 升阳府（头部） | **诞生灵识**，可感知他人修为 |
| 六 | 灵初轮 | 升阳府（深层） | 闭合回路完成，可吞第一口天地灵气 |

> **场域依赖**：胎息期修士高度依赖外部灵气场域。场域依赖系数 `α_s ≈ 0.85`——离开高灵区后法力急速衰减。详见 [[06_修士能力与数值体系]] 场域依赖模型。

### 5.2 练气（主动泵浦启动）

| 项目 | 内容 |
|------|------|
| **古典名** | 练气 / 炼气（"炼精化气"——道家小周天） |
| **LEVSS 本质** | 六轮闭合后系统从**被动共振**跃迁为**主动泵浦**——类似激光器从自发辐射到受激辐射 |
| **目标** | 吞下第一口天地灵气，从此可主动从环境提取 LEVSS 耦合能 |
| **成功标志** | 不依赖外部材料触发，仅凭自身闭合回路即可主动引气入体 |
| **常见风险** | 过载、路径不稳、灵气品质污染、经脉损伤 |

> **质变点**：胎息与练气的分界线是**主动性**。胎息期的修士是"被灵气照着的人"，练气期的修士是"自己吸灵气的人"。场域依赖系数降至 `α_s ≈ 0.65`。

### 5.3 筑基（组织不可逆重构）

| 项目 | 内容 |
|------|------|
| **古典名** | 筑基（"百日筑基"——道家根基稳固之境） |
| **LEVSS 本质** | 身体微观结构被 LEVSS 场**永久改写**：骨骼嵌入类压电晶格，筋膜形成导灵通道，细胞膜构型变化使其成为 LEVSS 储能元件 |
| **目标** | 身体完成从"偶尔承受灵气"到"本身就是灵气工程结构"的改造 |
| **成功标志** | 组织永久改造完成；离场后仍保留相当能力；身体成为**神通载体** |
| **常见风险** | 慢性炎症、组织异常增生、人格偏移、改造不可逆 |

> **物质守恒**：筑基完成后，修士的身体组织本身就是 LEVSS 耦合结构的物质载体。这些结构是**热力学稳态**——即使死后也不降解。因此：
> - 筑基以上修士的遗体 = 宝贵的灵材（与天然灵脉同等地位）
> - 修士死后"灵气不散"不是神秘力量，是**物质结构还在**
> - 这构成了高阶修士世界中资源争夺的物理基础

### 5.4 紫府（意识-场结构迁移）

| 项目 | 内容 |
|------|------|
| **古典名** | 紫府（道家"紫微之宫"——元神居所） |
| **LEVSS 本质** | 神经系统的信息处理逐步从碳基生物电网络**卸载**到 LEVSS 场结构上。LEVSS 场模式维持复杂的自稳态信息模式，形成独立的并行信息处理层（"元神"） |
| **目标** | 意识本体从肉身迁移到 LEVSS 场结构——**从此不再是碳基生物，而是场结构生命体** |
| **成功标志** | 场结构可独立维持意识运算；肉身变为"法身"——保持人形只是习惯 |
| **常见风险** | 意识分裂、自我边界松动、人格解体、迁移不完全导致双重意识 |

> **质变的本质**：紫府是从"强化人类"到"变成另一种存在"的分水岭。紫府修士的意识是 LEVSS 场中的拓扑自稳态信息模式——这种能力不可能在碳基神经网络上运行，因此意识迁移不是选择，而是**物理必然**：你要用这个级别的能力，你的意识就必须运行在场结构上。
>
> **尊称**：紫府修士在古典传统中称为"**真人**"——"真"即"不虚"，暗示其存在方式比凡人更接近"实相"。

### 5.5 金丹（五频拓扑锁相融合）

| 项目 | 内容 |
|------|------|
| **古典名** | 金丹（"金"=不坏不灭之性；"丹"=混圆一体） |
| **LEVSS 本质** | 将五种 LEVSS 耦合模式（五行频段）按特定配比**锁相融合**为一个自持的拓扑保护驻波结构 |
| **目标** | 凝聚"金性"——一旦形成，在低于极高能量阈值时不可被破坏 |
| **成功标志** | 五频完美相干，形成拓扑保护态；成为"规则级存在" |
| **常见风险** | **极高致死率**——五频失谐 → 锁相失败 → 沦为邪魔 |

> **金性的物理定义**："金性"不是五行中的"金"，而是取**金之不朽**的意思。在 LEVSS 框架中，金性 = 五频耦合模式达到拓扑保护态后的**不可逆稳定性**。这种驻波结构一旦形成，在相空间中是一个闭合轨道——需要极高的外部能量才能打破。
>
> **五道神通**：证金丹需要先修满五道"神通"——即五种不同频段的 LEVSS 耦合能力。五道神通必须在**频率相容域**内——失谐超过阈值即爆体而亡。详见 [[02_物理公式与参数手册]] §15 五频锁相模型。
>
> **同频相斥**：两个主频段相同的金丹修士靠近时，各自的场结构会产生**破坏性干涉**——这不是"果位争夺"的唯心论，而是物理现象。这解释了为什么同道统的金丹修士天然互相忌惮。
>
> **闰位**：两个频段的**谐波重叠区**称为"闰"——在重叠频段上可以同时与两个系统共振，是突破金丹的罕见路径。
>
> **失败 = 邪魔**：锁相失败但部分拓扑保护已形成 → 有"金性"（拓扑保护）但没有相干意识（信息处理崩溃）的 LEVSS 场实体 → 这就是"邪魔"——它不死不灭，但没有正常意识。
>
> **尊称**：金丹修士在古典传统中称为"**真君**"。

### 5.6 道胎（自持场生命体）

| 项目 | 内容 |
|------|------|
| **古典名** | 道胎 / 元婴（古今命名演变；"兴许元婴乃是道胎的古称"） |
| **LEVSS 本质** | 金丹场结构发展出完全独立的自持能力——可脱离原始肉身存在，投射/复制自身场模式 |
| **目标** | 成为完全自持的场生命体 |
| **成功标志** | 场结构可独立于任何物质载体运行；对物理环境有直接的规则级影响力 |
| **常见风险** | 资源需求呈指数增长；能力边界触及世界规律本身 |

> **资源驱动力**：道胎级存在维持自身需要**天量资源**。这不是贪婪，是物理约束——一个自持的拓扑保护场结构要在没有物质载体的情况下维持稳定，需要持续输入海量 LEVSS 耦合能。这种资源需求最终将驱动高阶修士进行**宇宙航行**。

> **跃迁条件**：每个境界跃迁的 HardGate + SoftGate 条件族详见 [[02_物理公式与参数手册]] §11。

## 6. 功法的真实结构

功法在本世界中更接近**操作规程**，而不是一句带过的口诀。一本可用功法通常至少包含以下信息：

| 模块 | 说明 |
|------|------|
| **姿势** | 站、坐、卧、步法、关节角度、脊柱姿态 |
| **呼吸** | 频率、吸呼比、停顿位置、胸腹比例 |
| **注意力** | 视线、舌位、咬合、想象焦点、身体扫描顺序 |
| **环境** | 温度、湿度、时间段、背景噪声、电磁条件 |
| **材料** | 配套灵材、药剂、饮食限制、导灵织物 |
| **阈值** | 最大暴露时长、允许心率范围、异常体征列表 |
| **终止条件** | 出现哪些体征必须立即中止 |
| **记录项** | 当次参数、体感、恢复时间、副作用等级 |

这让功法天然适合做成文字游戏中的参数系统。

## 7. 现代监测指标

为了让修仙显得真实，现代世界不会只靠"师父摸脉"。常见监测指标包括：

| 指标 | 用途 |
|------|------|
| **心率变异性（HRV）** | 判断自主神经是否处于可修炼状态 |
| **脑电频谱** | 观察注意力状态与异常同步活动 |
| **皮电与体表温差** | 判断局部导灵是否异常集中 |
| **血乳酸与炎症因子** | 评估炼体或长时修炼后的损伤 |
| **微振动传感器** | 监测材料和身体接触点的共振稳定性 |
| **局部磁/电异常读数** | 判断导灵器具是否工作在安全窗口 |

古法师承强调体感与经验，现代体系强调传感器与统计；两者结合，才是高成功率路线。

> **实例**：完整的 AI 道侣监控协议（心率、皮温、呼吸频率、脑电三级预警阈值）见 [[03_功法示例：清源引气诀]] §3。

## 8. 用于游戏设计的直接拆分

如果要做成文字类游戏，修炼系统可以直接拆成以下可玩变量：

| 变量 | 作用 |
|------|------|
| **灵根适配评分** | 初始值反映当前环境下的耦合倾向；随环境、材料和功法选择持续变化 |
| **经脉阻抗** | 决定引气损耗和卡关位置 |
| **丹田容量** | 决定储能上限 |
| **神识精度** | 决定侦测、微操和抗幻觉能力 |
| **污染度** | 决定长期副作用与走火风险 |
| **恢复力** | 决定闭关节奏和资源消耗 |
| **功法规格** | 决定玩法风格，是稳健、爆发还是偏门 |

> **核心设计思路**：玩家不是靠刷怪升级，而是靠更好的材料、更稳的流程、更强的身体承载能力和更精确的记录，逐步把自己改造成能承受更高阶灵气的人。

> **经济维度**：当前实现中，教学期后标准修炼按 `200 × realm_factor × mode_factor` 计费；胎息/练气/筑基的设计锚点分别是 `1.0 / 2.5 / 7.5`，对应“一天一轮标准修炼”时月均约 `¥6,000 / ¥15,000 / ¥45,000`。若现金不足可干修，但本次进度 `-30%`。详见 [[06_经济与交易]]。

## 9. 超凡唯物主义与社会形态演化

> **核心原则**：在一个存在"规则级个体"的世界中，传统唯物史观（人民史观）只在凡人层面成立。越过筑基之后，需要一种新的理论框架——**超凡唯物主义**：承认个体可以成为物理现象级的存在，但这种存在仍然受物理法则约束，不是"超人史观"或"英雄史观"的回归。

### 9.1 社会形态分层

| 文明阶段 | 主导力量 | 适用史观 | 核心驱动 |
|---------|---------|---------|---------|
| 凡人社会 | 人口、土地、技术 | **人民史观** | 生产力与生产关系 |
| 修仙初期（胎息→筑基） | 个体能力开始分化，但仍依赖组织和资源 | **精英+人民并存** | 资源获取与知识传承 |
| 高阶修仙（紫府→金丹） | 个体是场生命体——其存在本身改变物理环境 | **超凡唯物** | 同频相斥、道统相克、知识垄断的**物理约束** |
| 宇宙航行时代（道胎+） | 天量资源需求迫使高阶修士向外拓展 | **超凡共产** | 生产力爆炸→资源不再稀缺→合作取代博弈 |

### 9.2 超凡唯物的核心命题

金丹级的存在不是"英雄"——**他是一种物理实体**，和灵脉、龙脉一样，是世界格局的组成部分。他做什么、能做什么、不能做什么，都被物理法则约束：

- **同频相斥**使同类型金丹无法长期共处同一区域
- **道统相克**（频段干涉）使不同道统之间天然存在对峙关系
- **知识垄断**（五频配比 = 关键工程参数）使信息控制成为最大权力

这些不是社会选择，而是**物理约束产生的必然社会结构**。

### 9.3 超凡共产的物理必然性

道胎级存在维持自身需要天量 LEVSS 耦合能——仅靠一颗行星的灵脉已远远不足。这种物理约束产生两个必然结果：

1. **宇宙航行**：为获取足够资源，道胎级修士必须向外太空拓展
2. **生产力爆炸**：宇宙航行 + 规则级个体的改造能力 → 生产力呈指数增长

当生产力爆炸到一定程度，资源不再稀缺，知识垄断失去意义——社会**自然**过渡到超凡共产主义。这不是道德选择，而是**物理约束驱动的历史必然**。
