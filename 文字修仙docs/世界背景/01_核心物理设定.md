## 1. 底层物理：为什么这个世界有"灵气"

### 1.1 宇宙常数没有变

精细结构常数 α ≈ 1/137，普朗克常量 h，光速 c ——全部如同现实。

基本粒子物理、化学、电子学、材料科学——**一切如常**。日常世界不崩。

### 1.2 发生了什么？

**局域激发真空态偏移（Local Excited Vacuum State Shift, LEVSS）**。

约公元前 12000 年，太阳系在绕银河系运动过程中，穿越了一片**暗物质密度异常增大的区域**（学界称为"暗物质流"或"暗物质壁"）。这次穿越没有产生任何经典力学意义上的碰撞，但导致了一个极其微妙的后果：

> 地球周围的**真空基态能级**产生了极其微小的、空间分布不均匀的偏移。

这个偏移量极其微弱——约为正常真空零点能密度的 10⁻³⁸ 级别——在绝大多数场景下完全可以忽略。但在特定条件下，它意味着：

- 真空量子涨落中持续产生的**虚粒子-反粒子对**，在正常条件下应瞬间湮灭
- LEVSS 偏移提供了额外的真空能量补充，使部分虚粒子**获得足够能量而不再湮灭**，成为准稳态的真实粒子
- 这些真实化粒子保留了真空态赋予的**自旋关联**，形成宏观可探测的量子关联场
- 不同自旋关联模式的粒子与特定晶格结构、化学键、分子构型产生弱耦合
- 当耦合条件精确满足时，这些真实化粒子携带的能量可以被**提取、积累、储存**

> **科学锚点**：STAR 合作组 2026 年在 *Nature* 发表的实验（Nature 650, 65-71）首次直接证实：高能碰撞可使真空中的虚夸克-反夸克对真实化，且真实化后保留真空态的自旋关联（近距产生时 ~100% 自旋对齐）。这为 LEVSS"虚粒子在能量补充下真实化"的核心机制提供了实验类比。

### 1.3 关键限制条件

LEVSS 效应**不是随处可用的**：

| 条件 | 说明 |
|------|------|
| **空间不均匀** | 地球上不同区域的偏移程度不同，受地质结构、地磁场、海拔、纬度影响 |
| **材料选择性** | 只有特定晶格/分子结构能与偏移后的涨落模式耦合 |
| **阈值效应** | 耦合需要最低密度/纯度/体积才能稳定发生 |
| **时间积累** | 能量提取速率极低，需要长时间才能积累到宏观可用量 |

这解释了为什么灵气不是"到处都有"，而是集中在特定地点、特定材料中。

### 1.4 工程边界：为什么这不是无限能源

LEVSS 效应虽然真实存在，但它**不会直接把世界变成廉价超能力社会**。原因在于它有非常严格的工程边界：

| 边界 | 说明 |
|------|------|
| **功率密度极低** | 单位时间内可提取的能量很少，远不足以替代常规电网或工业热源 |
| **退相干极快** | 灵气一旦离开合适载体，遇到热噪声、电磁噪声和机械扰动会迅速失稳 |
| **远距输运困难** | 灵气在空气、水体和普通导体中损耗极高，难以像电力那样跨区域调度 |
| **材料成本高昂** | 高效灵材往往要求高纯度、低缺陷、特定掺杂与复杂热处理，难以大规模普及 |
| **人体效率更低** | 人体不是理想容器，吸收、储存和释放过程中伴随显著损耗与生理风险 |
| **载体会老化失活** | 灵材在长期充放灵过程中会出现晶格疲劳、杂质积累和耦合频谱漂移 |

因此，灵气更像一种**高价值、低功率、强条件依赖**的边缘能源与生理工程技术，而不是能取代现代工业体系的万能力量。

> **量化定义**：以上边界条件已在 [[02_物理公式与参数手册]] 中量化为衰减函数、噪声模型和风险阈值。灵气浓度 Q 归一化范围 0.0–10.0。

## 2. 核心概念定义

这些是游戏世界中**学术界使用的标准定义**，古代修士的术语只是同一现象的早期描述。

### 2.1 灵气（Qi / Vacuum-Coupled Energy, VCE）

> **灵气 = LEVSS 偏移导致真空涨落中虚粒子真实化后，形成的准稳态量子关联粒子流。**

灵气的物理实体是**从真空涨落中获得能量补充而未湮灭的真实化粒子**。这些粒子携带真空态赋予的自旋关联，在特定材料中可被捕获和利用。它有以下特征：

- **可储存**：在合适的载体中可以长期保持（真实化粒子被晶格束缚）
- **可转移**：通过物理接触或特定场结构可以在载体间流动
- **可感知**：生物神经系统在适应后可以微弱感知其自旋关联场
- **可衰减**：脱离载体后，因量子退相干而逐步丧失关联——即灵气"散逸"
- **难远距输运**：离开高匹配载体后退相干加速，与 STAR 实验中远距超子对关联衰减一致
- **有"频段"**：不同自旋关联模式对应不同的场效应（古人低维映射为金木水火土）

### 2.2 灵材（Qi-Active Material, QAM）

> **灵材 = 具有稳定零点耦合截面的天然或人工材料。**

灵材的核心参数：

| 参数 | 含义 |
|------|------|
| **耦合截面（σ_q）** | 该材料捕获真空涨落能的效率 |
| **容量密度（ρ_q）** | 单位体积能储存的灵气总量 |
| **耦合频谱** | 该材料响应哪些频率的涨落（决定"属性"） |
| **衰减时间（τ_d）** | 灵气在其中的半衰期 |
| **导灵率（κ_q）** | 灵气在其中的传导速度与损耗 |

> **量化接口**：以上参数均为派生量，由翻译函数从现实矿物/植物数据计算得出，非手工赋值。计算规则见 [[02_物理公式与参数手册]] §4.3。灵材数据库见 [[01_现实矿物灵材数据库首轮筛查]]，命名体系见 [[02_灵材命名体系与批量转译规则]]。

### 2.3 修炼（Somatic Vacuum-Coupling Adaptation, SVCA）

> **修炼 = 人体对真空耦合能的吸收、缓冲、驯化与器官化利用过程。**

修炼不是"吸收天地灵气然后变强"这种黑箱。它是一个**可描述、可量化、有风险**的生理改造过程：

1. **暴露**：将身体置于灵气浓度足够的环境中
2. **引导**：通过特定姿势、呼吸、注意力模式，使灵气沿特定路径流经身体
3. **适应**：身体组织在反复暴露下发生微观结构变化，提高耐受和利用能力
4. **储存**：特定区域的组织形成稳定的灵气储存结构
5. **运用**：学会主动释放和控制储存的灵气

## 3. "灵气属性"的物理本质

古代修士根据经验将灵气分为五行，现代物理给出了更精确的解释。

需要注意的是，**五行首先是经验分类，不是现代实验室的一级学科目录**。实验室通常按主耦合机制、频谱响应和材料形态命名；民间与古籍为了记忆和传承，才把复杂现象归并为五类。

五行本质上是**不同频段/模式的真空耦合**，因为 LEVSS 偏移不是单一频率的，而是在多个频段上有不同程度的增强。

| 古称 | 现代名称 | 耦合模式 | 对应物理现象 | 典型灵材 |
|------|----------|----------|-------------|---------|
| **金** | 声子纵模耦合 | 晶格纵向振动与真空涨落共振 | 高频机械波、切割力 | 石英、金属晶体 |
| **木** | 手性光子耦合 | 手性分子结构与真空涨落的圆偏振模式耦合 | 生长、修复、有机催化 | 纤维素纳米晶、多糖螺旋体 |
| **水** | 氢键网络耦合 | 水分子氢键网络的集体振动与真空涨落耦合 | 流动、渗透、溶解、适应 | 含结晶水矿物、冰晶 |
| **火** | 铁电极化耦合 | 铁电/压电材料的自发极化与真空涨落耦合 | 热释放、能量爆发 | 钛酸钡、电气石 |
| **土** | 层状结构耦合 | 层状硅酸盐的层间振动与真空涨落耦合 | 稳定、防御、吸附、封印 | 云母、蒙脱石、高岭土 |

> **关键设定**：五行不是"五种灵气"，而是同一种能量在不同频段/模式下的表现。更准确地说，它是古人对复杂耦合谱的低维命名法，就像民间把多种天气都统称为"风雨"，而现代气象学会进一步拆分。

## 4. 灵气的地理分布

灵气浓度不均匀，**高灵区**通常具备以下地质特征（全部有现实对应）：

| 高灵区特征 | 现实地理对应 | 原因 |
|-----------|------------|------|
| 大型石英矿脉带 | 秦岭山脉、巴西米纳斯吉拉斯 | 大面积压电晶体形成天然聚灵阵 |
| 断层活动区 | 龙门山断裂带、圣安德烈亚斯断层 | 应力变化激活压电效应 |
| 地磁异常区 | 库尔斯克磁异常区、鞍山-本溪铁矿带 | 磁场结构影响真空态偏移方向 |
| 高海拔低气压区 | 青藏高原、安第斯高原 | 气压降低减少大气对灵气的衰减散射 |
| 火山/地热带 | 冰岛、日本、黄石 | 热梯度激活热释电效应 |
| 盐湖/卤水区 | 察尔汗盐湖、死海 | 高浓度电解质溶液有特殊导灵特性 |

> **古人说的"洞天福地"、"龙脉"，就是对高灵区的经验标注。**

> **详细数据**：南瞻 25 处 + 海外 12 处高灵区的完整 Match_B 评分、六因子评分、噪声评分、主频谱数据见 [[05_高灵区地理志]]。判定规则见 [[02_物理公式与参数手册]] §3.4。

## 6. 核心冲突轴线

### 6.1 认知冲突
灵气是**科学**还是**新玄学**？学界内部分裂严重。LEVSS 理论有实验支撑但不完善，古代典籍有大量有效经验但混杂神话。两边互不服气。

### 6.2 资源冲突
高阶灵材（高纯度石英、稀土掺杂晶体、天然超导矿）被**国家战略管控**。普通人只能接触低阶产品。黑市、走私、矿区争夺是常态。

### 6.3 身体冲突
修炼**不是升级，是高风险生理改造**。每个阶段都有可能导致：
- 神经损伤（灵气冲击未适应的神经通路）
- 器官异变（组织在灵气浸润下发生不可控的微观结构改变）
- 精神异常（过度感知导致的感觉过载、幻觉、偏执）
- 走火入魔 = 灵气失控在体内形成破坏性共振

### 6.4 伦理冲突
- 为了筑基，你需要改变自己的神经系统——**你还是你吗？**
- 灵材开采对环境的破坏
- 修士与普通人之间日益扩大的能力鸿沟
- 军事化修炼的伦理边界

## 8. 游戏核心循环设想

```
采集/购买材料 → 鉴定纯度与属性 → 加工处理（切割/研磨/萃取/煅烧）
       ↓
选择功法（操作规程）→ 配置辅助材料 → 开始修炼（控制参数）
       ↓
承受副作用 → 判定修炼结果 → 恢复休整
       ↓
记录数据 → 优化方案 → 解锁更高阶工艺与理论
```

**不是刷怪拿经验，而是"实验→修炼→恢复→记录→改进"。**
