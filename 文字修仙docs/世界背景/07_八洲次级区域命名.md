# 07_八洲次级区域命名

> **一句话版**：八洲不是八个均质的板块——每个洲内部都有多个功能区、战略走廊、工业带和灵脉分区。本文档为八洲定义次级地理单元，供后续势力、经济、叙事文档使用。
> **关联文档**：[[03_八洲文明格局与超凡纪年]] · [[05_高灵区地理志]] · [[06_裂地玄洲热战详写]]

> [!note] 命名规则
> - **古称层**：用于叙事和设定文本，带历史/地理/文化感
> - **机构层**：用于制度文件和技术文档，强调功能和坐标
> - 所有名称禁止使用现实国名/城市名

## §1 南瞻部洲

### 1.1 核心区

| 古称层 | 机构层 | 地理特征 | 战略功能 |
|--------|--------|---------|---------|
| **中原沃土** | 南瞻中央平原区 | 大河冲积平原，人口稠密 | 政治中心、QESA 总部、修士人口基数最大 |
| **昆仑天脊** | 南瞻西陆矿脊 | 高原-山脉带，极端海拔 | 军事管制区、S 级灵脉、筑基/紫府工程中心 |
| **东海灵湾** | 南瞻东海沿岸区 | 近海城市群、港口、岛链 | 灵芯片制造、灵材进出口、海洋灵气监测 |
| **南岭药带** | 南瞻岭南走廊 | 亚热带山地、河谷 | 灵药产业带、罗浮山灵药研发集群 |
| **巴蜀灵盆** | 南瞻西南高灵盆 | 盆地+周边高山（峨眉、青城） | 筑基工程中心（峨眉站）、火属性灵材 |
| **长白冰原** | 南瞻东北特区 | 火山-冻土-深湖叠加 | 天池灵能特区（共管）、水火双修路线 |

### 1.2 辅助区

| 古称层 | 机构层 | 特征 |
|--------|--------|------|
| **终南隐谷** | 南瞻关中古法保留区 | 全真龙门派核心据点、古法传承中心 |
| **黄淮学府带** | 南瞻中部教育区 | 修炼学院密集、新手培训基地 |
| **南海台礁** | 南瞻南方海域区 | 海洋灵脉勘探、灵材走私节点 |
| **丝路走廊** | 南瞻西北通连带 | 连接昆仑与中原的物流动脉、灵材运输线 |

## §9 跨洲战略通道

八洲之间的联系不只是外交——灵材贸易、修士流动和情报交换依赖物理通道。核心不对称：南瞻控制道统知识输出，其他洲用工业品/算力/灵材交换。

| 通道名 | 连接 | 功能 |
|--------|------|------|
| **道统输出-工业交换轴** | 南瞻 → 全球 | 南瞻有选择地输出修炼知识，换取各洲工业品、算力、灵材（核心战略杠杆） |
| **制造-药材轴** | 南瞻 ↔ 西牛 | 灵芯片/工业品换灵药/原料；西牛修炼知识完全依赖南瞻 |
| **算力-逆向工程轴** | 鹰洲 ↔ 东胜 | AI 算力/精密检测共享；试图用算力弥补道统传承缺失 |
| **不对称铁幕** | 南瞻阵营 vs 鹰洲阵营 | 不是对等冷战——南瞻掐修炼知识，鹰洲/东胜掐芯片/算力。双方互有筹码但南瞻在修炼核心上占压倒性优势 |
| **裂地走廊**  | 全洲 → 裂地 | 各方投送矿区资源/军力的物流动脉 |
| **极地暗线** | 北俱 ↔ 鹰洲 | 非正式情报和灵材走私通道 |
| **珊瑚航道** | 南溟 ↔ 东胜 | 特种矿物出口、战略缓冲区贸易 |
