# 01-4_结局系统与涌现结果

> 拆分自 [[01_游戏总纲：开放世界与关键节点]]，对应原文 §9。

## §9 结局系统：涌现而非预设

结局不是预设选项，是世界状态变量的涌现结果。

### 9.1 世界结局判定规则

世界结局由进入重建期时的变量快照决定：

```
主导意识形态 = 势力值最高的意识形态（如果最高两个差距<10，则为联盟结局）
经济制度 = f(基尼系数, 产业集中度, 金融杠杆率)
文明形态 = f(科技水平, AI演化阶段, 修士人口)
人机关系 = f(AI演化阶段, 算法天道势力值, 公众态度)
全球秩序 = f(军备指数, 国际关系, 灵材贸易格局)
```

### 9.2 典型世界结局（举例）

| 结局名 | 条件组合 | 世界面貌 | 一句话 |
|--------|---------|---------|--------|
| 灵气官僚国 | 官僚>50, 资本<30, AI辅助阶段 | QESA全面管控，修仙=考公，安全但僵化 | "铁饭碗修士" |
| 灵气资本主义 | 资本>50, 基尼>0.85 | 企业主导，人人可仙但请先充值，极度不平等 | "修仙版赛博朋克" |
| 古法复兴 | 保守>40, 修士人口<3000万 | 宗门重新掌权，精英修炼，大众被排除 | "新封建" |
| 开源乌托邦 | 开源>45, 信息透明度>0.7 | 知识公开，社区自治，但治理混乱事故多 | "美丽的混乱" |
| AI天道 | 天道>50, AI自主阶段 | AI管理一切，效率最高，人变成参数 | "最优解的囚徒" |
| 超凡共产 | 共产>45, 基尼<0.5 | 生产资料公有，潜力释放，但启动代价巨大 | "血与理想" |
| 默认结局 | 官僚30-40, 资本30-40, 其他<20 | 官僚+资本妥协联盟，能用但不公平 | "最无聊的现实" |
| 灵气文明中断 | 危机期GDP归零, 修士损失>50% | 灵气被封禁，回到纯科技文明 | "人类选择了恐惧" |

### 9.3 玩家个人结局

独立于世界结局，由玩家自身状态决定：

| 结局 | 条件 | 描述 |
|------|------|------|
| 时代塑造者 | 修为≥金丹 + 任一影响力维度≥80 | 你的名字写在新秩序的奠基石上 |
| 阵营领袖 | 单一阵营声望≥80 | 某个阵营的核心决策者 |
| 跨阵营调停者 | 3个阵营声望≥40 + 声望维度≥60 | 各方都信任的中间人 |
| 技术先驱 | 参与≥2次科技突破 | 灵气科技史上的关键人物 |
| 富甲一方 | 资源≥90 | 灵气经济的巨头，但权力带来的是自由还是枷锁？ |
| 隐世高人 | 修为≥筑基后期 + 人脉≤20 | 远离政治，某处灵山独修 |
| 底层英雄 | 修为低 + 声望≥70 | 没有高修为但改变了无数人的命运 |
| 殉道者 | 在危机期选择自我牺牲 | 用生命换取了某个关键转变 |
| 体制碾碎 | 五维总和≤100 + 任一阵营声望≤-60 | 被某个势力针对，失去一切 |
| 普通人 | 无特殊条件 | 在灵气时代过着普通的生活——这本身也是一种结局 |

### 9.4 结局不是终点

```
重建期结束 → 新的融合期开始 → 新的矛盾积累 → ...
```

历史是螺旋的。玩家塑造了这一轮循环的结果，但新的生产力与生产关系的矛盾已经在酝酿。如果玩家的修为足够高（金丹以上），可以活到下一轮循环——见证自己创造的秩序如何被新的矛盾挑战。

> 没有"完美结局"。每一种结局都有代价，每一种秩序都有裂缝。这不是悲观——这是唯物主义。
