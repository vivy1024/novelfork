## §3 世界模拟引擎：条件-概率-转变

世界不按剧本走。每个"历史转变"都是条件积累到阈值后的概率事件。

### 3.1 转变触发公式

```
每游戏月结算一次：
  对每个潜在转变 T：
    成熟度 M(T) = Σ(条件_i × 权重_i)    // 0–100
    随机扰动 R = uniform(0.7, 1.3)
    关键人物修正 P = 1 ± NPC影响系数     // 0.5–1.5
    
    触发值 = M(T) × R × P
    
    if 触发值 ≥ 阈值(T):
        转变发生，世界状态更新
    else:
        条件继续积累，下月重算
```

> 同样的世界状态，因为随机扰动 R 的存在，每次游戏的历史轨迹都不同。玩家的作用是改变 M(T) 的条件值和 P 的人物修正——提高或降低某个转变的发生概率。

### 3.2 转变类型

| 类型 | 驱动力 | 阈值范围 | 频率 | 举例 |
|------|--------|---------|------|------|
| 经济转变 | 生产力/市场/金融指标 | 60–80 | 高 | 灵材价格崩盘、企业并购、金融危机 |
| 科技转变 | 研发投入/突破条件 | 70–90 | 中 | AI 修炼突破、新灵材合成、功法革命 |
| 政治转变 | 势力值/公众情绪/国际关系 | 65–85 | 中 | 政策变革、政权更迭、国际冲突 |
| 社会转变 | 基尼系数/事故率/意识形态 | 75–95 | 低 | 修士起义、意识形态革命、文明范式转换 |
| 灾变 | 灵气浓度/地质/生态 | 85–99 | 极低 | 灵气潮汐、灵脉枯竭、灵气污染灾难 |

### 3.3 条件积累机制

条件不是凭空出现的，它们由底层变量持续驱动：

```
经济条件 ← 灵材供需比、产业集中度、基尼系数、金融杠杆率
科技条件 ← 研发投入、人才密度、数据积累、国际合作度
政治条件 ← 势力值差距、公众不满度、国际压力、腐败指数
社会条件 ← 修士人口增速、事故率、阶层流动性、信息透明度
灾变条件 ← 灵气浓度波动、过度开采指数、生态承载力
```

> **玩家的真正战场**：不是在"事件"发生时做选择，而是在事件发生之前，通过杠杆改变底层条件的积累速度和方向。

## §4 经济与金融系统

灵气经济不是背景板，是世界变化的第一推动力。

### 4.1 宏观经济变量

| 变量 | 初始值(2035) | 范围 | 驱动什么 |
|------|-------------|------|---------|
| 灵气产业GDP | ¥3.2万亿 | ¥1–50万亿 | 就业、税收、国际地位 |
| 灵材价格指数 | 100 | 30–500 | 修炼成本、企业利润、黑市活跃度 |
| 修炼资源基尼系数 | 0.72 | 0.25–0.95 | 社会稳定、底层情绪、革命概率 |
| 产业集中度(CR3) | 55% | 15–90% | 垄断程度、创新活力、政策博弈 |
| 灵材供需比 | 0.85 | 0.3–2.0 | 价格、走私、国际贸易 |
| 金融杠杆率 | 1.2 | 0.5–5.0 | 泡沫风险、系统性危机概率 |
| 黑市占GDP比 | 8% | 2–30% | 社会秩序、腐败、底层生存 |

### 4.2 金融系统

| 金融工具 | 出现条件 | 效果 | 风险 |
|---------|---------|------|------|
| 灵材期货 | 产业GDP > ¥5万亿 | 价格发现+套期保值 | 投机泡沫 |
| 修炼贷 | 金融杠杆率 > 1.5 | 底层修士获得资金 | 债务陷阱（参考《没钱修什么仙？》） |
| 灵根保险 | 修士人口 > 5000万 | 走火入魔风险分担 | 道德风险+逆向选择 |
| 灵材ETF | 产业集中度 < 40% | 散户投资灵气产业 | 散户被割 |
| 修为证券化 | 金融杠杆率 > 3.0 | 将修士未来收入打包出售 | 2008式系统性崩溃 |
| 灵脉REITs | 灵脉私有化程度 > 50% | 灵脉收益权证券化 | 资源金融化+过度开采 |

> **玩家可以做什么**：做空灵材期货制造价格危机、创建修炼互助基金对抗修炼贷、推动灵根保险公共化、揭露修为证券化的庞氏骗局。

### 4.3 经济危机触发条件

| 危机类型 | 触发条件 | 概率修正 | 后果 |
|---------|---------|---------|------|
| 灵材价格崩盘 | 供需比 > 1.5 且 杠杆率 > 2.0 | +20%/月 | 企业破产潮、矿工失业、黑市暴涨 |
| 修炼贷危机 | 杠杆率 > 3.5 且 事故率上升 | +15%/月 | 底层修士破产、社会动荡 |
| 灵脉枯竭恐慌 | 过度开采指数 > 80 | +10%/月 | 灵材价格暴涨、国际冲突 |
| 系统性金融危机 | 杠杆率 > 4.0 且 任一危机已发生 | +30%/月 | 全面经济崩溃、政治洗牌 |

### 4.4 经济联动规则

```
灵材价格↑ → 修炼成本↑ → 底层修士退出↑ → 散修占比↓ → 基尼系数↑
                                                    → 黑市占比↑
产业集中度↑ → 企业利润↑ → 游说能力↑ → QESA被俘获概率↑
                       → 创新活力↓ → 科技突破概率↓
金融杠杆率↑ → 短期繁荣 → 泡沫积累 → 危机概率↑ → 崩溃后重建
基尼系数 > 0.85 → 社会转变阈值降低20点（革命更容易发生）
```

## §5 科技与 AI 演化树

科技不是线性进步，是条件积累后的概率突破。AI 是独立的变量轴。

### 5.1 科技演化树

```
灵气基础科学
├── 灵气物理学 → 灵气场方程 → 灵气预测模型 → 灵气天气控制
├── 灵材科学 → 人工灵材 → 灵材纳米工程 → 可编程灵材
├── 灵气生物学 → 灵根机制 → 灵根改造 → 人工灵根
└── 灵气工程学 → 灵气网络 → 公共灵气基础设施 → 灵气文明

修炼科学
├── 功法工程 → 标准化功法 → AI辅助功法 → 自适应功法
├── 丹药科学 → 工业化制药 → 精准丹药 → 个性化灵药
├── 法器工程 → 量产法器 → 智能法器 → 灵气-电子融合
└── 安全科学 → 风险预警 → 自动干预 → 零事故修炼

AI 演化（独立轴）
├── 辅助阶段 → AI修炼助手、灵气数据分析
├── 协作阶段 → AI参与功法设计、灵材配方优化
├── 自主阶段 → AI独立决策修炼方案、资源分配
└── 超越阶段 → AI修炼？AI成仙？AI文明？
```

### 5.2 科技突破条件

| 突破 | 前置条件 | 研发投入阈值 | 基础概率 | 影响 |
|------|---------|------------|---------|------|
| 低成本灵材合成 | 灵材科学Lv2 + 产业GDP>¥5万亿 | ¥500亿累计 | 3%/月 | 灵材价格腰斩，矿业集团崩溃 |
| AI 修炼突破 | AI协作阶段 + 修炼数据>1亿条 | ¥200亿累计 | 2%/月 | 修炼效率×3，但主体性争议 |
| 灵根改造技术 | 灵气生物学Lv3 + 人体实验数据 | ¥1000亿累计 | 1%/月 | 灵根不再是天赋，社会结构剧变 |
| 公共灵气网络 | 灵气工程Lv2 + 基础设施投资 | ¥2000亿累计 | 5%/月 | 普通人也能用灵气，文明级变革 |
| 灵气武器化 | 法器工程Lv3 + 军方投入 | ¥800亿累计 | 4%/月 | 军事平衡打破，国际危机 |
| 零事故修炼 | 安全科学Lv3 + AI自主阶段 | ¥300亿累计 | 2%/月 | 走火入魔成为历史，但代价是什么？ |

### 5.3 AI 演化的独立时钟

AI 不受玩家控制，它有自己的演化节奏：

| 阶段 | 触发条件 | 预计时间 | 世界影响 |
|------|---------|---------|---------|
| 辅助→协作 | 修炼数据 > 1000万条 | 2038–2042 | 灵途平台效率暴涨，人类修炼师开始失业 |
| 协作→自主 | AI 参与决策 > 50% + 无重大事故 | 2045–2055 | "天道代理人"问题——谁在修炼，人还是AI？ |
| 自主→超越 | AI 修炼成功（概率极低） | 2070–2100 | 文明级问题——AI 是新物种还是工具？ |

> **玩家可以做什么**：加速或延缓 AI 演化（投入资源/破坏数据中心）、决定 AI 的价值函数（谁来定义"最优"？）、在 AI 超越阶段选择合作还是对抗。

### 5.4 科技-经济联动

```
科技突破 → 生产力跃升 → 旧生产关系不适配 → 社会矛盾激化 → 制度变革压力
                                                              ↓
                                              变革成功 → 新均衡
                                              变革失败 → 危机/战争/倒退
```

> 这就是历史唯物主义的核心循环。每一次科技突破都不是"好事"——它是矛盾的催化剂。

## §6 社会与政治动力学

### 6.1 社会变量

| 变量 | 初始值(2035) | 范围 | 驱动什么 |
|------|-------------|------|---------|
| 注册修士人口 | 2000万 | 500万–5亿 | 市场规模、社会结构、政治权重 |
| 散修占比 | 40% | 10–70% | 社会稳定、黑市规模、革命潜力 |
| 年事故率 | 56.5万起 | 5万–500万 | 公众恐慌、监管压力、AI需求 |
| 修士-凡人关系 | 好奇+焦虑 | 崇拜↔恐惧↔仇恨 | 政策方向、隔离/融合 |
| 阶层流动性 | 0.15 | 0.02–0.5 | 社会活力、底层希望/绝望 |
| 信息透明度 | 0.35 | 0.05–0.9 | 公众认知、阴谋论、信任度 |
| 国际灵气军备指数 | 40 | 10–100 | 军事化程度、技术封锁、战争风险 |

### 6.2 六大意识形态势力值

| 意识形态 | 初始值 | 自然增长条件 | 自然衰退条件 |
|---------|--------|------------|------------|
| 国家官僚 | 35 | 事故率↑、国际威胁↑ | 腐败曝光、效率低下 |
| 平台资本 | 25 | GDP↑、科技突破 | 金融危机、垄断反弹 |
| 古法保守 | 15 | 事故率↑（"现代方法不安全"） | 修士人口↑（规模化需求） |
| 开源共修 | 10 | 信息透明度↑、基尼系数↑ | 重大事故（"没监管不行"） |
| 算法天道 | 10 | AI演化推进、事故率↓ | AI事故、主体性运动 |
| 超凡共产 | 5 | 基尼系数>0.8、经济危机 | 经济繁荣、阶层流动性↑ |

> 势力值不是玩家"选"出来的，是世界条件"长"出来的。玩家能做的是改变条件。

### 6.3 政治转变触发

| 转变 | 条件 | 阈值 | 后果 |
|------|------|------|------|
| 修炼管制收紧 | QESA权力>70 + 事故率上升 | 65 | 散修被打压，黑市暴涨 |
| 修炼管制放松 | 散修占比>50% + 公众支持 | 70 | 事故率短期上升，长期创新 |
| 灵脉国有化 | 国家官僚>50 + 超凡共产>20 | 75 | 企业反弹，国际制裁风险 |
| 平台反垄断 | 平台资本>40 + 公众不满>60 | 70 | 灵途拆分，散修获益 |
| 修士起义 | 基尼>0.85 + 事故率>100万 + 超凡共产>30 | 85 | 社会动荡，所有势力重新洗牌 |
| 灵气军备竞赛 | 军备指数>70 + 国际冲突 | 60 | 资源军事化，民用萎缩 |
| AI 治理危机 | AI自主阶段 + 算法天道>40 | 75 | "谁控制AI"成为核心政治问题 |

### 6.4 国际关系矩阵

| 关系轴 | 初始状态 | 恶化条件 | 改善条件 |
|--------|---------|---------|---------|
| 中美灵气竞争 | 紧张合作 | 军备指数↑、技术封锁 | 联合研究、灾变合作 |
| 中日灵材贸易 | 互补 | 稀土灵材限制出口 | 技术换资源 |

| 全球灵气治理 | IVCERA主导 | 大国退出、标准分裂 | 灾变后合作 |

> 国际关系不是背景——灵材贸易战、技术封锁、灵脉主权争端都会直接影响国内经济和政治。
