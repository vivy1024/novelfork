## §7 四个时代的条件矩阵

每个时代不是固定年份，而是**条件触发的相变**。当足够多的条件满足时，世界进入下一个时代。

起点是 2035 年的南瞻部洲——全球第二大经济体，完整工业体系，5G/AI/量子计算已在应用，核武器国家，一带一路贸易网络，14 亿人口。灵气复苏是叠加在这个基础上的新变量，不是替代。

### 7.1 融合期（游戏起始状态，约 2035–）

**现实基础**：南瞻部洲 GDP ¥120 万亿，灵气产业 ¥3.2 万亿（占 2.7%）。现有制度、法律、经济体系全部在运转，灵气只是一个新增的产业部门。

**核心矛盾**：现有制度能不能容纳灵气这个新变量？QESA 用管核电的思路管修炼，企业用做互联网的思路做灵途平台，宗门用传统师徒制应对现代社会——每一方都在用旧框架套新事物。

| 维度 | 状态 |
|------|------|
| 经济 | 灵气是新增长点，传统产业+灵气产业并行，VC 疯狂涌入 |
| 科技 | AI+灵气交叉研究起步，现有科研体系（高校/中科院/企业R&D）主导 |
| 金融 | 灵材现货交易为主，期货/保险等衍生品刚出现，监管跟不上 |
| 社会 | 修士是"新职业"而非"新阶级"，公众态度是好奇+焦虑 |
| 政治 | QESA 主导监管，但权力边界模糊，与现有部委（科技部/卫健委/自然资源部）职能重叠 |
| 国际 | 各国独立研究，IVCERA 协调但无约束力，灵材贸易刚起步 |

**进入分化期的条件**（满足 3/5 即触发相变）：
1. 灵气产业占 GDP > 10%（灵气不再是"新兴产业"而是"支柱产业"）
2. 修士人口 > 5000 万（修士成为一个有政治权重的群体）
3. 第一次灵气相关的系统性事件（金融危机/重大事故/国际冲突，任一）
4. AI 进入协作阶段（AI 不再只是工具，开始参与决策）
5. 基尼系数 > 0.80（修炼资源分配的不平等已经引发社会问题）

### 7.2 分化期

**核心矛盾**：灵气产业已经大到不能忽视，但利益分配严重不均。六大意识形态从"学术讨论"变成"政治纲领"，每一方都在争夺对灵气文明的定义权。

| 维度 | 状态 |
|------|------|
| 经济 | 灵气产业成为支柱，传统产业被灵气改造或淘汰，寡头格局成型 |
| 科技 | AI 参与修炼决策，灵根改造技术出现争议，军事化研究加速 |
| 金融 | 灵材期货/修炼贷/灵根保险全面铺开，金融杠杆率攀升 |
| 社会 | 修士-凡人分化加剧，"灵气鸿沟"成为核心社会议题 |
| 政治 | 六大意识形态全面对抗，QESA 内部分裂，地方与中央博弈 |
| 国际 | 灵材贸易战，技术封锁与反封锁，灵脉主权争端升温 |

**进入危机期的条件**（满足 3/5 即触发相变）：
1. 基尼系数 > 0.88 或 系统性金融危机已发生
2. 国际灵气军备指数 > 75
3. AI 进入自主阶段
4. 至少一次灵脉枯竭或灵气污染灾难
5. 大规模修士社会运动已发生（罢工/起义/占领）

### 7.3 危机期

**核心矛盾**：旧的制度框架已经无法容纳灵气带来的生产力变革，矛盾以最激烈的方式爆发。但这是 2035+ 的世界——有核武器、有互联网、有全球供应链，危机的形态和历史上完全不同。

| 维度 | 状态 |
|------|------|
| 经济 | 全面危机或极端分化，旧金融体系承压，灵气经济可能脱钩 |
| 科技 | 技术被各方争夺/封锁，AI 的角色成为核心争议 |
| 金融 | 修为证券化崩盘/灵材期货暴跌/修炼贷违约潮（取决于哪个先爆） |
| 社会 | 大规模社会重组，旧阶级结构动摇，新的社会契约在混乱中酝酿 |
| 政治 | 极端路线崛起，温和派被挤压，但核武器的存在限制了冲突烈度 |
| 国际 | 灵气冷战或局部热战，但全面核战争的恐惧限制了升级 |

**进入重建期的条件**（满足 2/4 即触发相变）：
1. 危机造成灵气产业 GDP 下降 > 30%
2. 至少两个主要势力/企业崩溃或重组
3. 新的国内或国际协议框架达成
4. 某种意识形态组合取得主导地位（任一势力值 > 55 或两个 > 40 的联盟）

### 7.4 重建期

**核心矛盾**：危机之后建什么？这是玩家影响力最大的时代——旧秩序已碎，新秩序未定。但重建不是从零开始——南瞻的工业基础、科技积累、人口规模、国际地位都还在，问题是用什么制度来组织它们。

| 维度 | 状态 |
|------|------|
| 经济 | 重建+新制度设计，资源重新分配的窗口期 |
| 科技 | 危机期的技术积累转为民用，AI 角色在新制度中被重新定义 |
| 社会 | 新社会契约谈判——修士和凡人的关系、灵气资源的分配方式 |
| 政治 | 新的治理框架，可能是一种意识形态主导，也可能是多种妥协 |
| 国际 | 新的全球灵气治理体系，可能比 IVCERA 更有约束力 |

### 7.5 时代转换规则

```
融合期 → 分化期 → 危机期 → 重建期 → （新的融合期...）
```

- 时代只能前进，不能跳过
- 每个时代的持续时间取决于条件积累速度——可能 10 年，可能 30 年
- 玩家的修为决定你能经历几个时代（胎息期寿命正常，金丹期 150–200 年，道胎期理论无限）
- 如果危机期处理极差（灵气产业 GDP 归零 + 修士人口损失 > 50%），触发"灵气文明中断"——不是回到过去，而是灵气成为被封禁的技术，人类回到纯科技文明路线

## §8 关键人物与势力生态
