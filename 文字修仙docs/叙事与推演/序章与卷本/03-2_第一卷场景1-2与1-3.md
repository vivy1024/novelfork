### 合流文本

```

序章的 78% 提升到了 85% 左右。L_fit 在 43-47 之间波动——
取决于天气、灵材、和你前一天的睡眠质量。
```

### 功法学习

```

不管你在哪个阵营，这套功法的核心操作是一样的（来自 QESA

  │  第三段·守炼  守气不散 → 丹田初步蓄能      │
```

## 场景 1-3：灵材的价格 [M 主线共享]

> **叙事目标**：引入经济线 + 灵材选择困境 + B Story 角色相遇
> **天数**：Day 17
> **数值事件**：`money` 第一次消耗

### 主场景

```

无论你在哪个阵营，灵材都是消耗品。石英会在反复使用后出现
微裂纹（学院解释：晶格应力疲劳导致 d33 下降），萤石会
慢慢脱水变浑浊（古法说法：灵气被"吸干了"），社区那块
闲鱼石英原石已经被三个人轮着用了。

你需要买新材料了。

  📊 经济状态:
  ├ 月度可支配: ¥3,200
  ├ 房租: ¥1,500 (灵气区价格)
  ├ 伙食: ¥800
  ├ 日用: ¥400
  └ 修炼预算: ¥500

  📊 灵材价格 (终南隐谷外缘·灵启十一年春):
  ├ 低纯度石英原石 (d33≈2.0): ¥30-50/块 (可用10次)
  ├ 中纯度石英切片 (d33≈3.5): ¥200-400/块 (可用30次)
  ├ 高纯度石英晶体 (d33≈5.0): ¥2,000-5,000/块 (可用100次)
  ├ 天然萤石 (含杂质): ¥80-150/块 (可用20次, Poll +0.3/次)
  ├ 高纯萤石 (实验级): ¥1,500+/块 (可用50次, Poll≈0)
  └ 通灵云母片: ¥500-800/块 (土属性专用)

```

```
改的，老板姓马，以前卖中药，三个月前转型。

"高纯的我这里没有。" 马老板在柜台后面翘着腿。"那东西
QESA 管控，要证件。你有灵材购买许可吗？"

你没有。学院派的人有实验室配发，不用自己买。古法派的师父
有自己的渠道。开源社区——全靠自己。

"这个，" 马老板指了指角落里一筐灰白色的石头，"天然石英，
终南隐谷外缘本地采的。30 块一个，量大从优。纯度嘛——差点意思，
但对你们初学者够用了。"

这时候，门帘被掀开了。

一个你没见过的人走了进来。
```
