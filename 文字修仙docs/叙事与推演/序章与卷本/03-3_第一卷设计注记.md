## 设计注记

### 场景 1-1 → 1-3 验证了什么

| 系统 | 验证内容 |
|------|----------|
| 三阵营分支 | 1-1 的三个独占场景形式正确，各有独特氛围 |
| 数值面板 | AI 道侣报告格式与 [[01_序章脚本：灵气回归之日]] 一致，且旧版感气/引气标签已折叠进练气层级 |
| 选择→后果 | 每个选择都有变量变化，无纯装饰性选项 |
| 经济系统 | 1-3 首次引入资金消耗和灵材价格差异 |
| B Story | 1-3 根据阵营匹配不同的 B Story 角色 |
| 污染累积 | 古法路线 Poll 开始累积（0.3/次），学院 0，开源 0.1 |
| 走火风险 | 1-2 激进路线触发接近预警的 Risk 值 |

### 格式约定

- `[M]` = 主线共享场景
- `[F:阵营]` = 阵营独占场景
- `[J]` = 合流点
- `[C:条件]` = 条件触发场景
- `[IF faction == X]` = 文本变体（同一场景不同阵营不同描述）
- `📊` = 数值面板
- `📱` = 手机通知（信息补充 + 世界观展开）
- 选择效果用 `效果:` 标注全部变量变化
