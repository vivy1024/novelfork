# 01-4_经济冲突与参数接口

> 拆分自 [[01_经济系统设计]]，对应原文 §7-§8。

## §7 经济冲突与叙事钩子

### 7.1 灵材垄断

**冲突核心**：矿业集团控制上游灵材供给，通过限产提价获取超额利润。

**垄断的形成机制**：

灵材开采的垄断不是一夜之间形成的，而是经历了清晰的演化路径：

```
2025-2027：自由竞争期
  大量企业涌入灵材开采，价格混乱，品质参差不齐
    ↓
2027-2029：整合期
  QESA出台开采许可制度，小企业被淘汰或被收购
    ↓
2029-2031：寡头形成期
  三大矿业集团控制了60%以上的高灵区开采权
    ↓
2031-2033：垄断巩固期
  三大集团通过交叉持股和默契限产，形成事实上的卡特尔
    ↓
2033-2035：博弈期
  QESA开始反垄断调查，但进展缓慢；黑市和人工灵材开始挑战垄断
```

**垄断的具体表现**：

| 垄断行为 | 手段 | 影响 | 证据难度 |
|---------|------|------|---------|
| 联合限产 | 三大集团默契减产 10–15% | 灵石价格上涨 20–30% | 高（无书面协议） |
| 囤积居奇 | 在低价期大量收购，高价期缓慢释放 | 价格波动加剧 | 中（可追踪库存） |
| 排挤竞争 | 低价倾销打击小型矿企 | 市场集中度进一步上升 | 中（需证明低于成本） |
| 游说监管 | 推动提高开采许可门槛 | 新进入者被挡在门外 | 低（公开的政策游说） |
| 技术封锁 | 收购关键精炼技术专利 | 精炼环节也被控制 | 低（专利记录公开） |

| 角色 | 立场 | 行动 |
|------|------|------|
| 矿业集团 | "我们承担了开采风险和环保成本，高价是合理回报" | 联合限产、囤积居奇、游说 QESA 提高准入门槛 |
| 制药/法器企业 | "原料成本占比过高，挤压了我们的利润空间" | 寻找替代供应、投资人工灵材研发、向 QESA 投诉 |
| 散修/普通修士 | "修炼资源被垄断，穷人修不起仙" | 网络请愿、非法采集、转向黑市 |
| QESA | "维护市场秩序，但也不能打击企业积极性" | 反垄断调查（进展缓慢）、释放战略储备（杯水车薪） |
| 黑市 | "正规渠道买不到？我这里有" | 走私、盗采、仿制 |

**叙事场景**：

> 玩家发现自己常用的 B 级灵石突然涨价 40%。调查后发现，三大矿业集团秘密达成了限产协议。玩家可以选择：
> - 帮助 QESA 收集垄断证据（正义路线，但得罪矿业集团）
> - 帮助矿业集团维持垄断（利益路线，获得稳定低价供应）
> - 自己去黑市找替代货源（灰色路线，风险自担）
> - 投资人工灵材研发（长线路线，短期无收益但可能改变格局）

### 7.2 药价争议

**冲突核心**：救命丹药的定价伦理——类比现实中的天价药问题。

**药价争议的社会背景**：

2035 年，南瞻约有 3,000 万活跃修士（占总人口 2%），其中约 10% 在修炼过程中会遭遇需要丹药干预的健康问题。这意味着每年约 300 万人次需要修复类或急救类丹药——而这些丹药的价格往往超出普通修士的承受能力。

**"灵药医保"争论**：

QESA 于 2033 年提出将部分基础灵药纳入医保体系，引发激烈争论：

| 立场 | 支持者 | 核心论点 |
|------|--------|---------|
| 支持纳入 | 修士群体、公共卫生专家 | 修炼是合法活动，修炼导致的健康问题应获得社会保障 |
| 反对纳入 | 财政部门、传统医疗界 | 修炼是个人选择，不应由全体纳税人买单；财政负担过重 |
| 有条件纳入 | QESA 主流意见 | 只纳入基础缓冲剂和修复剂，激发剂和炼体剂不纳入 |
| 市场化方案 | 保险公司 | 开发商业"修炼保险"，由修士自愿购买 |

截至 2035 年，"灵药医保"仍在讨论中，尚未正式实施。但部分城市已开始试点"修炼健康保险"商业产品。

| 场景 | 药品 | 工业品价格 | 古法品价格 | 争议 |
|------|------|-----------|-----------|------|
| 走火入魔急救 | 镇魂散（修复+缓冲复合剂） | ¥5,000/剂 | ¥80,000/剂 | 工业品效果不够，古法品太贵 |
| 灵气污染排毒 | 净灵丸（排毒+修复） | ¥2,000/剂 | ¥30,000/剂 | 穷修长期使用低品质灵材导致污染，却买不起排毒药 |
| 筑基突破辅助 | 筑基丸（缓冲+修复） | ¥800/剂 | ¥15,000/剂 | 没有辅助药物的筑基成功率低 30%，但药价是普通修士月收入的数倍 |

**各方立场**：

| 角色 | 立场 |
|------|------|
| 制药企业 | "研发成本高、审批周期长、失败率高——高价是维持创新的必要条件" |
| 古法药堂 | "手工炮制耗时耗力，原料稀缺——价格反映了真实成本" |
| 患者/修士 | "修炼出了问题不是我的错，凭什么要我倾家荡产买药？" |
| QESA | "正在推动灵药纳入医保体系，但财政压力巨大" |
| 黑市药贩 | "我卖的便宜，虽然品质差点，但总比没有强" |

> **游戏设计**：药价争议是最容易引发玩家情感共鸣的经济冲突。当玩家的同伴因走火入魔需要天价丹药时，玩家面临的不是"打怪升级"的选择，而是"钱从哪来"的现实问题。

### 7.3 黑市恶性循环

```
QESA管制收紧
    ↓
正规渠道供给不足 + 价格上涨
    ↓
修士转向黑市购买
    ↓
黑市规模扩大，竞争加剧
    ↓
黑市商家为降低成本，偷工减料
    ↓
黑丹药品质下降，事故频发
    ↓
舆论压力 → QESA进一步收紧管制
    ↓
（循环加剧）
```

**黑市的内部生态**：

黑市不是铁板一块，而是由多个层级和角色组成的复杂生态系统：

| 层级 | 角色 | 功能 | 利润分配 |
|------|------|------|---------|
| 顶层 | 黑市"教父"/组织头目 | 战略决策、资源调配、关系维护 | 30–40% |
| 中层 | 区域经理/渠道负责人 | 管理分销网络、质量控制（有限的） | 20–30% |
| 执行层 | 走私者/制造者/销售者 | 实际操作——运输、生产、交易 | 15–25% |
| 底层 | 线人/跑腿/望风 | 信息收集、小额交易、安全警戒 | 5–10% |
| 外围 | 腐败官员/企业暗线 | 提供保护伞、内部信息、合法掩护 | 10–20% |

**黑市的"品牌化"趋势**：

讽刺的是，部分黑市组织开始建立自己的"品牌"和"信誉体系"：

| "品牌" | 特色 | 信誉 | 风险 |
|--------|------|------|------|
| "昆仑影"（虚构） | 高端灵材走私，品质有保证 | 高 | 价格高，但品质可靠 |
| "百草堂"（虚构） | 黑丹药，仿制正规产品 | 中 | 品质波动大 |
| "古器斋"（虚构） | 古代法器和古籍盗卖 | 中高 | 来源不明，可能有诅咒 |
| "无名氏"（虚构） | 低端黑市，什么都卖 | 低 | 假货率高，安全风险大 |

> **游戏设计**：黑市的"品牌化"为玩家提供了有趣的选择——在黑市中也有"高端"和"低端"之分。选择"昆仑影"意味着更高的价格但更可靠的品质，选择"无名氏"意味着更低的价格但更高的风险。

**黑市事故的连锁反应**：

| 事故类型 | 频率 | 后果 | 经济影响 |
|---------|------|------|---------|
| 黑丹药中毒 | 月均 200–500 起 | 轻则修炼受损，重则走火入魔 | 医疗费用+修炼损失 |
| 假灵材欺诈 | 月均 1,000+ 起 | 经济损失+信任崩塌 | 市场信心下降 |
| 走私被截获 | 月均 50–100 起 | 货物没收+人员拘留 | 黑市供给波动 |
| 黑市火并 | 季均 10–20 起 | 人员伤亡+区域混乱 | 局部市场瘫痪 |

### 7.4 修士阶层固化

**核心矛盾**：修炼资源集中导致"强者越强、弱者越弱"的马太效应。

| 阶层 | 月修炼预算 | 使用灵材等级 | 修炼效率 | 突破概率 | 占修士总数 |
|------|-----------|-------------|---------|---------|-----------|
| 顶层（企业高管/宗门核心） | ¥100,000+ | S/A 级 | 极高 | 高 | 2–3% |
| 上层（中层管理/宗门弟子） | ¥30,000–100,000 | A/B 级 | 高 | 中高 | 8–12% |
| 中层（普通白领修士） | ¥8,000–30,000 | B 级 | 中等 | 中等 | 25–30% |
| 下层（蓝领修士/散修） | ¥2,000–8,000 | B/C 级 | 低 | 低 | 35–40% |
| 底层（贫困修士） | <¥2,000 | C/D 级 | 极低 | 极低 | 20–25% |

**各阶层的典型生活画像**：

**顶层修士**（占 2–3%）：
- 住在高灵区附近的豪华修炼别墅，配备定制聚灵阵
- 使用 S/A 级灵材，由专属炼丹师配制个人化丹药
- 拥有多件手工定制法器，定期由法器师保养
- 修炼指导来自宗门长老或顶级学院派教授
- 走火入魔风险极低（资源充足，可以从容修炼）
- 社交圈：企业高管、宗门核心、QESA 高层、军方将领

**中层修士**（占 25–30%）：
- 住在城市，在公共修炼室或家中简易修炼角修炼
- 使用 B 级灵材，服用工业化标准丹药
- 拥有 1–2 件工业法器，偶尔升级
- 修炼指导来自网络课程或持证修炼师
- 走火入魔风险中等（资源有限，偶尔需要冒险加速）
- 社交圈：同事、修炼社群、网络论坛

**底层修士**（占 20–25%）：
- 住在城市边缘或农村，在野外或简陋环境中修炼
- 使用 C/D 级灵材，偶尔从黑市购买来路不明的丹药
- 法器简陋或没有，依赖自身体质
- 修炼指导来自网络免费资源或散修口传
- 走火入魔风险高（资源匮乏，被迫使用低品质材料）
- 社交圈：散修群体、黑市边缘、底层劳动者

**阶层流动的可能路径**：

| 路径 | 从→到 | 概率 | 条件 |
|------|-------|------|------|
| 天赋突破 | 底层→中层 | 5–10% | 灵根适配度极高，被发现并资助 |
| 宗门收徒 | 下层→上层 | 2–5% | 被古法宗门看中，获得师承 |
| 创业成功 | 中层→顶层 | 1–3% | 灵材交易或技术创业成功 |
| 意外发现 | 任意→上升 | <1% | 发现未知灵材矿脉或古代遗迹 |
| AI 辅助优化 | 下层→中层 | 10–15% | 利用 AI 找到最优修炼方案，弥补资源不足 |
| 阶层下滑 | 任意→下降 | 5–15% | 走火入魔、投资失败、健康问题 |

> **游戏设计**：阶层流动的低概率但非零概率，是游戏的核心驱动力。玩家的目标不是"打败最终 Boss"，而是在经济和修炼的双重约束下，找到属于自己的上升路径。AI 辅助优化（10–15%）是"代际加速"设定在经济层面的体现——技术进步正在缓慢打破阶层壁垒。

**阶层固化的机制**：

```
高阶修士 → 高收入 → 买得起好灵材 → 修炼效率高 → 更快突破 → 更高收入
    ↑                                                          │
    └──────────────────────────────────────────────────────────┘

低阶修士 → 低收入 → 只能用差灵材 → 修炼效率低 → 难以突破 → 收入不涨
    ↑                                                          │
    └──────────────────────────────────────────────────────────┘
```

> **设计意图**：阶层固化是"人人可仙"理想与"资源有限"现实之间的根本矛盾。这个矛盾驱动了 [[01_势力机构与学派冲突]] 中"修炼自由 vs 持证上岗"的政治争论，也是 [[02_意识形态对抗深度设计]] 中多条意识形态路线的经济根源。

### 7.5 可直接用于叙事的经济冲突事件

| 事件 | 涉及势力 | 冲突类型 | 叙事潜力 |
|------|---------|---------|---------|
| **"昆仑玉髓"价格操纵案** | 矿业集团 vs QESA | 垄断 | 玩家可作为调查员或内部人 |
| **"QR-17 召回事件"** | 制药企业 vs 消费者 | 产品安全 | 发现工业丹药的隐藏副作用 |
| **"终南山采矿权争夺"** | 矿业集团 vs 龙门系 | 资源争夺 | 古法宗门的修炼圣地被商业开发威胁 |
| **"黑丹药连环中毒案"** | 黑市 vs QESA vs 受害者 | 公共安全 | 追查黑丹药源头，发现背后是正规企业的影子 |
| **"灵材期货崩盘"** | 投机者 vs 市场 | 金融危机 | 某大型投机者做空灵石期货，引发市场恐慌 |
| **"散修联盟罢工"** | 散修 vs 矿业集团 | 劳资冲突 | 灵材采集工人要求提高分成比例 |
| **"跨境走私大案"** | 走私网络 vs 多国执法 | 国际犯罪 | 玩家被卷入一条从昆仑到中东的走私链 |
| **"灵药专利战"** | 恒灵药业 vs 茅山系 | 知识产权 | 古方配方的归属权之争 |
| **"高灵区环保危机"** | 矿业集团 vs 环保组织 vs 当地居民 | 环境 | 过度开采导致灵气环境恶化 |
| **"修士医保争议"** | 修士群体 vs 政府 vs 保险公司 | 社会保障 | 走火入魔的治疗费用该谁出？ |

**详细叙事场景展开**：

**场景 A："昆仑玉髓"价格操纵案**

| 要素 | 内容 |
|------|------|
| 背景 | 昆仑矿业宣布"设备检修"停产两周，实际是联合限产，灵石价格一周内暴涨 25% |
| 主要角色 | 矿业集团高管、QESA 反垄断调查员、受影响的中小药企、散修群体 |
| 核心冲突 | 调查员发现限产是人为操纵，但证据链不完整；矿业集团通过游说施压 |
| 玩家介入点 | 被雇佣收集证据、被矿业集团收买销毁证据、作为记者报道事件 |
| 结局分支 | (A) 调查成功，矿业集团被罚但换马甲继续；(B) 调查失败，灵石价格居高不下；(C) 玩家找到替代供应源 |

**场景 B："黑丹药连环中毒案"**

| 要素 | 内容 |
|------|------|
| 背景 | 某城市一周内 47 名修士因服用仿制"清心丹"中毒，3 人死亡 |
| 主要角色 | 中毒修士家属、黑市药贩"百草堂"、QESA 执法队、恒灵药业公关部、医院灵药科 |
| 核心冲突 | 中毒修士买不起正规药（¥3,000/剂），被迫使用黑市仿制品（¥500/剂）；恒灵药业趁机涨价 |
| 玩家介入点 | 追查毒丹药源头、帮助中毒修士获得救治、揭露恒灵药业的趁火打劫 |
| 结局分支 | (A) 捣毁黑市工坊，但底层修士更难获药；(B) 推动"灵药医保"试点；(C) 发现恒灵药业暗中资助黑市制造恐慌 |

**场景 C："终南山采矿权争夺"**

| 要素 | 内容 |
|------|------|
| 背景 | 信灵科技获得终南山某区域的灵材开采许可，该区域是龙门系百年修炼场地 |
| 主要角色 | 信灵科技项目经理、龙门系长老、QESA 资源管理中心、当地村民、环保组织 |
| 核心冲突 | 开采许可合法但不合情——龙门系在此修炼百年但从未申请产权；开采会破坏修炼环境 |
| 玩家介入点 | 帮助龙门系维权、帮助信灵科技推进项目、寻找双方都能接受的妥协方案 |
| 结局分支 | (A) 龙门系胜诉，但树立了"宗门可以阻止合法开采"的先例；(B) 信灵科技开采，龙门系被迫迁移；(C) 双方合作，龙门系获得分成 |

> **游戏设计**：每个场景都设计了多个介入点和结局分支，确保不同类型的玩家（战斗型、社交型、调查型、商人型）都能找到参与方式。场景之间可以串联——例如"黑丹药中毒案"可能与"灵石价格操纵"有因果关系（灵石涨价→正规药涨价→修士转向黑市→中毒事件）。

> **游戏设计**：每个事件都可以展开为一条支线任务或主线剧情的一部分。关键是让玩家感受到经济冲突的"真实感"——这些不是抽象的数字游戏，而是影响到具体人物命运的利益博弈。

### 7.6 经济危机场景

以下是可能在游戏中触发的系统性经济危机：

#### 7.6.1 "灵石荒"——供给侧危机

**触发条件**：昆仑矿区因地质灾害停产 + 长白山因中朝关系紧张限制开采

```
S/A级灵材供给骤降 30%
    ↓
SQEX灵石价格一周内飙升 60%
    ↓
制药企业原料断供，丹药价格翻倍
    ↓
散修群体恐慌性囤积，加剧短缺
    ↓
黑市灵材价格暴涨 300%
    ↓
QESA释放战略储备，但仅能覆盖需求的 20%
    ↓
社会层面：修炼中断、走火入魔事故激增、民众恐慌
```

**玩家可参与的角色**：
- 危机调查员：查明供给中断的真实原因（天灾还是人祸？）
- 物资调配者：帮助 QESA 分配有限的储备资源
- 黑市商人：趁机囤积居奇（高利润但道德风险）
- 替代方案开发者：加速人工灵材研发

#### 7.6.2 "灵材泡沫"——金融危机

**触发条件**：投机资本过度涌入灵材期货市场

```
灵材期货价格被炒至实际价值的 3 倍
    ↓
大量散户跟风买入
    ↓
某大型投机者突然平仓，价格崩盘
    ↓
期货市场连锁爆仓，SQEX 触发熔断
    ↓
现货市场恐慌性抛售
    ↓
矿业集团股价暴跌，部分企业面临破产
    ↓
上下游产业链连锁反应
```

#### 7.6.3 "黑丹药瘟疫"——公共卫生危机

**触发条件**：某批黑市丹药含有未知毒性成分，导致大规模中毒

```
数百名修士同时出现异常症状
    ↓
QESA紧急排查，发现源头是同一批黑市丹药
    ↓
公众对所有丹药（包括正规产品）信心崩塌
    ↓
丹药销售额暴跌 40%
    ↓
制药企业被迫召回产品、加强检测
    ↓
QESA出台更严格的管制措施
    ↓
黑市短期萎缩，但很快以更隐蔽的方式恢复
```

### 7.7 经济系统与修炼体系的交叉影响

经济状况直接影响修炼进度，修炼进度又反过来影响经济能力：

| 修炼阶段 | 月均修炼开支 | 主要经济活动 | 经济瓶颈 |
|---------|------------|-------------|---------|
| 胎息期 | ¥2,000–5,000 | 购买入门灵材+修炼指导 | 初始资金不足 |
| 练气期 | ¥5,000–15,000 | 稳定灵材消耗+基础法器 | 收入尚未跟上消耗 |
| 筑基期 | ¥15,000–40,000 | 中阶灵材+丹药辅助+场地租赁 | 需要稳定收入来源 |
| 紫府期 | ¥40,000–100,000 | 高阶灵材+定制丹药+专业法器 | 资源获取渠道成为关键 |

| 道胎期 | 不可估量 | 极稀缺资源+独特条件 | 经济因素退居次要，条件匹配成为关键 |

> **游戏设计**：修炼阶段与经济开支的对应关系，确保玩家在每个阶段都面临有意义的经济决策。低阶时是"省钱还是投资"，中阶时是"稳定还是冒险"，高阶时是"独占还是分享"。

### 8.4 Match 评分到市场价格的映射接口

本节定义 [[02_物理公式与参数手册]] 中的 Match_A/B/C 评分如何映射到市场价格，为经济系统提供量化基础。

#### 8.4.1 灵材价格映射（Match_A → 价格）

```
Price_material(Match_A) = P_base × exp(k_price × (Match_A - 50))
```

| 参数 | 默认值 | 调节方向 |
|------|--------|---------|
| P_base（基准价，¥/g） | 800 | ↑ 整体抬升灵材价格水平 |
| k_price（指数系数） | 0.06 | ↑ 加大高低等级间的价差 |

**验证**：
- Match_A = 20（D 级）：800 × exp(0.06 × -30) ≈ ¥13/g ✓
- Match_A = 40（B 级下限）：800 × exp(0.06 × -10) ≈ ¥439/g ✓
- Match_A = 60（A 级下限）：800 × exp(0.06 × 10) ≈ ¥1,458/g ✓
- Match_A = 80（S 级下限）：800 × exp(0.06 × 30) ≈ ¥4,844/g ✓
- Match_A = 95（S 级高端）：800 × exp(0.06 × 45) ≈ ¥11,802/g ✓

> 实际市场价格在此基础上叠加 §4.2 中的修正因子（属性稀缺度、产地认证、形态等）。

#### 8.4.2 高灵区经济价值映射（Match_B → 经济产出）

```
EconValue(Match_B) = V_base × (Match_B / 50)^α_econ
```

| 参数 | 默认值 | 调节方向 |
|------|--------|---------|
| V_base（基准年产值，亿¥） | 10 | ↑ 整体抬升高灵区经济价值 |
| α_econ（幂指数） | 2.5 | ↑ 加大高低等级间的产值差距 |

**验证**：
- Match_B = 30（低灵区）：10 × (30/50)^2.5 ≈ ¥2.8亿/年 ✓
- Match_B = 50（中灵区）：10 × (50/50)^2.5 ≈ ¥10亿/年 ✓
- Match_B = 72（高灵区，终南山）：10 × (72/50)^2.5 ≈ ¥28亿/年 ✓
- Match_B = 93（洞天候选，昆仑）：10 × (93/50)^2.5 ≈ ¥64亿/年 ✓

#### 8.4.3 丹药价格映射（Match_C → 价格）

```
Price_elixir(Match_C) = E_base × exp(k_elixir × (Match_C - 45)) × CraftMultiplier
```

| 参数 | 默认值 | 调节方向 |
|------|--------|---------|
| E_base（基准价，¥/剂） | 200 | ↑ 整体抬升丹药价格水平 |
| k_elixir（指数系数） | 0.08 | ↑ 加大高低品质间的价差 |
| CraftMultiplier（工艺倍率） | 工业品=1.0，古法品=10–100 | ↑ 加大古法溢价 |

**验证**（工业品，CraftMultiplier=1.0）：
- Match_C = 30（低品质）：200 × exp(0.08 × -15) ≈ ¥60/剂 ✓
- Match_C = 45（中品质）：200 × exp(0) = ¥200/剂 ✓
- Match_C = 60（高品质）：200 × exp(0.08 × 15) ≈ ¥664/剂 ✓
- Match_C = 75（极高品质）：200 × exp(0.08 × 30) ≈ ¥2,208/剂 ✓

#### 8.4.4 法器价格映射（ArtifactScore → 价格）

```
Price_artifact(Score) = A_base × exp(k_artifact × (Score - 30)) × CraftMultiplier
```

| 参数 | 默认值 | 调节方向 |
|------|--------|---------|

| 灵材战略储备释放阈值 | §4.5 | ¥1,100/SQU | 决定政府干预市场的触发点 |
| 灵材战略储备增储阈值 | §4.5 | ¥650/SQU | 决定政府增储的触发点 |
| 黑市溢价系数 | §3.6 | 2–5× | 决定黑市相对正规市场的价格倍数 |
| 运输灵气衰减率 | §6.1 | 2–8%/全程 | 决定物流成本和区域价差 |

> **设计原则**：所有价格参数都是可调旋钮，不是世界观 canon。调整这些参数可以改变经济系统的"松紧度"——提高 k_price 会加剧贫富分化，降低 CraftMultiplier 会削弱古法派的经济优势。具体数值应在游戏测试中根据玩家体验反馈迭代调整。

### 8.10 未来扩展方向

本文档的 v1.0 版本覆盖了经济系统的核心框架。以下方向可在后续版本中扩展：

| 方向 | 优先级 | 说明 |
|------|--------|------|
| 灵气税收制度详细设计 | 高 | 各类灵气税的税率、征收方式、减免政策 |
| 灵气保险精算模型 | 中 | 基于 Risk 函数的保险定价公式 |
| 国际灵气贸易协定 | 中 | IVCERA 框架下的多边贸易规则 |
| 灵气金融监管体系 | 中 | SQEX 的交易规则、熔断机制、反操纵措施 |
| 灵气劳动法 | 低 | 修士雇佣的法律框架、工伤认定、社保 |
| 灵气环保经济学 | 低 | 灵气污染的外部性定价、碳交易类比 |
| 灵气教育经济学 | 低 | 修炼培训市场的供需分析 |

> **设计原则**：经济系统的扩展应遵循"需求驱动"原则——只有当叙事或游戏机制需要时，才扩展相应的经济子系统。避免为了"完整性"而过度设计。
