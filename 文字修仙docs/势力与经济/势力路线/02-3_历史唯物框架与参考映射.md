# 02-3_历史唯物框架与参考映射

> 拆分自 [[02_意识形态对抗深度设计]]。  
> **阅读目标**：给路线争论一个底层分析工具，避免把复杂世界写成抽象站队。

---

## §1 固定的不是事件顺序，而是时代基本矛盾

在本项目里，真正固定的不是“2035 一定发生 A，2045 一定发生 B”，而是：

- 灵气回升与现代工业第一次同时出现
- 高阶突破需要组织、材料、医疗、算力与心性共同支持
- 一旦一方加速，其他方就被迫加速
- 高阶失控会反过来改写政治、伦理和分配结构

因此，历史唯物主义在这里不是削弱个体，而是提醒：

> 关键人物之所以重要，是因为他们站在结构张力的交叉点上。  
> 没有结构条件，个人无法凭空造历史；  
> 但没有个人与组织的选择，结构也不会自动实现。

---

## §2 五类核心变量

分析任何世界节点时，至少同时看这五类变量：

| 变量 | 内容 |
|------|------|
| **结构变量** | 资源、人口、交通、灵气浓度、材料体系、生产力水平 |
| **组织变量** | 国家、军工、平台、宗门、社区的动员、纪律与财政能力 |
| **人物变量** | 关键人物的权威、判断、健康、性格、关系网 |
| **思想谱系变量** | 某条路线是谁提出、谁继承、谁验证、谁传播 |
| **偶然变量** | 战败、失控、天灾、实验成功、暗杀、奇遇等节奏改写因素 |

这五类变量叠加后，才决定一个节点会不会发生、由谁主导、造成什么后果。

## §3 六大问题仍然有效

[[01-5_历史唯物主义六大问题]] 仍然是最重要的检查表。  
可以把它简化成六句直接问法：

1. 谁控制灵气生产资料？
2. 谁有资格进入修炼体系？
3. 修士被怎样组织成劳动者、士兵、研究者或工具？
4. 事故发生时谁负责、谁买单？
5. 穷修与富修的成本差异体现在哪？
6. 一条路线凭什么被说成正统？

任何势力和意识形态，如果不能回答这六个问题，就只是空口号。

## §4 参考作品该借什么

这些参考作品借的是社会结构方法，不是剧情。

| 参考作品 | 最值得借的东西 | 落点 |
|---------|---------------|------|
| **《没钱修什么仙？》** | 修炼如何被债务、教育、企业和宗门垂直组织 | 写平台资本、底层修士成本和生产资料垄断 |
| **《黎明之剑》** | 超凡如何基础设施化、公共化、教育化 | 写公共灵网、普筛、公共功法教育与生产力释放 |
| **《魔神乐园》** | 超凡等级如何嵌入选拔制度、学院和国家机器 | 写资格筛选与上升通道 |
| **《食物链顶端的男人》** | 生产力革命、信息分层治理、思想控制风险 | 写 AI 军备竞赛、国家能力与超凡独裁风险 |

### 4.1 最需要守住的边界

- 不能把《黎明之剑》借成“伟人替全社会做完正确决定”
- 不能把《食物链》借成“只要效率高，思想托管也没问题”
- 不能把《没钱修什么仙？》借成单纯黑暗夸张
- 不能把《魔神乐园》借成超凡特权天然高于现代制度

## §5 节点设计如何真正历史化

一个重要节点，不该写成简单分支：

- 选 A，世界线 A
- 选 B，世界线 B

更好的写法是写成“受力点”：

| 要素 | 含义 |
|------|------|
| **必要条件** | 缺一个就不会发生 |
| **促进条件** | 越多越容易发生 |
| **阻碍条件** | 越强越难发生 |
| **关键人物选择** | 改变权重，不单独决定结果 |
| **默认结局** | 玩家不介入时，世界按现有力量对比推进 |
| **后续波及** | 会改变哪些资源、组织、思想与下一批节点 |

这样世界才会真的“活”，而不是作者在后台手工排剧情树。

---

## §6 本篇的使用方式

- 写势力时，用本篇校正自己是不是在写结构，而不是写标签。
- 写世界节点时，用本篇判断哪些变量必须同时出现。
- 写路线斗争时，用本篇提醒自己：没有哪条路线是纯道德争论，它们都绑定着具体的生产资料与事故责任。
