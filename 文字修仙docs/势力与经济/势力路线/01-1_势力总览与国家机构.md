## §1 判断势力的三个原则

### 1.1 势力不是地图颜色

本项目中的势力，不是几块颜色不同的领土。

- `八洲` 决定文明区与资源格局。
- `组织主体` 决定谁控制训练、矿带、灵药、灵芯、军工与突破项目。
- `路线` 决定他们如何解释正统性、效率与风险。

所以，势力分析首先看“组织功能”，不先看“地理边界”。

### 1.2 真正要争夺的不是秘籍，而是三段跃迁

| 跃迁 | 时间窗口 | 谁最有优势 | 关键生产资料 |
|------|---------|-----------|-------------|
| **练气标准化** | 2035 前后 | 国家-学院复合体、平台资本 | 训练数据、基础功法、公共高灵区、低中阶灵材 |
| **筑基工程化** | 2045 前后 | 国家-学院复合体、古法改革派、算力军工轴 | 定制灵材、闭环医疗、AI 建模、场域实验设施 |
| **紫府国家工程化** | 2050+ | 洲级军工系统、最高级 AI 体系、极少数古法心性线路 | 战略矿带、意识迁移模型、军用高灵闭环区 |

### 1.3 旧稿迁移说明

旧稿里的 `QESA`、现实国家机关和若干现实国家对照，当前都视为过渡期写法。  
在新的八洲口径下，默认替换成：

- `八洲灵能安全协约会`：洲际协调与最低共识平台
- `各洲灵能工程总署/总院/总局`：洲内国家-学院复合体
- `裂地玄洲矿带联席会`：围绕矿带和停火协议临时形成的战区协调机构

## §2 六类主要组织主体

| 主体 | 主要文明区 | 最擅长的阶段 | 核心资产 | 最大弱点 |
|------|-----------|-------------|---------|---------|
| **国家-学院复合体** | 南瞻、西牛、西极 | 练气标准化、稳态筑基 | 标准、医院、实验区、公共高灵区、海量训练数据 | 突破时过于保守，容易被军工和平台反超 |
| **算力-军工复合体** | 隔海、东胜、北俱 | 筑基加速、紫府项目 | 算力、灵芯、军方预算、战略实验区 | 对心性和长期社会成本不敏感 |
| **平台-资本复合体** | 南瞻、隔海、东胜 | 练气扩张、消费化修炼 | 用户入口、付费服务、数据壁垒、分发渠道 | 容易把修炼变成付费阶梯 |
| **古法宗门保守派** | 南瞻、西牛、北俱、南溟 | 心性校正、场域识别、旧法脉守护 | 身体经验、地脉经验、师承纪律、风险诚实 | 规模化极差，天然排外 |
| **古法宗门改革派** | 南瞻、西牛、西极 | 筑基协同、紫府风险校正 | 旧法经验 + 现代实验合作能力 | 两边都不信任，常被夹击 |
| **散修-开源网络与灰色体系** | 各洲都有，裂地最密 | 边缘创新、异常样本、黑市流通、战区承包 | 灵活、低成本、对异常环境适应快 | 事故率高、组织脆弱、易被收编或渗透 |

## §3 公权机构与洲际结构

### 3.1 八洲灵能安全协约会

| 项目 | 定位 |
|------|------|
| **性质** | 洲际技术-安全协商平台 |
| **权力类型** | 有标准权、认证权、协调权，执行力弱 |
| **能做什么** | 发布最低安全协议、共享事故分类、跨洲古器与战区灵材认证 |
| **做不到什么** | 强制干预各洲的紫府项目与军工计划 |

它的存在意义，不是维持和平，而是让世界至少有一套“吵架时还能引用的共同语言”。

### 3.2 各洲国家-学院复合体

| 文明区 | 代表性组织形式 | 强项 | 典型短板 |
|------|---------------|------|---------|
| **南瞻部洲** | 灵能工程总署 + 学院网络 + 公共高灵区体系 | 修炼道统源头垄断、规模、工业底盘、长期项目治理 | 早期高端灵芯受外部卡脖子，AI 算力曾落后 |
| **西牛贺洲** | 药脉总院 + 高原实验带 + 心性学校 | 灵药与身心训练、低成本普及 | 高端材料与算力不足，修炼体系依赖南瞻输出 |
| **北俱芦洲** | 深寒防务院 + 极寒试验场 | 军工、重装、极端环境稳定性 | 民用扩散能力弱，修炼知识依赖南瞻输出 |
| **西极金洲** | 高灵认证庭 + 标准院系网络 | 认证、协议、跨洲制度化接口 | 量产和战区投送能力弱，修炼根基依赖南瞻输出 |
| **东胜神洲** | 灵芯工艺联盟 + 近海实验段 | 精密器件、灵芯、微阵列 | 资源纵深不足，修炼体系依赖南瞻输出 |
| **隔海鹰洲** | 高能计划局 + 算力军研网 | AI算力优势（修炼算法根基仍依赖南瞻数据）、军工、紫府快速立项 | 社会撕裂和商业化过深，修炼道统无独立源头 |

### 3.3 裂地玄洲矿带联席会

这是一个典型的“名义协调、实为互相监视”的战区机构。

| 功能 | 实际情况 |
|------|---------|
| 战区灵材开采配额 | 名义共享，实际上谁有驻军谁说了算 |
| 高灵事故通报 | 只在失控外溢时才会被动共享 |
| 停火与撤军监督 | 形式上存在，执行力极弱 |
| 战区医学与失控隔离 | 只有当损失已不可承受时，各方才勉强合作 |

它的重要性不在于有效，而在于它证明了：  
各洲明知彼此互不信任，也仍被迫为高阶失控保留最低限度的共同接口。

## §4 各主体真正想要什么

| 主体 | 公开说法 | 真正要抢的东西 |
|------|---------|---------------|
| 国家-学院复合体 | 安全、普及、秩序 | 第一条可控、可复制、可军民两用的高阶路线 |
| 算力-军工复合体 | 威慑、突破、领先 | 紫府与战区场结构的战略先发权 |
| 平台-资本复合体 | 人人可修、服务升级 | 用户入口、训练数据和付费依赖 |
| 古法宗门保守派 | 守护传承、护住心性 | 场域解释权与高阶正统性 |
| 古法宗门改革派 | 古今合流、救亡图存 | 把旧法经验变成新时代不可替代的关键接口 |
| 散修-开源与灰色体系 | 生存、自由、低门槛 | 试错权、异常知识与非官方流通权 |
