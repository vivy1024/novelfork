# 01-2_企业宗门与民间力量

> 拆分自 [[01_势力机构与学派冲突]]。  
> **阅读目标**：看清平台资本、古法宗门保守派、古法宗门改革派和民间力量各靠什么生存，又为什么注定互相需要。

---

## §1 平台-资本复合体

### 1.1 他们不是单纯“企业”

平台资本在本世界里真正掌握的是三层入口：

| 层级 | 掌握内容 | 对修炼者意味着什么 |
|------|---------|------------------|
| **消费入口** | 练气课程、灵药订阅、场地预约、社交网络 | 修炼被日常化、服务化 |
| **数据入口** | 体征、失败记录、功法偏好、风险曲线 | 个体成为训练数据源 |
| **算法入口** | 推荐路径、资源排位、风险阈值 | 修炼被隐性调度 |

### 1.2 他们在时代中的历史作用

| 阶段 | 他们做对了什么 | 他们制造了什么问题 |
|------|---------------|------------------|
| **2035 前后** | 把练气服务规模化，降低普通人的首次进入门槛 | 让修炼入口迅速被订阅制和信用评分接管 |
| **2040-2045** | 提供大量练气到筑基的行为数据 | 数据私有化，训练成果不回流社会 |
| **2045-2050** | 为筑基工程和紫府项目提供最灵活的算法人才与器件采购 | 商业目标与军工目标高度耦合，推动加速主义 |

### 1.3 平台资本最常见的四种组织形态

| 形态 | 主营 | 在世界中的作用 |
|------|------|---------------|
| **修炼服务平台** | 训练、预约、社交、订阅 | 把修炼变成可分发服务 |
| **灵药与健康公司** | 缓冲剂、诱导剂、修复剂、监测方案 | 把风险管理商品化 |
| **灵芯与微阵列公司** | 灵芯片、微型法器、控制模块 | 给 AI 和高阶项目提供硬件底座 |
| **矿带金融与物流公司** | 期货、保险、跨洲运输、战区承保 | 把裂地玄洲等前沿战区纳入资本循环 |

### 1.4 他们真正害怕什么

- 基础修炼被彻底公共化
- 训练数据变成公共资源
- 古法改革派与国家-学院复合体达成稳定合作
- 裂地玄洲长期战乱导致供应链失控

## §2 古法宗门保守派

### 2.1 他们保留的不是“完整高阶答案”

在新的世界背景下，古法宗门不能再被写成“已掌握成熟紫府与金丹道路的旧世界统治者”。  
他们真正的价值是：

| 资产 | 含义 |
|------|------|
| **身体经验** | 知道哪些训练顺序能让肉身不先崩 |
| **场域经验** | 知道哪些山、洞、矿、时段是真的可用 |
| **风险经验** | 对走火、偏执、感知污染和超忆症更敏感 |
| **心性技术** | 戒律、磨炼、关系结构和自我约束，不是玄学装饰，而是高阶前置条件 |

### 2.2 他们为什么仍然重要

因为现代体系虽然能在 `2035` 制造练气、在 `2045` 稳定筑基，但它在两个地方天然薄弱：

- 对异常个体和边缘案例的解释力不足
- 对高阶心性、权力异化和长期人格稳定的处理能力不足

这正是古法保守派最强的部分。

### 2.3 他们最大的历史局限

| 局限 | 具体表现 |
|------|---------|
| **天然排外** | 只愿意把最深的知识交给极少数弟子 |
| **组织规模太小** | 无法承担文明级普及任务 |
| **容易把经验神圣化** | 把阶段性正确经验误当永恒真理 |
| **容易被时代抛下** | 若拒绝现代材料、医学与 AI，最终只剩守山与守旧 |

## §3 古法宗门改革派

### 3.1 他们是新时代最关键的桥

古法改革派不是“向现代投降的宗门”，而是最早承认下面这个事实的人：

> 古代留下的是真实但不完整的经验。  
> 现代拿到的是强大的工业与计算工具。  
> 两者不结合，就都到不了真正的高阶。

### 3.2 他们的核心主张

| 主张 | 解释 |
|------|------|
| **基础知识可公开** | 低阶安全协议和基础功法不应被继续神秘化 |
| **高阶突破必须校正心性** | 紫府不是纯工程问题 |
| **古法经验要进入可验证流程** | 不是丢掉师承，而是把师承变成可传递的工程接口 |
| **宗门要从神秘垄断者变成高价值合作者** | 宗门的未来不是闭门称尊，而是成为高阶校正与场域知识的关键节点 |

### 3.3 他们在世界中的战略位置

- 对国家-学院复合体来说，他们是高阶校正的稀缺合作者。
- 对平台资本来说，他们是难以被完全平台化的知识源头。
- 对保守派来说，他们是叛徒。
- 对开源网络来说，他们又是最有可能打开第一道高墙的人。

## §4 企业、宗门与学院为什么互相离不开

| 组合 | 合作原因 | 冲突原因 |
|------|---------|---------|
| **平台资本 ↔ 国家-学院** | 平台有数据和体验，国家有标准和合法性 | 谁来定义训练协议与事故责任 |
| **平台资本 ↔ 改革宗门** | 平台想吃到宗门经验红利，改革宗门想扩大影响 | 平台想私有化数据，宗门不愿被降格为内容供应商 |
| **国家-学院 ↔ 改革宗门** | 一个有规模，一个有高阶校正能力 | 谁掌握最终解释权 |
| **国家-学院 ↔ 保守宗门** | 国家需要他们的风险经验 | 宗门拒绝成为制度配件 |
| **平台资本 ↔ 保守宗门** | 平台想买独家内容与高阶背书 | 宗门警惕被商品化和去神圣化 |

## §5 民间力量的真实位置

“民间”不是浪漫背景板，而是整个系统里最早感到压力、也最早发明土办法的人群。

| 力量 | 他们提供什么 | 他们承担什么 |
|------|-------------|-------------|
| **散修个体** | 异常样本、非标准路径、边缘经验 | 最高事故率和最弱保障 |
| **社区互助组织** | 低成本知识扩散、共享灵材、事故互救 | 最容易被判定为非法设施或灰区实验 |
| **民间作坊与半合法供应商** | 低价材料和灵药 | 质量波动、法律风险和事故责任甩锅 |

民间力量不是决定终局的主角，但它们决定了：

- 世界是否还有不被平台、国家和宗门完全垄断的空间
- 新路线能否先在边缘长出来
- 高阶社会是否还记得普通人如何活着
