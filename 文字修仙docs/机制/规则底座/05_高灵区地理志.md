## 1. 判定方法说明

### 1.1 数据来源

| 数据类型 | 来源 | 用途 |
|----------|------|------|
| 地质构造与断裂带 | 南瞻地质调查局、USGS | Fault 评分 |
| 矿脉与矿产分布 | 矿产资源数据库、地质图 | Ore 评分 |
| 地磁异常 | 国际地磁参考场(IGRF)、区域地磁图 | Magnet 评分 |
| 水系与地下水 | 水文地质图、泉域分布 | Water 评分 |
| 地热与热流 | 地热资源调查数据 | Thermal 评分 |
| 人口与工业 | 统计年鉴、遥感数据 | NoiseCore 评分 |
| 古代记载 | 道教志、地方志、考古报告 | HistBonus |

### 1.2 评分公式（简化版）

```
GeoCore = β₁·Ore + β₂·Fault + β₃·Magnet + β₄·Water + β₅·Thermal + β₆·Isolation + Synergy
NoiseCore = γ₁·Urban + γ₂·EM + γ₃·Traffic + γ₄·Industrial + γ₅·Crowd
Match_B = 100 × sigmoid(GeoCore - NoiseCore + λ_c·Conf + λ_h·HistBonus)
```

默认系数见 [[02_物理公式与参数手册]] §3.4。

### 1.3 等级映射

| Match_B 区间 | 判定 | 适用场景 |
|-------------|------|----------|
| 0-20 | 普通区 | 无修炼价值 |
| 20-40 | 低灵区 | 零散感知，新手体验 |
| 40-60 | 中灵区 | 稳定修炼，小型设施 |
| 60-80 | 高灵区 | 长期修炼，宗门据点 |
| 80-100 | 洞天候选 | 高价值节点，战略资源 |

### 1.4 地点条目模板

每个地点包含以下字段：

| 字段 | 说明 |
|------|------|
| 名称 | 现实地名 + 游戏内称号 |
| 坐标 | 经纬度 |
| 地质概况 | 岩性、构造、矿脉 |
| 六因子评分 | Ore/Fault/Magnet/Water/Thermal/Isolation（0-1） |
| 噪声评分 | Urban/EM/Traffic/Industrial/Crowd（0-1） |
| Match_B | 总分 |
| 等级 | 普通/低灵/中灵/高灵/洞天候选 |
| 主频谱 | 五行偏向 |
| 古代记载 | 历史文献中的相关记录 |
| 2035现状 | 游戏开始时的控制状态 |
| 适用人群 | 新手/中阶/高阶/宗门/封锁 |

## 4. 高灵区分布规律

### 4.1 地质控制因素

通过对 57 处高灵区的统计分析，可以提取以下分布规律：

| 规律 | 说明 | 验证 |
|------|------|------|
| **断裂带控制** | 85% 的高灵区及以上地点位于主要断裂带 30km 以内 | 昆仑、长白山、阿法尔三联点、东非大裂谷均在深大断裂带上 |
| **矿脉富集** | 洞天候选级地点 100% 含有 B 级以上灵材矿脉 | 昆仑（和田玉）、长白山（温泉矿化）、达纳基尔（钾盐+硫磺）、尼拉贡戈（稀土） |
| **水热协同** | Water 和 Thermal 同时高于 0.5 的地点，GeoCore 平均高出 15% | 长白山天池、北极灵岛、堪察加、阿苏火口 |
| **噪声压制** | 同等地质条件下，城市周边地点 Match_B 平均低 20-30 分 | 房山(32) vs 终南山(72)，地质相似但噪声差异巨大 |
| **古代记载验证** | HistBonus 高的地点中，90% 的 GeoCore 也独立达标 | 说明古人选址确实基于对地质条件的经验感知 |
| **极寒增强** | 北俱芦洲高灵区的 Conf 值平均高出同级别 5-8% | 低温环境增强灵气耦合稳定性，与理论预测一致 |

- 终南山（Match_B=72，高灵区，道教圣地加成，但2035年已被QESA监控）
- 龙虎山（Match_B=65，高灵区，丹鼎派传承地）
- 武当山（Match_B=68，高灵区，内家功法传承地）

新手角色不应直接进入洞天候选级地点——灵气浓度过高会触发过载型走火（见 [[04_走火入魔机制]] §3.1）。

### 5.2 势力分布

各势力对高灵区的控制关系见 [[01_势力机构与学派冲突]]：
- QESA（学院派）：优先控制已公开的高灵区，建立监测站和修炼基地
- 古法门派：占据传统道教名山，以"文化遗产保护"名义维持独立
- 企业/平台：争夺城市周边中灵区的商业开发权
- 黑市：活跃于未被监控的偏远高灵区和地下矿脉

### 5.3 经济联动

高灵区的灵材产出直接影响 [[01_经济系统设计]] 中的资源供给：
- 洞天候选级地点的S级灵材产出率最高
- 中灵区是大众修炼的主要场所，消耗品需求最大
- 低灵区适合新手训练，但灵材品质有限
