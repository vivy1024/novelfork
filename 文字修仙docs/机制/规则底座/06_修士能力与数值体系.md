# 06_修士能力与数值体系

> **一句话版**：修士不是携带永恒超能力的存在，而是特定条件下能显著超越常人的改造体——低阶修士**必须站在场域里**才有能力；紫府以上**不再是碳基生物**，衰亡由场结构退相干决定；金丹**自成循环**，不依赖外部灵气场，只需定期吸收灵物补能。
> **关联文档**：[[02_修炼体系与人体映射]] · [[02_物理公式与参数手册]] · [[04_走火入魔机制]] · [[01_核心物理设定]] · [[03_八洲文明格局与超凡纪年]]

## §0 建模原则

### 0.1 三个核心前提

1. **场域依赖前提**：低阶（胎息~筑基）修士能力 = `f(内部储能, 外部场强, 匹配度)` 的条件函数，离场大幅衰减。紫府以上场域依赖锐减。**金丹及以上不依赖外部灵力场**——五频锁相自成循环，只需定期吸收灵物补充能量。
2. **超忆前提**：修炼强化神经网络后，记忆保真度不可逆上升。既是学习红利，也是心魔物理根源。
3. **锚定原则**：所有超常能力必须锚定人体真实参数的倍率增幅，不允许"一句话设定破坏力等于核弹"。

### 0.2 与现有体系的对接

| 现有参数（见 [[02_物理公式与参数手册]]） | 本文档扩展 |
|---|---|
| `L_fit`（灵根适配） | 所有能力的**基础乘数** |
| `V_d`（丹田容量） | 决定**离场续航**（仅胎息~紫府） |
| `MeridianFlow`（通畅度） | 决定**输出效率和控制精度** |
| `Risk_S`（心理风险） | 接入**记忆保真度 → 心魔**机制 |
| `StageIndex`（境界序号） | 所有能力表的**行索引**（胎息=1, 道胎=6） |

## §1 场域依赖与能量模型

### 1.1 低阶公式（胎息~紫府）

修士在时刻 t、位置 x 的**实际能力**：

```math
Ability(t,x) = Ability_{peak} × [α_s · FieldRatio(x,t) + (1 - α_s) · ReserveRatio(t)]
```

| 符号 | 定义 |
|------|------|
| `Ability_peak` | 该境界该维度的理论峰值（最优场域 + 满储能） |
| `α_s` | **场域依赖系数**，随境界递减 |
| `FieldRatio` | `clamp(Q(x,t) / Q_optimal, 0, 1.2)`，当前场强与最优场强之比 |
| `ReserveRatio` | `V_d_current(t) / V_d_max`，丹田余量占比 |

### 1.2 场域依赖系数 α_s

> 设计逻辑锚定 [[02_修炼体系与人体映射]] §5 的 LEVSS 本质描述。
> **核心区分**：胎息~筑基依赖**外部灵气场**；紫府依赖减弱；**金丹以上不依赖外部灵力场**——五频拓扑锁相自成循环，只需定期吸灵物补能。

| 境界 | StageIndex | `α_s` | 含义 | LEVSS 物理依据 |
|------|-----------|-------|------|---------------|
| **胎息** | 1 | 0.85 | 85% 靠外部场 | 仅有六轮微耦合节点，无稳定储能结构 |
| **练气** | 2 | 0.65 | 65% 靠外部场 | 主动泵浦启动，有初步循环但容量有限 |
| **筑基** | 3 | 0.45 | 45% 靠外部场 | 身体微观结构永久改写，骨骼嵌入类压电晶格 |
| **紫府** | 4 | 0.15 | 15% 靠外部场 | 意识迁移到场结构，场本身成为主要储能载体 |
| **金丹** | 5 | **0** | **不依赖外部场** | 五频拓扑锁相=自循环内部场源，只需灵物补能 |
| **道胎** | 6 | **0** | **不依赖外部场** | 完全自持场生命体，只需灵物补能 |

> **三个分水岭**：
> - **筑基→紫府**（α从0.45到0.15）：从"强化人类"到"场结构生命"。紫府以上修士可在普通城市维持绝大部分能力。
> - **金丹**（α=0）：**自循环分水岭**。五频锁相后自成循环，完全不需要外部灵力场。行走在灵气枯竭区也毫无影响——只要内部灵物储备不耗尽就行。
> - **与"灵物补给"的区别**：低阶修士需要"泡在灵气场里"才能维持能力（像鱼需要水）；金丹修士不需要"水"，只需要定期"吃东西"——吸收灵物补充五频锁相结构的能量消耗。

### 1.3 金丹以上的灵物补给模型

金丹和道胎不使用 §1.1 的场域依赖公式（α_s=0）。它们的能力维持取决于**内部灵物储备**：

```math
Ability_{金丹+}(t) = Ability_{peak} × ReserveRatio(t)
```

```math
ReserveRatio(t) = clamp(E_{stored}(t) / E_{max}, 0, 1)
```

其中 `E_stored` 通过吸收灵物回充：

```math
E_stored(t) = E_stored(t-1) - ConsumptionRate × Δt + ΣAbsorption(灵物_i)
```

| 境界 | 日均消耗（灵石当量） | 满储备续航 | 补给方式 |
|------|-------------------|-----------|----------|
| 金丹 | ~5 枚中品灵石/日 | ~6 个月 | 吸收灵材、灵药、天然灵脉能量 |
| 道胎 | ~20 枚中品灵石/日 | ~3 个月 | 吸收高品灵材、矿脉深层能量、天外灵材 |

> **叙事含义**：金丹修士不怕"离开灵山"——他们怕的是"断粮"。一个被封锁灵物供给的金丹修士，六个月后储备耗尽，能力归零（场结构仍在，但无法输出）。**高阶修士不怕战斗，怕断粮。**

### 1.4 低阶场域效应示例

以**练气修士**为例（α_s=0.65）：

```
场景 A：高灵区洞府（FieldRatio=1.0）
  丹田 80% → Ability = Peak × [0.65×1.0 + 0.35×0.8] = Peak × 0.93 → 接近满

场景 B：普通城市（FieldRatio=0.1）
  丹田 80% → Ability = Peak × [0.65×0.1 + 0.35×0.8] = Peak × 0.345 → 仅剩 34%

场景 C：灵气枯竭区（Q=0）且丹田耗尽
  Ability = 0 → 完全回到普通人
```

对比**金丹修士**在同一场景 B（α_s=0，不受场强影响）：
```
  灵物储备 80% → Ability = Peak × 0.80 → 就是 80%，和在不在灵气场里无关
```

> 这解释了为什么**宗门建在灵山上**——低阶修士离场后衰减剧烈。也解释了为什么**金丹修士可以去任何地方**——他们只需带够灵物。

### 1.5 离场衰减（仅胎息~紫府适用）

脱离灵气场后，储能按泄漏率消耗（金丹以上不适用此模型）：

```math
V_d(t) = V_d(0) · e^{-λ_leak · t}
```

| 境界 | `λ_leak` (/h) | 半衰期 | 实战含义 |
|------|--------------|--------|---------|
| 胎息 | 0.20 | ~3.5 h | 出门两小时就开始衰退 |
| 练气 | 0.10 | ~6.9 h | 支撑一个白天的外出 |
| 筑基 | 0.03 | ~23 h | 可在普通区过夜 |
| 紫府 | 0.006 | ~4.8 天 | 近一周续航 |
| 金丹 | — | — | 不适用。自循环，只消耗灵物储备 |
| 道胎 | — | — | 不适用。自循环，只消耗灵物储备 |

## §2 寿命模型

### 2.1 关键区分：碳基衰老 vs 场结构退相干

修士寿命不是一个统一公式能覆盖的。根据 [[02_修炼体系与人体映射]] §5 的 LEVSS 本质：

| 阶段 | 存在形态 | 衰亡机制 | 适用模型 |
|------|---------|---------|----------|
| **胎息~筑基** | 碳基生物（被 LEVSS 增强） | 细胞衰老（端粒/DNA/干细胞） | Gompertz 抗衰模型 |
| **紫府** | 碳基→场结构过渡态 | 场结构信息熵积累 + 碳基残余衰老 | 混合模型 |
| **金丹~道胎** | 非碳基（LEVSS 场生命体） | **场结构退相干** + **心魔崩溃** | 场退相干模型 |

> **核心洞见**：紫府以上的修士**不是碳基生物了**——他们的意识运行在 LEVSS 场结构上，肉身只是法身的习惯性投影。所以 Gompertz 衰老模型不适用。金丹修士的"死亡"不是器官衰竭，而是**拓扑保护场结构的长期退相干** + **千年记忆积累导致的心魔崩溃**。

---

### 2.2 碳基段寿命公式（胎息~筑基）

```math
Lifespan_{碳基} = L_base × SM_bio(stage) × Q_env × C_poll × C_life
```

| 参数 | 定义 | 默认/范围 |
|------|------|----------|
| `L_base` | 人类基础寿命 | 80 年 |
| `SM_bio` | 碳基延寿倍率 | 见推导 |
| `Q_env` | 长期灵气环境质量系数 | 0.8 ~ 1.2 |
| `C_poll` | 污染折损系数 = `1 - 0.003 × Poll` | 0.5 ~ 1.0 |
| `C_life` | 生活条件系数 | 0.7 ~ 1.2 |

**Gompertz 推导**：人类死亡率 `μ(t) = μ_0 · e^(β·t)`。LEVSS 耦合通过降低 `β` 延缓衰老：

```math
β_修士 = β_0 / (1 + κ · CouplingDepth(stage))
```

| 境界 | CouplingDepth | SM_bio | 理论寿命 | 物理解释 |
|------|--------------|--------|---------|----------|
| 普通人 | 0 | **1.0** | 80 年 | 基准 |
| 胎息 | 5.7 | **1.2** | ~96 年 | 应激轴稳定，慢性炎症降低 |
| 练气 | 15.2 | **1.8** | ~144 年 | 灵气微循环修复端粒 |
| 筑基 | 35.8 | **3.5** | ~280 年 | 系统性细胞修复机制建立 |

> **筑基是碳基生物的寿命上限。** 即使最完美的 LEVSS 增强也无法让碳基生物活过约 350 年——因为信息存储的物理载体（DNA/蛋白质）的累积错误率有不可逾越的下限。

---

### 2.3 场结构段存续公式（紫府~道胎）

紫府以上修士不再是碳基生物。他们的意识运行在 LEVSS 场结构上。"衰老"不再是细胞层面的事，而是两种威胁：

**威胁一：场结构退相干**

场结构不是永恒的。即使有拓扑保护，宇宙背景噪声、自身场波动和外部干扰会导致**相干度缓慢下降**——当相干度降到临界值以下，场结构崩溃，意识消散。

```math
Coherence(t) = C_0 × e^{-t / τ_decoherence(stage)}
```

**威胁二：心魔崩溃**

千年级别的完美记忆（MemFidelity > 0.88）+ 近零的情绪衰减 → 未整合创伤无限积累 → `InnerDemon` 超过意识场承载上限 → 场结构自毁。这是高阶修士最常见的"死因"。

```math
T_心魔上限 = MentalCapacity(stage) / (TraumaRate × MemFidelity × (1 - EmotionDecay))
```

**综合存续时间** = `min(T_退相干, T_心魔上限) × C_supply`

| 境界 | τ_退相干 | T_心魔上限† | 综合存续 | 死亡方式 |
|------|---------|-----------|---------|----------|
| **紫府** | ~800 年 | ~700 年 | **~640 年** | 多为心魔崩溃（意识分裂） |
| **金丹** | ~5000 年 | ~1800 年 | **~1500 年** | 多为心魔崩溃（念头不通达） |
| **道胎** | ~50000 年 | ~4000 年 | **~3600 年** | 心魔是唯一现实威胁 |

† 心魔上限按"中等心性修为、平均创伤率"估算。心性极佳者可大幅延长。

> **关键推论**：
> - 金丹的拓扑保护场结构退相干时间长达 5000 年——**理论上金丹修士可以活 5000 年**。但没有人做得到，因为 1500 年的完美记忆积累就会把心魔压过承载上限。
> - **道胎的"死亡"几乎只来自心魔**——场结构退相干时间 50000 年，物理上近乎永恒。但 3600 年的心理负荷已是极限。
> - 这就是为什么 [[03_八洲文明格局与超凡纪年]] §6.7 心性改革是文明级事件：**心性修炼直接决定高阶修士能活多久**。

---

### 2.4 灵物供给对存续的影响

金丹以上的存续还需要一个前提条件：**灵物供给不中断**。

```
C_supply = 1    （灵物充足时）
C_supply → 0    （灵物断供后，储备耗尽即场结构因能量不足崩溃）
```

金丹修士理论上可以活 1500 年——但如果被封锁灵物供给 6 个月（见 §1.3），场结构因无法维持五频锁相所需的能量输入而崩溃。**高阶修士不怕战斗，怕断粮。**

### 2.5 寿命总览

| 境界 | 存在形态 | 理论最优寿命 | 衰亡限制因素 | 公式来源 |
|------|---------|------------|------------|----------|
| 普通人 | 碳基 | 80 年 | 细胞衰老 | Gompertz |
| 胎息 | 碳基增强 | ~96 年 | 细胞衰老（减缓） | Gompertz + LEVSS |
| 练气 | 碳基增强 | ~144 年 | 细胞衰老（减缓） | Gompertz + LEVSS |
| 筑基 | 碳基改造 | ~280 年 | 碳基信息存储极限 | Gompertz + LEVSS |
| 紫府 | 过渡态 | ~640 年 | **心魔崩溃** > 退相干 | 场退相干 + 心魔 |
| 金丹 | 场生命体 | ~1500 年 | **心魔崩溃** ≫ 退相干 | 场退相干 + 心魔 + 灵物供给 |
| 道胎 | 自持场生命 | ~3600 年 | **心魔是唯一现实威胁** | 心魔上限（退相干近乎无限） |

### 2.6 寿命的叙事含义

> **为什么越强越容易死于心魔？**
>
> 金丹修士活 1500 年，每一秒的记忆以 0.96 保真度永久存储。1500 年的得失、爱恨、恐惧、屈辱、背叛——**全部以初次体验的强度同时存在于意识中**。场结构可以用拓扑保护撑 5000 年，但承载这些记忆的意识场在 1500 年左右就会因信息过载而自毁。
>
> 这就是心性修炼的终极意义：不是学会压制情绪，而是学会**整合记忆**——将每一段创伤从"永恒的开放性伤口"变成"已经愈合的疤痕"。
>
> 这与 [[03_八洲文明格局与超凡纪年]] §4.3 的心性改革完全一致：高阶修士的心性不是道德可选项，是**物理层面上活下去的唯一方法**。

## §3 记忆保真度与心魔

### 3.1 核心机制

> 修炼强化神经网络 → 突触修剪变慢 → 记忆保真度上升 → 情绪记忆自然衰减机制失效 → **任何时候想起过去的创伤，情绪强度接近初次体验** → 未整合的创伤永远是活雷 → **心魔的物理本质**。

### 3.2 记忆保真度

```math
MemFidelity(stage) = MF_0 + MF_gain × StageIndex
```

| 境界 | StageIndex | MemFidelity | 含义 |
|------|-----------|-------------|------|
| 普通人 | 0 | 0.40 | 正常遗忘 |
| 胎息 | 1 | 0.52 | 开始出现异常清晰闪回 |
| 练气 | 2 | 0.64 | 近两年事件几乎过目不忘 |
| 筑基 | 3 | 0.76 | 数年前对话逐字回忆 |
| 紫府 | 4 | 0.88 | 近乎完美自传体记忆 |
| 金丹 | 5 | 0.96 | 真正超忆症——永不褪色 |
| 道胎 | 6 | 0.99 | 全息级记忆重现 |

> 默认：`MF_0 = 0.40`，`MF_gain = 0.12`

### 3.3 情绪衰减率

普通人的情绪痛苦随时间自然衰减。修士**失去了这个安全阀**：

```math
EmotionDecay(stage) = ED_0 × (1 - MemFidelity)^γ
```

| 境界 | EmotionDecay(/月) | 含义 |
|------|------------------|------|
| 普通人 | 0.95 | 3个月后痛苦降至 14% |
| 胎息 | 0.82 | 半年后仍保留 20% |
| 练气 | 0.65 | 一年后仍 28% |
| 筑基 | 0.42 | 三年后还 35% |
| 紫府 | 0.18 | 十年前伤痛犹在 |
| 金丹 | 0.05 | **几乎不衰减** |
| 道胎 | 0.01 | 永恒现在时 |

> 默认：`ED_0 = 0.95`，`γ = 1.5`。接入走火入魔公式详见 [[04_走火入魔机制]]。

### 3.4 学习速度

```math
LearningSpeed(stage) = 1 + κ × MemFidelity(stage)
```

| 境界 | 倍率 | 含义 |
|------|------|------|
| 普通人 | ×1.0 | 基准 |
| 胎息 | ×1.3 | 资料不容易忘 |
| 练气 | ×1.6 | 可同时消化多套功法 |
| 筑基 | ×1.9 | 海量知识过目不忘 |
| 紫府 | ×2.2 | 观战一次即可复盘 |
| 金丹 | ×2.4 | 千年积累完美保持 |
| 道胎 | ×2.5 | 全息级经验回溯 |

> 默认：`κ = 2.0`。这不是智力提高，是信息留存和提取效率提高。
