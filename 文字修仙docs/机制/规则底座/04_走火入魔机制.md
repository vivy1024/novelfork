# 04_走火入魔机制

> **一句话版**：在这个世界里，走火入魔不是玄学惩罚，而是灵气输入、人体承载、环境噪声、污染积累和心理失衡共同导致的系统性失控事故；它有明确类型、阶段、预警信号、干预窗口和后遗症谱系。
> **关联文档**：[[01_核心物理设定]] · [[02_修炼体系与人体映射]] · [[01_灵材丹药与法器目录]] · [[01_势力机构与学派冲突]] · [[02_意识形态对抗深度设计]]

## 1. 定义与设计目标

### 1.1 游戏内定义

> **走火入魔 = 修炼过程中，灵气流动、组织承载、神经调控和心理解释框架发生持续性失配，最终演化成不可自行收束的破坏性状态。**

它不是单一事件，而是一条事故链：

- 先有微小偏差
- 偏差在错误环境中被放大
- 身体和意识开始用错误方式补偿
- 进入恶化与失控
- 留下急性损伤、长期后遗症或人格改变

### 1.2 设计目标

本系统要解决四个问题：

1. 让“走火入魔”从模糊概念变成**可分类、可观察、可预防**的机制
2. 让玩家在修炼中感到真实风险，而不是简单的随机暴毙
3. 给学院派、古法派、企业和黑市提供不同的处理逻辑
4. 为后续数值系统预留接口，但**不等待公式手册完成**

> **并行原则**：本文件先定义结构、事件树和文本表现；后续可由 `10_物理公式与参数手册` 回填风险权重和概率函数。

## 2. 事故总框架

### 2.1 五个主因子

任意一次走火入魔，都可以拆成五类主因子的组合失衡：

| 因子 | 含义 | 典型来源 |
|------|------|---------|
| **输入过强** | 灵气输入速率超过当前承载上限 | 高浓度高灵区、暴烈法器、过量药剂 |
| **路径失谐** | 经脉/节点导引方向错误或共振频段偏移 | 姿势错误、功法版本不匹配、局部旧伤 |
| **载体污染** | 灵材杂质、环境残留或体内污染度过高 | 低纯度灵材、黑市药剂、长期超负荷修炼 |
| **心理崩塌** | 对异常体感的解释失控，形成恐惧、妄想或偏执 | 睡眠剥夺、长期压力、孤立闭关、过度暗示 |
| **回收失败** | 修炼结束时未能泄压、散功、排毒或重建秩序 | 贪功冒进、中断过晚、缺少辅助器材 |

### 2.2 事故链四阶段

| 阶段 | 特征 | 玩家感受 |
|------|------|---------|
| **预警** | 指标和体感出现轻微异常，但仍可收束 | “不太对劲，但还撑得住” |
| **恶化** | 异常开始自我放大，简单休息已不足以逆转 | “身体开始不听使唤” |
| **失控** | 局部或全身进入破坏性循环，必须外部干预 | “不是修炼，是事故现场” |
| **后果** | 事故结束，但损伤、污染、人格偏差持续存在 | “活下来了，但不是毫发无损” |

> **设计原则**：玩家多数时候不是输在“突然暴死”，而是输在忽视了预警阶段。

## 3. 四大基本类型

REQ-3 要求的四大类型为：过载型、失谐型、污染型、心理型。它们可以单独出现，也可以叠加。

### 3.1 过载型

> **定义**：灵气输入功率大于当前身体、经脉或储能结构所能承受的安全窗口。

| 项目 | 内容 |
|------|------|
| **常见诱因** | 强行吸纳高灵区灵气、爆发型丹药、连续多轮修炼、带伤突破 |
| **预警信号** | 局部灼热、心悸、耳鸣、眼压升高、指尖麻木、呼吸不匀 |
| **恶化表现** | 胸闷、肌肉抽搐、短时失明、局部组织胀痛、动作失控 |
| **失控后果** | 经脉灼伤、组织坏死、丹田泄漏、器官损伤、短时昏迷 |
| **典型人群** | 急于突破者、竞技型修士、使用高爆发法器者 |
| **古法术语** | 火毒上冲、阳火暴走、真气逆灌 |
| **学院派术语** | 高输入失载事故 |

### 3.2 失谐型

> **定义**：灵气流动方向、频谱模式或节点切换错误，导致系统内部共振结构错位。

| 项目 | 内容 |
|------|------|
| **常见诱因** | 功法版本不适配、姿势角度偏差、错误模仿高阶路线、局部陈旧损伤 |
| **预警信号** | 一侧身体明显更冷或更热、左右压感不对称、心口发空、头顶紧束感 |
| **恶化表现** | 局部阻塞、气感乱窜、动作僵硬、语言迟滞、方向感紊乱 |
| **失控后果** | 节点锁死、慢性偏差、错误经路固化、后续功法永久受限 |
| **典型人群** | 自学者、跨流派乱修者、长期用盗版功法者 |
| **古法术语** | 岔气、错脉、逆周天 |
| **学院派术语** | 导引相位偏移事故 |

### 3.3 污染型

> **定义**：灵气载体中混入无法被当前系统正确处理的杂质信息、异常频谱或化学毒性，逐渐改变身体与意识的反应模式。

| 项目 | 内容 |
|------|------|
| **常见诱因** | 低纯度灵石、黑市丹药、受污染高灵区、失效法器、长期不排毒修炼 |
| **预警信号** | 修炼后口苦、异味汗、睡眠异常、皮肤斑驳、梦境重复、轻微幻听 |
| **恶化表现** | 持续性疲劳、情绪钝化或暴躁、对特定灵材异常依赖、感知噪声增加 |
| **失控后果** | 慢性人格偏移、长期污染度上升、灵材上瘾、局部异化、难以逆转的体质改变 |
| **典型人群** | 穷修、黑市修士、矿区工人、长期使用工业灵材者 |
| **古法术语** | 阴煞入体、丹毒积骨、浊气不散 |
| **学院派术语** | 耦合污染与慢性异化事故 |

### 3.4 心理型

> **定义**：修炼引发的异常体感被个体错误解释，并通过恐惧、执念、狂喜、救世妄想等心理结构不断放大，最终反向塑造生理反应。

| 项目 | 内容 |
|------|------|
| **常见诱因** | 长期独修、睡眠剥夺、过度依赖 AI 评分、强烈执念、宗教性暗示 |
| **预警信号** | 对微弱体感过度赋义、开始“听见”指导声音、强迫记录、情绪飙升 |
| **恶化表现** | 妄想自己突破、敌意增强、对外界劝阻极端反感、现实检验能力下降 |
| **失控后果** | 自残、他伤、群体性失控、人格分裂样表现、长期精神后遗症 |
| **典型人群** | 闭关者、孤立高压修士、被意识形态强烈裹挟的人 |
| **古法术语** | 心魔作祟、神乱、识海蒙尘 |
| **学院派术语** | 修炼诱发认知失稳事故 |

## 4. 复合类型与典型组合

真实事故通常不是单一类型，而是复合型链条。

| 组合 | 说明 | 常见场景 |
|------|------|---------|
| **过载 + 失谐** | 输入太猛，且导引路径偏了 | 强行冲击筑基 |
| **过载 + 污染** | 便宜高效的爆发药把杂质一并压入体内 | 黑市速成修炼 |
| **失谐 + 心理** | 错误体感被解释为“天启”或“异象” | 无师承自修 |
| **污染 + 心理** | 长期污染改变认知风格，逐步形成偏执信念 | 地下宗教化修炼圈 |
| **全叠加** | 典型灾难级事件 | 人工洞天事故、军用试验失控 |

> **游戏设计**：单类型事故适合教学；复合类型事故适合中后期高压情节。

## 5. 从预警到后果的事件树

### 5.1 通用事件树

```text
准备阶段异常
  -> 忽视预警
  -> 输入继续上升
  -> 身体/意识尝试错误补偿
  -> 进入恶化
  -> 玩家选择：停机 / 硬顶 / 求援 / 用药 / 改功法
  -> 成功收束 / 带伤收束 / 部分失控 / 全面失控
  -> 留下短期或长期后果
```

### 5.2 关键决策点

| 决策点 | 玩家选择 | 可能结果 |
|--------|---------|---------|
| **预警时是否停止** | 停 / 观察 / 硬顶 | 直接决定是否进入恶化 |
| **恶化时相信谁** | AI / 师父 / 自己 / 同伴 | 影响处理方式和路线归因 |
| **是否使用应急药剂** | 用 / 不用 / 超量用 | 可救命，也可能加重污染 |
| **是否暴露事故** | 上报 / 隐瞒 / 伪装 | 影响后续救治、门派声誉和派系关系 |
| **事故后是否休整** | 休整 / 带伤继续 / 转投他法 | 影响长期污染度和人格偏差 |

## 6. 预警指标体系

这里不直接给数值，而给后续 `REQ-1` 可回填的接口字段。

| 维度 | 轻度预警 | 中度预警 | 重度预警 |
|------|----------|----------|----------|

## 7. 救治与恢复路径

### 7.1 四类处理方式

| 方式 | 适用情况 | 优点 | 风险 |
|------|---------|------|------|
| **自然恢复** | 轻度预警、轻微污染 | 成本低，不引入新变量 | 慢，容易误判时机 |
| **药物干预** | 过载、炎症、睡眠崩坏、污染早期 | 快速缓冲，便于标准化 | 副作用、依赖、二次污染 |
| **法器干预** | 失谐、局部锁死、节点异常 | 定向强，能快速纠偏 | 需要设备和熟手 |
| **高阶干预** | 重度心理型、复合型、群体事故 | 能处理复杂情况 | 稀缺、昂贵、带有人身与政治依赖 |

### 7.2 恢复阶段

| 阶段 | 目标 | 典型措施 |
|------|------|---------|
| **止损** | 停止事故继续扩大 | 停修、散功、镇静、环境隔离 |
| **排压** | 释放残余过载与错位共振 | 缓释药剂、法器泄压、低频导引 |
| **排污** | 降低污染度和异常耦合残留 | 吸附材料、抗炎药、睡眠重建、饮食调整 |
| **重建** | 重建路径和对体感的正确解释 | 低强度复健、复盘、AI 模型再训练、师承校正 |

### 7.3 后遗症等级

| 等级 | 描述 | 例子 |
|------|------|------|
| **I 级** | 可逆短期损伤 | 一周内恢复的疲劳、局部疼痛 |
| **II 级** | 中期功能下降 | 数月内修炼效率下降、节点敏感化 |
| **III 级** | 长期体质改变 | 某类灵材永久不适配、经脉局部锁死 |
| **IV 级** | 人格与认知层改变 | 长期偏执、快感依赖、神识异常分层 |
| **V 级** | 社会性毁损 | 无法继续修炼、对他人危险、被机构隔离 |

## 8. 不同阵营的处理逻辑

### 8.1 学院派

- 优先监测指标、药理缓冲、康复训练
- 会把走火入魔拆成具体病理与事故类型
- 优势是标准化和可复制
- 弱点是对高阶复杂意识异常的处理有时过于机械

### 8.2 古法派

- 优先判断“气机”“心境”“师承路数”出了什么问题
- 重视师父直接介入、静室隔离、低频调息和长期修心
- 优势是对复杂主观体验敏感
- 弱点是经验强、难复制，且容易混入迷信和权威压迫

### 8.3 企业/平台体系

- 优先止损、控责、隐藏事故外溢
- 倾向用快速药物与模型调整维持用户可继续使用服务
- 优势是响应快
- 弱点是容易把事故压成“个体不规范操作”

### 8.4 黑市

- 优先压症状、保行动能力、避免暴露
- 常提供“看起来立刻有效”的高风险方案
- 短期可能救命，长期往往加重污染和人格后遗症

### 8.5 事故处理的经济与阶级维度

> 以下数据来自 [[01_经济系统设计]] §9.4，创作事故相关章节时必查。

**年度事故统计**：2035 年全国走火入魔事故约 565,000 起，总经济损失 ¥800–1,500 亿/年。

**赔偿的阶层差异**：

| 修士阶层 | 典型事故赔偿 | 救治资源 | 信息公开度 |
|---------|------------|---------|-----------|
| 顶层（金丹+/企业高管/宗门核心） | ¥200–800 万 | 专家团队+稀有灵药 | 严格保密 |
| 中层（筑基/企业骨干/学院研究员） | ¥10–50 万 | 标准医疗+中等灵药 | 内部通报 |
| 底层（胎息–引气/平台灵活就业） | ¥0–1 万 | 自费+社区互助 | 无人知晓 |
| 散修（无组织归属） | ¥0 | 自救或等死 | 可能被当作"非法修炼" |

**"三层真相"结构**（来源：[[02_意识形态对抗深度设计]] §9.4.2）：

| 层级 | 知道什么 | 谁在这一层 |
|------|---------|-----------|
| 公众真相 | 媒体报道的"官方版本"（淡化严重性） | 普通人、底层修士 |
| 机构真相 | QESA 内部报告（技术细节但不公开） | QESA 中层、企业管理层 |
| 核心真相 | 真正的事故原因（可能涉及高层失误/利益输送） | QESA 高层、企业决策层、宗门长老 |

> **创作要点**：写事故章节时，先确定视角人物在哪一层。同一起事故，从不同层级看到的"真相"完全不同。这是叙事张力的核心来源。

## 9. 游戏中的文本表现模板

### 9.1 预警文本模板

- **过载预警**：你能感觉到灵气像被灌进过细的管路，热意不再均匀，而是集中成几个危险的亮点。
- **失谐预警**：那股熟悉的流动感第一次偏离了记忆中的路径，像一枚齿轮卡在了本不属于它的槽里。
- **污染预警**：修炼结束后，灵气没有散净，反而在口腔后部留下金属一样的冷涩味，像有什么不属于你的东西还停在里面。
- **心理预警**：你开始本能地为每一种刺痛寻找意义，仿佛只要错过一次解释，就会错过某个命中注定的征兆。

### 9.2 恶化文本模板

- 你知道现在应该停下，但身体已经先于理智做出了另一个选择。
- 原本清晰的导引节奏开始彼此重叠，像两种不同的命令同时要求你服从。
- 每一次试图修正，都像在把偏差按得更深。

### 9.3 失控文本模板

- 这一刻已经不是修炼，而是系统在你体内自行运行。
- 你第一次意识到，所谓走火入魔，并不是“火”或“魔”，而是你失去了对自己这套结构的解释权。

### 9.4 事故后文本模板

- 事故结束了，但它并没有离开，只是从表面退回了更难看见的地方。
- 你保住了命，也保住了大半条路；可那条路已经不再和从前完全一样。

## 11. 结论

可以把走火入魔系统压缩成一句设计原则：

> **走火入魔不是修炼的随机惩罚，而是修炼文明必须付出学习成本的方式。**

真正决定一个社会是否成熟的，不是它有没有事故，而是：

- 它能否更早预警
- 能否更公平救治
- 能否把失败转化为公共知识
- 能否阻止强者把事故成本转嫁给弱者

这也正是 [[01_势力机构与学派冲突]] 与 [[02_意识形态对抗深度设计]] 中不同制度路线最终会分化的地方。
