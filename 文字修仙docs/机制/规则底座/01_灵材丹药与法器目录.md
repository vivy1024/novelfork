# 灵材、丹药与法器目录

> **一句话版**：游戏中每一件物品都有真实世界的化学/矿物/生物学原型。玩家看到的不是"九转玄冥草"，而是"经过 LEVSS 耦合激活后的、具有特定灵气参数的真实物质"。
> **关联文档**：[[01_核心物理设定]] · [[02_修炼体系与人体映射]] · [[04_修仙隐匿史与LEVSS波动理论]]

## 物品通用参数框架

游戏中所有灵气相关物品共享以下参数体系（对应 [[01_核心物理设定#2.2 灵材]] 中的定义）：

| 参数 | 缩写 | 含义 | 单位/范围 |
|------|------|------|----------|
| 耦合截面 | σ_q | 捕获/释放灵气的效率 | 低/中/高/极高 |
| 容量密度 | ρ_q | 单位体积可储存的灵气总量 | 以"标准灵石单位"计 |
| 主耦合频谱 | Spec | 对应五行中的哪种耦合模式 | 金/木/水/火/土/混合 |
| 衰减半衰期 | τ_d | 灵气在其中自然衰减到一半的时间 | 小时/天/月/年 |
| 导灵率 | κ_q | 灵气在其中传导的速度与损耗 | 低/中/高 |
| 纯度 | P | 影响一切参数的效率系数 | 0–100% |
| 毒性/副作用 | Tox | 对人体的危害等级 | 无/微/低/中/高/致死 |

> **设计原则**：玩家看不到数字公式，但能通过鉴定、实验和经验逐步了解物品的参数范围。高纯度材料和低纯度材料可以是同一种矿物，但表现天差地别。

## 二、金属灵材

金属在灵气工程中主要承担**传导、塑形和防护**功能，而非储存功能（金属晶格对灵气的储存能力一般远不如非金属晶体）。

| 游戏名 | 现实原型 | 主要用途 | 特性说明 |
|--------|---------|---------|---------|
| **赤铜** | 紫铜 Cu | 低阶导灵材料 | 便宜、稳定、加工容易；民用法器的主要结构材料 |
| **灵银** | 高纯银 Ag | 中阶导灵材料 | 抗氧化能力中等，但导灵信号损耗极低；适合针灸器具、精密法器线路 |
| **封金** | 高纯金 Au | 灵气隔绝/内衬材料 | 化学惰性极高，灵气几乎不与之反应；做核心储能容器内衬，防止灵气污染/泄漏 |
| **轻钛** | 钛合金 Ti-6Al-4V | 现代法器外壳 | 轻且耐腐蚀，对灵气中性——不增不减；现代武器化法器首选结构材料 |
| **重钨** | 钨 W | 高能法器承载体 | 高密度高熔点，适合承受暴烈放能过程（类比枪管承受膛压） |
| **活汞** | 汞 Hg | 古代炼丹核心介质 | 液态金属，灵气活性极高但剧毒；古代外丹术核心材料，现代严格管控 |

> **游戏设计**：金属灵材的主要玩法是**锻造和合金工艺**。不同配比、不同温度曲线锻造出的合金，导灵率和承受阈值完全不同。

## 三、有机灵材（药材/生物材料）

有机灵材的核心作用不是"直接提供灵气"，而是**调节人体对灵气的响应**——降低炎症、缓冲过载、改善代谢、诱导特定生理状态。

### 3.1 植物药材

| 游戏名 | 现实原型 | 核心功效（LEVSS 框架） | 传统用途对照 |
|--------|---------|----------------------|------------|
| **参须** | 人参（Panax ginseng） | 提高应激耐受和代谢恢复速率；**不直接补灵气**，而是让身体更高效地处理灵气 | 大补元气 |
| **芪片** | 黄芪（Astragalus） | 增强长期修炼后的体力恢复和免疫调节 | 补气固表 |
| **灵芝** | 灵芝（Ganoderma） | 免疫调节、抗炎、改善睡眠质量——修炼恢复期核心药材 | 扶正固本 |
| **虫草** | 冬虫夏草（Cordyceps） | 特殊代谢调节剂：在低氧/高应激下提高线粒体效率 | 补肺益肾 |
| **茶碱源** | 茶（Camellia sinensis） | 温和提神凝神，提高短时注意力和感知信噪比 | 清心明目 |

## 四、现代人工灵材

2025 年 LEVSS 理论确立后，材料科学家开始有意识地设计和制造灵材。到 2035 年游戏开始时，已有以下成熟产品：

| 游戏名 | 现实原型 | 类型 | 特性 |
|--------|---------|------|------|
| **滤灵膜** | 活性炭复合膜 | 净化 | 吸附灵气中的杂质频谱成分——类比水处理中的活性炭滤芯 |
| **导灵丝** | 碳纳米管纤维 | 传导 | 当前人工材料中导灵率最高的；价格昂贵，用于高端法器内部连线 |
| **聚灵板** | 石墨烯薄膜 | 感知/传导 | 二维材料，极大的比表面积带来高灵气敏感性；高端探测器的核心元件 |
| **灵芯片** | 稀土掺杂氧化物薄膜 | 储存/处理 | 现代"灵气集成电路"的核心——将古代符箓的灵气电路功能微型化 |
| **储灵瓶** | 锂离子电池改型 | 储存 | 用改性电解液代替锂盐，实现灵气的便携储存——现代版"法力值" |
| **光导灵缆** | 光纤 | 远距传导 | 利用光子作为灵气信号的载波，实现相对低损耗的远距离传导——但带宽和功率都有限 |
| **共振阵列** | 相控阵天线改型 | 聚灵/投射 | 多组件协同形成定向灵气场——现代版"法阵" |

> **游戏设计**：现代灵材代表"学院派/工业派"的技术路线，高效、标准化、但昂贵且依赖供应链。与"古法派"使用天然灵材形成有趣的对比张力。

## 五、丹药体系

### 5.1 分类框架

丹药不是"吃了就升级"的道具，而是按功能分为六大类——每类都有现实中的药理学对照：

| 类型 | 功能 | 药理学类比 | 风险等级 |
|------|------|-----------|---------|
| **激发剂** | 短时间大幅提高灵气感知/吸收效率 | 兴奋剂/拟交感药 | 高——心悸、过载、成瘾 |
| **缓冲剂** | 防止灵气冲击导致组织损伤 | 抗炎药/钙通道阻滞剂 | 低——但长期使用可能降低耐受上限 |
| **修复剂** | 促进修炼后组织和神经的恢复 | 抗氧化剂/生长因子 | 低——但过量可致组织异常增生 |
| **诱导剂** | 帮助进入特定修炼状态（入静、低代谢、专注） | 镇静剂/冥想辅助 | 中——依赖性和意志力削弱 |
| **载灵剂** | 本身携带少量特定属性灵气 | 缓释制剂 | 看属性——火属性载灵剂可能灼伤内脏 |
| **炼体剂** | 通过可控毒性刺激身体生成适应性结构 | 疫苗/激素类药物 | 高——剂量控制是生死线 |

### 5.2 剂型

让修仙更有"技术感"的关键——丹药不只是"丸"：

| 古称 | 现代剂型 | 特点 |
|------|---------|------|
| 丹/丸 | 缓释丸剂 | 缓慢释放，适合长周期修炼 |
| 散 | 散剂/粉末 | 起效快但持续时间短 |
| 膏 | 浸膏/软膏 | 外用或舌下吸收，避开消化系统 |
| 酒 | 酊剂 | 酒精萃取的灵药溶液 |
| 香 | 吸入剂 | 通过呼吸系统吸收，直接作用于肺经 |
| 露 | 蒸馏液 | 高纯度水溶性成分 |
| — | 贴剂 | 现代透皮给药技术，精确定点投放 |
| — | 胶囊 | 现代标准化量产制剂 |
| — | 注射液 | 高风险高回报——绕过一切吸收障碍，直入血液 |

> **古今对照设计**：古代宗门叫"养气丹"，现代灵药公司可能叫"QR-17 型真空耦合缓释胶囊（500mg×30粒/盒）"。这个**命名反差**本身就是游戏世界观的一部分。

## 六、法器体系

法器 = **灵气功能器件**。它不是魔杖，而是工程产品。

### 6.1 法器的基本构成

任何法器都可以拆解为以下功能模块（与 [[04_修仙隐匿史与LEVSS波动理论]] 中符箓=灵气电路的设计一脉相承）：

| 模块 | 功能 | 常用材料 |
|------|------|---------|
| **灵源** | 灵气供应——自蓄、外接或修士灌注 | 高容量灵石、储灵瓶、修士丹田直连 |
| **导灵通路** | 灵气在器件内的传导线路 | 银线、铜线、碳纳米管、朱砂绘线 |
| **功能核** | 对灵气进行特定处理——放大、变频、聚焦、释放 | 压电晶体、铁电陶瓷、稀土掺杂晶体 |
| **控制界面** | 修士与法器的交互方式 | 符文触发、意识感应、物理开关 |
| **外壳/结构** | 保护内部组件、承受工作应力 | 钛合金、硬木、高分子复合材料 |
| **隔灵层** | 防止灵气泄漏或外部干扰 | 云母、金箔、法拉第笼结构 |

### 6.2 法器分类与示例

| 类型 | 功能 | 古法版 | 现代版 |
|------|------|--------|--------|
| **聚灵器** | 从环境中汇聚灵气 | 玉琮、铜鼎法阵 | 相控阵聚灵板、压电采集器 |
| **储灵器** | 储存灵气供后续使用 | 灵石、丹炉内胆 | 储灵瓶（改型锂电池）、超导储罐 |
| **导灵器** | 在空间中传导灵气 | 铜镜、玉管 | 光导灵缆、碳纳米管线束 |
| **感知器** | 探测灵气浓度和属性 | 罗盘、测气盘 | 灵气光谱分析仪、场强计 |
| **投射器** | 将灵气定向释放 | 法剑、符箓 | 定向灵气发射器、手持法术器 |
| **防护器** | 抵挡灵气冲击或隔绝灵气 | 护身玉佩、阵法 | 灵气屏蔽服、法拉第式护甲 |
| **辅修器** | 辅助修炼过程 | 蒲团、铜磬、灵香炉 | 导引头环、HRV监测表、神经反馈仪 |

### 6.3 法器品质与老化

法器和灵材一样会老化（[[01_核心物理设定#1.4]] 的工程边界）：

| 老化现象 | 原因 | 后果 |
|---------|------|------|
| **耦合频谱漂移** | 晶格在反复充放中产生位错 | 法器"属性偏移"——原来是火属性，用久了偏向木属性 |
| **导灵率下降** | 导线氧化、接触面污染 | 需要更多灵气输入才能达到同样效果 |
| **结构疲劳** | 灵气应力导致的微裂纹积累 | 极端情况下法器炸裂——"炸炉"就是这个原理 |
| **灵气残留** | 之前使用者的灵气印记 | 二手法器有"磨合期"，严重的会产生排斥反应 |

> **游戏设计**：法器有耐久度、需要保养、可以升级改装、也可以因老化报废。高阶法器可能是传承数百年的古物——它的"灵气印记"既是优势（蕴含前人经验）也是风险（与新主人的灵气模式不匹配）。

## 八、物品系统与游戏循环的接口

```
世界探索 → 发现/购买原材料
    ↓
鉴定（确认纯度、属性、品质）
    ↓
加工处理（研磨、萃取、煅烧、合金化、组装）
    ↓
三条输出路线：
  ├── 炼丹 → 丹药 → 服用辅助修炼
  ├── 炼器 → 法器 → 装备/使用/交易
  └── 直接使用 → 布置修炼环境、设置阵法、交易换资源
```

> **核心体验**：玩家不是在"刷装备"，而是在**做材料科学实验和药物调配**。每次炼丹炼器的结果取决于原料品质、工艺参数、环境条件和修士自身状态的综合作用。
