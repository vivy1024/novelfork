## §1 引言与定位

### 1.1 本文档的角色

[[01_灵材丹药与法器目录]] §四用一张 7 行表格概述了 2035 年已有的现代人工灵材。那张表回答的是"有什么"；本文档回答的是"怎么来的、怎么判定、怎么用、往哪走"。

具体覆盖范围：

| 维度 | 04 文档 §四 | 本文档 |
|------|------------|--------|
| 产品列表 | 7 种，一句话描述 | 7 种深度规格 + 8 种前沿概念 |
| 技术参数 | 无 | 规则集 A 全参数 + 样本层认证 |
| 产业链 | 无 | 生产方、使用方、价格区间、技术壁垒 |
| 与天然灵材关系 | "对比张力"一句话 | 多维对比矩阵 |
| 叙事接口 | 无 | 技术冲突场景、伦理争议、国际博弈 |

### 1.2 时间线

- **2025**：LEVSS 理论确立，材料科学家开始有意识地筛选和改造灵材
- **2027**：第一批实验室级人造灵材（滤灵膜 v0.1、导灵丝 v0.1）通过规则集 A 判定
- **2029**：储灵瓶和灵芯片进入小批量试产，军方率先采购
- **2031**：光导灵缆和共振阵列完成工程验证，民用市场开放
- **2033**：聚灵板量产成本降至实验室阶段的 1/20，成为高端探测器标配
- **2035**（游戏开始时）：七大核心产品全部进入工业量产或军用特供阶段

### 1.3 与天然灵材的关系

人造灵材不是天然灵材的替代品，而是**标准化补充**：

- 天然灵材的上限更高（正式 S 级 51 个样本中 44 个是天然矿物单晶样片），但批次一致性差、供应链不可控
- 人造灵材的上限较低（目前最高 A 级），但批次偏差 <5%、可按需定制、供应链可控
- 两者在法器制造中经常混用：天然灵材做核心，人造灵材做通路和封装

> **设计意图**：这种"天然 vs 人造"的张力是游戏世界观的核心冲突之一。古法宗门坚持天然灵材的"灵性"不可替代；工业派认为标准化才是修仙普及的唯一出路。玩家的选择会影响剧情走向。

## §2 人造灵材的分类体系

### 2.2 按技术路线分类

| 技术路线 | 定义 | 典型产品 | 技术成熟度 |
|----------|------|----------|-----------|
| 改性天然材料 | 对天然矿物/有机物进行定向改造 | 滤灵膜（活性炭改性）、储灵瓶（锂电改型） | 高 |

### 2.3 按产业成熟度分类

| 阶段 | 特征 | 产品示例 | 价格区间（标准灵石单位/kg） |
|------|------|----------|--------------------------|
| 实验室阶段 | 仅学术机构可制备，批次 <100g | 拓扑灵导体、生物灵芯 | 不可商业定价 |
| 小批量试产 | 军方/大型门派定制，批次 <10kg | 灵气超材料、等离子灵振器 | 50,000–200,000 |
| 工业量产 | GMP 标准生产线，批次 >100kg | 滤灵膜、储灵瓶、光导灵缆 | 500–5,000 |

> **设计意图**：产业成熟度直接影响玩家获取难度。工业量产品在药店/网购就能买到；实验室阶段的东西需要学术关系或黑市渠道。这为不同玩家路线提供了差异化的资源获取体验。

## §3 七大核心人造灵材详细规格

> 以下参数均基于 [[02_物理公式与参数手册]] §4 规则集 A 的框架。σ_q、ρ_q、Spec、τ_d、κ_q、P、Tox 的定义见 [[01_灵材丹药与法器目录]] 物品通用参数框架。样本层认证标准见 [[03_灵材样本层与工艺认证]] §4.4。

### 3.1 滤灵膜（活性炭复合膜）

**现实原型**：活性炭-PVDF 复合超滤膜，孔径 0.01–0.1 μm，比表面积 800–1200 m²/g。现实中用于水处理、空气净化和医疗透析。灵材化改造的关键在于将活性炭的微孔结构与 LEVSS 耦合频谱的选择性吸附对齐。

**灵材化关键工艺**

| 步骤 | 工艺 | 目的 |
|------|------|------|
| 1 | 椰壳炭高温活化（900°C/N₂） | 形成均匀微孔分布 |
| 2 | 稀土盐（La₂O₃）浸渍处理 | 引入灵气频谱选择性吸附位点 |
| 3 | PVDF 相转化成膜 | 提供机械支撑和疏水性 |
| 4 | 低温等离子体表面改性 | 激活表面 LEVSS 耦合活性 |
| 5 | 定向拉伸取向 | 提高孔道各向异性，增强选择性 |

**规则集 A 判定参数**

| 参数 | 值 | 说明 |
|------|-----|------|
| σ_q | 0.35 | 中低——净化功能不需要高耦合截面 |
| ρ_q | 0.15 | 低——膜材不储存灵气，只过滤 |
| Spec | [0.10, 0.05, 0.55, 0.05, 0.25] | 水属性主导（氢键网络吸附机制） |
| τ_d | >720 h | 膜本身不蓄灵，衰减概念不适用 |
| κ_q | 0.20 | 低——灵气通过膜时有意被减速以实现过滤 |
| P | 0.82 | 工业级纯度 |
| Tox | 0.02 | 几乎无毒 |
| Match_A | 38.5 | C 级——功能性辅材定位 |

**样本层认证**：`qualified`（sample_match=72.3）。滤灵膜作为功能性辅材，不追求 S 级认证；其价值在于稳定的净化效率而非灵气密度。

**产业链**

| 维度 | 信息 |
|------|------|
| 主要生产方 | 水处理企业转型（如膜科技公司灵材事业部） |
| 主要使用方 | 修炼场所空气净化、法器前置滤波、丹药提纯 |
| 价格区间 | 800–2,000 标准灵石单位/m² |
| 产业成熟度 | 工业量产 |

**技术瓶颈**：选择性不足——当前膜材对五行频谱的分辨率约 ±15%，无法精确过滤单一属性杂质。下一代目标是引入分子印迹技术实现 ±5% 精度。

### 3.2 导灵丝（碳纳米管纤维）

**现实原型**：多壁碳纳米管（MWCNT）纺丝纤维，直径 10–50 μm，拉伸强度 1–3 GPa，电导率 10⁴–10⁵ S/m。现实中用于航空航天复合材料、柔性电子和高强度缆绳。灵材化的核心在于碳纳米管的一维弹道传导特性与 LEVSS 灵气信号的低损耗传输高度匹配。

**灵材化关键工艺**

| 步骤 | 工艺 | 目的 |
|------|------|------|
| 1 | CVD 法生长定向 CNT 阵列 | 获得高取向度纳米管森林 |
| 2 | 湿法纺丝+拉伸取向 | 形成连续纤维，提高各向异性 |
| 3 | 高温退火（2800°C/Ar） | 消除缺陷，提高晶格有序度 |
| 4 | 稀土离子（Nd³⁺）掺杂 | 引入灵气频谱调谐位点 |
| 5 | 聚酰亚胺绝缘包覆 | 防止灵气侧向泄漏 |

**规则集 A 判定参数**

| 参数 | 值 | 说明 |
|------|-----|------|
| σ_q | 0.52 | 中高——高有序度碳结构的耦合效率 |
| ρ_q | 0.18 | 低——纤维截面小，总容量有限 |
| Spec | [0.45, 0.15, 0.05, 0.10, 0.25] | 金属性主导（声子纵模+高刚性） |
| τ_d | 48 h | 中等——纤维中灵气会沿轴向自然流失 |
| κ_q | 0.85 | 极高——当前人工材料中导灵率最高 |
| P | 0.88 | 高纯度（退火后缺陷密度 <10⁻³/nm） |
| Tox | 0.04 | 极低（包覆后纳米颗粒不外泄） |
| Match_A | 62.7 | A 级——优质导灵材料 |

**样本层认证**：`precert`（sample_match=81.2）。接近 certified 门槛，主要受限于 form_factor（纤维形态 vs 单晶样片的评分差异）。

**产业链**

| 维度 | 信息 |
|------|------|
| 主要生产方 | 碳纳米材料企业（如南瞻灵丝科技、东胜东丽灵材部） |
| 主要使用方 | 高端法器内部连线、军用灵气通信线缆、精密阵法布线 |
| 价格区间 | 15,000–45,000 标准灵石单位/km |
| 产业成熟度 | 小批量试产→工业量产过渡期 |

**技术瓶颈**：长度限制——当前单根连续纤维最长约 500m，超过此长度后缺陷累积导致 κ_q 急剧下降。目标是实现公里级无缺陷纺丝。

### 3.3 聚灵板（石墨烯薄膜）

**现实原型**：大面积单层/少层石墨烯薄膜，通过 CVD 法在铜箔上生长后转移。载流子迁移率 >10,000 cm²/(V·s)，比表面积 2630 m²/g（理论值）。现实中用于柔性电子、传感器和能量存储。灵材化的核心在于石墨烯的二维电子气特性提供了极大的灵气感知面积。

**灵材化关键工艺**

| 步骤 | 工艺 | 目的 |
|------|------|------|
| 1 | CVD 法铜箔上生长单层石墨烯 | 获得大面积高质量二维晶格 |
| 2 | PMMA 辅助湿法转移至 SiO₂/Si 基底 | 保持完整性的同时转移到功能基底 |
| 3 | 氮等离子体掺杂（N-doping） | 引入 p 型缺陷，调谐灵气频谱响应 |
| 4 | h-BN 封装 | 保护石墨烯免受环境污染，稳定灵气耦合 |
| 5 | 图案化电极阵列 | 实现空间分辨的灵气场探测 |

**规则集 A 判定参数**

| 参数 | 值 | 说明 |
|------|-----|------|
| σ_q | 0.58 | 中高——极大比表面积带来高灵气敏感性 |
| ρ_q | 0.12 | 低——单原子层厚度限制了总容量 |
| Spec | [0.40, 0.10, 0.10, 0.15, 0.25] | 金属性偏向（载流子主导+高有序度） |
| τ_d | 24 h | 较短——二维材料灵气保持能力有限 |
| κ_q | 0.78 | 高——面内传导极佳 |
| P | 0.90 | 高纯度（单层覆盖率 >95%） |
| Tox | 0.03 | 极低 |
| Match_A | 58.4 | B 级偏上——感知/传导双功能定位 |

**样本层认证**：`qualified`（sample_match=74.8）。石墨烯的二维形态在当前样本层框架中没有专属原型（介于薄膜和单晶之间），导致 form_factor 评分偏保守。

**产业链**

| 维度 | 信息 |
|------|------|
| 主要生产方 | 石墨烯企业（如南瞻第六元素、西极金洲 Graphene Flagship 成员） |
| 主要使用方 | 灵气场强计、高端探测器核心元件、灵气光谱分析仪 |
| 价格区间 | 8,000–25,000 标准灵石单位/m² |
| 产业成熟度 | 工业量产（探测器级）/ 小批量试产（高纯传导级） |

**技术瓶颈**：转移损伤——湿法转移过程中不可避免地引入褶皱和撕裂，导致实际 κ_q 比理论值低 30–40%。干法转移技术是突破方向。

### 3.4 灵芯片（稀土掺杂氧化物薄膜）

**现实原型**：HfO₂:La/Ce 铁电薄膜，厚度 5–20 nm，剩余极化 P_r ≈ 20–30 μC/cm²，耐久性 >10¹⁰ 循环。现实中用于 FeRAM 非易失存储器和神经形态计算芯片。灵材化的核心在于铁电畴翻转可以编码灵气信号的相位与频谱信息，实现"灵气集成电路"。

**灵材化关键工艺**

| 步骤 | 工艺 | 目的 |
|------|------|------|
| 1 | ALD 原子层沉积 HfO₂ 薄膜 | 亚纳米精度控膜厚 |
| 2 | La/Ce 离子注入（剂量 10¹⁴/cm²） | 稳定铁电正交相 |
| 3 | 快速热退火（500°C/30s/N₂） | 结晶化并激活铁电性 |
| 4 | TiN 顶/底电极溅射 | 形成 MFM 电容结构 |
| 5 | 光刻+刻蚀形成阵列 | 实现多位灵气信号存储/处理 |

**规则集 A 判定参数**

| 参数 | 值 | 说明 |
|------|-----|------|
| σ_q | 0.62 | 高——铁电极化提供强耦合 |
| ρ_q | 0.45 | 中高——薄膜阵列可堆叠扩容 |
| Spec | [0.15, 0.00, 0.10, 0.55, 0.20] | 火属性主导（铁电极化机制） |
| τ_d | 8,760 h（约 1 年） | 极长——非易失存储特性 |
| κ_q | 0.40 | 中——存储优先，传导非主业 |
| P | 0.92 | 极高（半导体级洁净室工艺） |
| Tox | 0.05 | 低（稀土氧化物生物惰性） |
| Match_A | 68.3 | A 级——核心储存/处理材料 |

**样本层认证**：`precert`（sample_match=83.6）。距 certified 仅差 evidence_bonus（当前 JARVIS 数据库中 HfO₂ 薄膜的匹配条目不足）。

**产业链**

| 维度 | 信息 |
|------|------|
| 主要生产方 | 半导体代工厂灵材专线（如南瞻灵芯制造、南瞻芯灵特种工艺线） |
| 主要使用方 | 灵气信号处理器、智能法器控制核心、灵气加密通信模块 |
| 价格区间 | 30,000–120,000 标准灵石单位/wafer（200mm） |
| 产业成熟度 | 小批量试产→工业量产过渡期 |

**技术瓶颈**：疲劳退化——铁电畴在 >10¹⁰ 次翻转后出现钉扎效应，导致灵气信号编码精度下降。目标是通过界面工程将耐久性提升至 10¹² 循环。

### 3.5 储灵瓶（锂离子电池改型）

**现实原型**：LiFePO₄/石墨体系锂离子电池，能量密度 150–200 Wh/kg，循环寿命 >3000 次。灵材化改造将电解液替换为含稀土盐的灵气耦合介质，使电化学储能过程同时耦合 LEVSS 灵气存储。

**灵材化关键工艺**

| 步骤 | 工艺 | 目的 |
|------|------|------|
| 1 | LiFePO₄ 正极材料碳包覆 | 提高电子/灵气双导率 |
| 2 | 石墨负极定向压延 | 增强层间灵气陷阱态 |
| 3 | 电解液改性（添加 0.5% CeCl₃） | 引入灵气耦合离子载体 |
| 4 | 陶瓷隔膜替代 PP 隔膜 | 提高灵气选择性透过率 |
| 5 | 铝塑膜+云母复合封装 | 灵气隔绝+电磁屏蔽双重防护 |

**规则集 A 判定参数**

| 参数 | 值 | 说明 |
|------|-----|------|
| σ_q | 0.48 | 中——电化学体系的耦合效率中等 |
| ρ_q | 0.65 | 高——体积储能优势明显 |
| Spec | [0.15, 0.05, 0.20, 0.25, 0.35] | 土属性偏向（层状结构+缓冲储能） |
| τ_d | 2,160 h（约 90 天） | 长——自放电率 <2%/月 |
| κ_q | 0.30 | 中低——储存优先，充放电速率受限 |
| P | 0.85 | 高（电池级纯度标准） |
| Tox | 0.08 | 低（LiFePO₄ 体系本身低毒，CeCl₃ 微量） |
| Match_A | 55.2 | B 级偏上——主力便携储能 |

**样本层认证**：`qualified`（sample_match=71.8）。电化学复合体系在当前样本层框架中缺乏专属原型，form_factor 按"致密烧结体"近似评估。

**产业链**

| 维度 | 信息 |
|------|------|
| 主要生产方 | 动力电池企业灵材产线（如南瞻时代灵储事业部、南瞻迪灵科技） |
| 主要使用方 | 便携法器供能、修炼室灵气缓冲、野外探险灵气储备 |
| 价格区间 | 1,500–6,000 标准灵石单位/kWh |
| 产业成熟度 | 工业量产 |

**技术瓶颈**：充放电速率——当前 C-rate 约 0.5C（2 小时充满），无法满足战斗法器的瞬时大功率需求。超级电容-储灵瓶混合方案是研发方向。

### 3.6 光导灵缆（光纤）

**现实原型**：掺铒光纤（EDF）与光子晶体光纤（PCF）。EDF 纤芯直径 4–8 μm，掺铒浓度 200–2000 ppm，工作波长 1530–1565 nm（C 波段），传输损耗 <0.2 dB/km。PCF 通过周期性空气孔阵列实现带隙导光，可定制色散和模场。现实中用于光通信干线、光纤激光器和光纤传感。灵材化的核心在于光子作为灵气信号载波，在光纤波导中实现低损耗远距传导——类比电信号在铜线中传输，但灵气信号的"载波"是受限光子模式。

**灵材化关键工艺**

| 步骤 | 工艺 | 目的 |
|------|------|------|
| 1 | MCVD 法制备掺铒预制棒 | 精确控制纤芯折射率分布和稀土掺杂浓度 |
| 2 | 拉丝塔高温拉制（2000°C） | 形成连续光纤，保持纤芯/包层同心度 |
| 3 | 紫外固化丙烯酸酯涂覆 | 机械保护+灵气侧向泄漏抑制 |
| 4 | 光子晶体结构引入（PCF 型） | 通过空气孔阵列增强灵气信号的模式约束 |
| 5 | 布拉格光栅刻写（FBG） | 在特定位置形成灵气频谱滤波/反射节点 |

**规则集 A 判定参数**

| 参数 | 值 | 说明 |
|------|-----|------|
| σ_q | 0.42 | 中——玻璃基质的耦合效率不如晶体 |
| ρ_q | 0.08 | 极低——光纤不储存灵气，只传输 |
| Spec | [0.35, 0.05, 0.30, 0.10, 0.20] | 金/水双属性（光子传导+波导结构中的 SiO₂ 氢键） |
| τ_d | N/A | 传输型材料，不适用衰减概念 |
| κ_q | 0.72 | 高——长距离传导损耗极低（<0.5 dB/km 灵气等效） |
| P | 0.86 | 高（光纤级纯度，OH⁻ 含量 <1 ppm） |
| Tox | 0.01 | 几乎无毒（SiO₂ 玻璃+丙烯酸酯涂层） |
| Match_A | 48.6 | B 级——专用远距传导材料 |

**样本层认证**：`qualified`（sample_match=69.4）。光纤的非晶态玻璃基质在 Order 维度天然吃亏，但其传导功能角色不依赖晶格有序度。

**产业链**

| 维度 | 信息 |
|------|------|
| 主要生产方 | 光纤企业灵材产线（如南瞻长飞灵导事业部、鹰洲鹰洲康宁灵缆部） |
| 主要使用方 | 修炼场所间灵气信号传输、大型阵法远程互联、灵气通信网络 |
| 价格区间 | 2,000–8,000 标准灵石单位/km（标准型）；50,000+/km（PCF 定制型） |
| 产业成熟度 | 工业量产（标准型）/ 小批量试产（PCF 型） |

**技术瓶颈**：功率承载——当前光导灵缆的灵气信号功率上限约 10 mW 等效，超过此值会引发非线性效应（受激布里渊散射的灵气等效），导致信号畸变。高功率传输需要大模场面积 PCF，但成本是标准型的 10 倍以上。

### 3.7 共振阵列（相控阵天线改型）

**现实原型**：X 波段有源相控阵天线（AESA），单元数 64–1024，单元间距 λ/2，波束扫描范围 ±60°，波束切换时间 <10 μs。现实中用于雷达、卫星通信和 5G 毫米波基站。灵材化的核心在于将电磁波的相位控制原理移植到灵气场——通过多个灵气发射单元的相位协调，形成可编程的定向灵气场，即"现代版法阵"。

**灵材化关键工艺**

| 步骤 | 工艺 | 目的 |
|------|------|------|
| 1 | 压电陶瓷（PZT）阵列制备 | 每个单元作为独立灵气发射/接收节点 |
| 2 | 灵芯片集成相位控制器 | 精确控制每个单元的灵气信号相位 |
| 3 | 导灵丝互联网络布线 | 单元间低损耗信号传输 |
| 4 | 钛合金框架+云母隔层封装 | 结构支撑+单元间灵气串扰隔离 |
| 5 | 校准与波束赋形编程 | 实现定向聚灵、扫描投射、多波束并行 |

| 参数 | 值 | 说明 |

| σ_q | 0.70 | 高——多单元协同的等效耦合截面远超单体 |

| Match_A | 72.5 | A 级——高端投射/聚灵系统 |

**样本层认证**：`precert`（sample_match=79.8）。共振阵列作为系统级产品，其认证需要整体评估而非单一材料评估——这是当前样本层框架尚未完全覆盖的场景。

## §4 新兴人造灵材（2030–2035 前沿）

> 以下材料均处于实验室或概念验证阶段，尚未进入 [[01_现实矿物灵材数据库首轮筛查]] 的正式数据库。Match_A 为基于现实材料参数的预估值，置信度 Conf ≤ 0.5。

### 4.1 量子点灵墨（QD-Ink）

| 维度 | 信息 |
|------|------|
| 现实原型 | CdSe/ZnS 核壳量子点胶体墨水，粒径 2–8 nm 可调，荧光量子产率 >80% |
| 技术原理 | 量子点的尺寸依赖能级结构使其可以精确调谐到特定 LEVSS 耦合频率；胶体形态允许喷墨打印，实现灵气电路的快速原型制造 |
| 灵材化潜力 | 替代传统朱砂绘线的符箓制作方式——从手工绘制升级为数字化打印。每个量子点就是一个纳米级灵气耦合节点，阵列化后可形成复杂灵气电路 |
| 五行频谱 | 可调——粒径 2 nm 偏火（高能级），8 nm 偏水（低能级），混合粒径可实现复合属性 |
| 当前状态 | 实验室阶段（清华大学灵材实验室、鹰洲理工量子灵学组） |
| 预估 Match_A | 45–65（取决于量子点组成和基底材料） |
| 关键挑战 | CdSe 含镉剧毒（Tox ≈ 0.45），InP 无毒替代方案的耦合效率仅为 CdSe 的 60% |

### 4.2 拓扑灵导体（Topo-Conductor）

| 维度 | 信息 |
|------|------|
| 现实原型 | Bi₂Se₃ / Bi₂Te₃ 拓扑绝缘体薄膜，表面态狄拉克锥，自旋-动量锁定 |
| 技术原理 | 拓扑绝缘体的表面态受拓扑保护，不受非磁性杂质散射——灵气信号在表面传导时理论上零损耗，即使材料有缺陷 |
| 灵材化潜力 | "灵气超导体的室温替代方案"。YBCO 超导灵核需要液氮温度，拓扑灵导体在室温下就能实现近零损耗传导（仅限表面态） |
| 五行频谱 | [0.50, 0.00, 0.25, 0.05, 0.20]——金属性主导（载流子传导），水属性次之（含硒/碲的氢键类耦合） |
| 当前状态 | 实验室阶段（南瞻物理研究所、鹰洲拓扑量子中心） |
| 预估 Match_A | 55–75（表面态 κ_q 极高，但体态 ρ_q 几乎为零） |
| 关键挑战 | 表面态仅存在于最外层几个原子层，灵气信号的功率承载极低；Bi₂Se₃ 在空气中快速氧化，需要 h-BN 或 MoS₂ 封装 |

### 4.3 仿生灵膜（Bio-Membrane）

| 维度 | 信息 |
|------|------|
| 现实原型 | 仿生离子通道膜——基于脂质双分子层嵌入蛋白质通道的人工膜，选择性离子透过率 >10⁴ |
| 技术原理 | 模拟人体经脉的选择性传导机制：特定五行属性的灵气可以通过，其他属性被阻断。核心是蛋白质通道的构象变化响应灵气频谱 |
| 灵材化潜力 | 真正的"人工经脉"——可以植入法器或修炼辅助设备中，实现对灵气流的精确分流和属性筛选 |
| 五行频谱 | [0.05, 0.55, 0.20, 0.05, 0.15]——木属性主导（生物相容性+手性结构） |

| 关键挑战 | 脂质膜在非水环境中不稳定，寿命仅数小时；蛋白质通道的灵气频谱选择性尚未实现可编程控制 |

### 4.4 灵气超材料（Meta-Spirit）

| 维度 | 信息 |
|------|------|
| 现实原型 | 电磁超材料（metamaterial）——亚波长周期结构实现负折射、隐身斗篷等反常电磁响应 |
| 技术原理 | 将超材料的亚波长结构设计原理应用于 LEVSS 灵气场：通过精心设计的灵材微结构阵列，实现灵气场的负折射、聚焦和波导——"灵气透镜" |
| 灵材化潜力 | 灵气隐身斗篷（引导灵气绕过目标）、超分辨灵气成像（突破衍射极限的灵气探测）、平面灵气透镜（替代笨重的共振阵列） |

### 4.5 相变储灵体（Phase-Store）

| 维度 | 信息 |
|------|------|
| 现实原型 | Ge₂Sb₂Te₅（GST）相变材料，非晶↔晶态可逆转变，转变时间 <50 ns，循环寿命 >10⁹ |

### 4.6 磁流灵液（Ferro-Flow）

## §7 与规则集 A 的接口

### 7.1 人造灵材如何接入 Match_A 评分

人造灵材使用与天然灵材完全相同的 [[02_物理公式与参数手册]] §4 翻译函数 A。输入字段不变：

```text
Crystal → Order    （人造材料通常有明确的晶系/结构数据）
d33 → Polar        （压电系数可从数据手册直接获取）
σ_e / E_g → Carrier / Trap
Stab → Stability
P → Purity         （工业品有精确的纯度检测报告）
Tox → ToxPenalty
Form → FormFactor
```

关键差异在于**数据置信度 Conf**：

| 数据来源 | 天然灵材 | 人造灵材 |
|----------|---------|---------|
| 晶系/结构 | 矿物数据库（q=1.0） | 制造商规格书（q=1.0） |
| 压电/铁电 | 文献区间值（q=0.7） | 实测值（q=1.0） |
| 导电/带隙 | JARVIS 计算值（q=0.7）或缺失（q=0.0） | 实测值（q=1.0） |
| 毒性 | 矿物毒性数据库（q=0.7） | GHS 安全数据表（q=1.0） |
| 纯度 | 估算或检测（q=0.4–1.0） | 批次检测报告（q=1.0） |

结论：人造灵材的 Conf 通常显著高于天然灵材（0.85–0.95 vs 0.50–0.75），这在 Match_A 公式中通过 `c_6 × Conf` 项提供 +3 到 +8 分的加成。

### 7.2 人造灵材的特殊判定规则

#### 7.2.1 σ_q 偏高但 τ_d 偏短

人造灵材的一个系统性特征是：工程优化提高了耦合效率（σ_q↑），但人工结构的热力学亚稳态导致灵气保持时间缩短（τ_d↓）。

```text
典型对比：
  天然灵石单晶：σ_q=0.65, τ_d=数年
  灵芯片薄膜：  σ_q=0.62, τ_d=约1年
  导灵丝纤维：  σ_q=0.52, τ_d=48小时
```

这意味着人造灵材更适合"即时使用"场景（法器运行、信号传输），而非"长期储存"场景（灵石货币、传承法器核心）。

#### 7.2.2 Form 字段的扩展

天然灵材的 Form 字段在 [[03_灵材样本层与工艺认证]] 中定义了明确的原型组（原矿/精粉/多晶/单晶等）。人造灵材需要扩展以下形态：

| 新增 Form | 适用产品 | form_factor 建议值 | 说明 |
|-----------|---------|-------------------|------|
| `thin_film` | 灵芯片、聚灵板 | 0.88 | 薄膜形态，高有序度但体积极小 |
| `fiber` | 导灵丝 | 0.82 | 纤维形态，一维高取向 |
| `composite_membrane` | 滤灵膜 | 0.65 | 复合膜，部分有序 |
| `electrochemical_cell` | 储灵瓶 | 0.70 | 电化学体系，多相复合 |
| `phased_array` | 共振阵列 | 0.85 | 系统级集成，整体有序度高 |
| `glass_fiber` | 光导灵缆 | 0.60 | 非晶态玻璃，Order 天然偏低 |

#### 7.2.3 evidence_bonus 的获取

人造灵材在 evidence_bonus 上有天然优势：

- `jarvis_matched`：人造材料的化学式通常在 JARVIS 数据库中有精确匹配（+1）
- `piezo_class` / `polar_class`：压电/铁电性能有明确的实测数据（+1）
- `magnetic_score`：磁性数据可从制造商获取（条件性 +1）

因此人造灵材更容易满足 `evidence_bonus >= 2` 的 certified 门槛——但它们在 purity 和 form_factor 上不一定能达到单晶样片的水平。

### 7.3 样本层认证的特殊考量

#### 7.3.1 process_factor 天然更高

人造灵材的 process_factor 反映的是工业制造工艺的成熟度，而非矿物加工工艺。由于工业品的工艺可重复性远高于矿物加工：

```text
天然灵材 process_factor 范围：0.30（原矿）– 0.90（单晶样片）
人造灵材 process_factor 范围：0.70（实验室原型）– 0.95（GMP 量产品）
```

这意味着人造灵材在样本层公式中的 `sample_stability` 和 `sample_confidence` 项天然占优。

#### 7.3.2 为什么人造灵材仍然难以达到 S 级

尽管 process_factor 高，人造灵材在以下维度受限：

1. **σ_q 上限**：人造材料的耦合截面受限于基础材料物理——碳纳米管再怎么优化也不如天然压电单晶
2. **form_factor 天花板**：薄膜（0.88）和纤维（0.82）的 form_factor 低于单晶样片（0.96）
3. **sample_match 公式中 σ_q 权重最高**（2.05），而 process_factor 的贡献通过 sample_stability 间接传递，权重较低

```text
验证计算（导灵丝 vs 石英单晶）：
  导灵丝：σ_q=0.52, sample_rho_q≈0.38, sample_eta_cond≈0.72, Tox=0.04
  → sample_match ≈ 81.2（precert，差 certified 约 3 分）
  
  石英单晶：σ_q=0.65, sample_rho_q≈0.58, sample_eta_cond≈0.68, Tox=0.02
  → sample_match ≈ 96.7（certified S 级）
```

> **设计意图**：规则集 A 对人造灵材没有歧视也没有优待——它用同一套物理规则评判所有材料。人造灵材的优势（高 Conf、高 process_factor、低 Tox）和劣势（σ_q 上限、form_factor 天花板）都是物理现实的自然结果。这确保了"天然 S 级单晶仍然是最稀缺的顶级资源"这一世界观锚点不被人造灵材冲垮。

## §8 叙事钩子

### 8.1 "灵芯片禁运"事件

2033 年，鹰洲宣布对南瞻部洲实施 7nm 以下灵芯片制造设备出口管制。南瞻芯灵特种线被迫停产三个月，导致洲内智能法器供应链断裂。QESA 紧急启动"灵芯自主计划"，从全洲高校和研究所抽调顶尖人才。

> **游戏场景**：玩家如果走学院派路线，可能被卷入这场"灵芯攻关"——在实验室里日夜赶工，面对技术瓶颈和政治压力的双重考验。成功突破可获得 QESA 特殊贡献勋章和稀缺灵芯片样品；失败则面临项目裁撤和学术前途受损。

### 8.2 "导灵丝之争"

南瞻灵丝科技"灵丝科技"声称实现了公里级无缺陷 CNT 纺丝，但东胜东丽质疑其数据造假。双方在 IVCERA 年会上公开对峙，最终 IVCERA 要求第三方实验室独立验证。验证过程中发现灵丝科技的样品确实存在 200m 处的隐性缺陷——但东丽的样品在 800m 处也有类似问题。

> **游戏场景**：玩家可能作为第三方验证实验室的成员参与此事件。选择如实报告（得罪双方但获得学术声誉）、偏向一方（获得该方资源但失去另一方信任）、或发现更深层的技术秘密（开启隐藏任务线）。

### 8.3 "量子点灵墨泄露"

南瞻生物灵学交叉实验室的一名博士生将含镉量子点灵墨的制备配方泄露给黑市。黑市工坊用廉价原料仿制，生产出大量低纯度、高毒性的"山寨灵墨"，导致多名使用者出现镉中毒症状。QESA 介入调查，发现泄露链条涉及学术腐败、黑市网络和跨境走私。

> **游戏场景**：玩家可能从多个角度介入——作为中毒受害者追查真相、作为 QESA 调查员追踪泄露链、作为黑市买家意外卷入、或作为实验室同事面对举报与沉默的道德抉择。

### 8.4 "磁流灵液实验事故"

南瞻磁性材料实验室在测试大规模磁流灵液阵列时，外加磁场突然失控，磁流体在实验室内形成不可控的灵气涡旋。三名研究员被困在涡旋中心，灵气过载导致走火入魔前兆症状。最终由一名古法派访问学者用传统"镇灵术"稳定了涡旋，为现代救援争取了时间。

> **游戏场景**：这个事件完美展示了"古法 vs 现代"的互补关系——现代技术制造了问题，古法经验解决了问题。玩家如果同时修习两派技能，在类似危机中会有独特优势。

### 8.5 "超材料隐身斗篷"军方竞标

DARPA-Q 和南瞻电科同时宣布在灵气超材料隐身斗篷上取得突破。双方都声称能实现"灵气探测不可见"，但技术路线完全不同——鹰洲方用金属 SRR 阵列，南瞻方用介质超材料。一场秘密的技术竞赛在两洲军方之间展开，民间修士和学术界被卷入其中。

> **游戏场景**：军方路线的玩家可能被要求参与超材料测试或情报收集。学术路线的玩家可能发现自己的基础研究被军方"征用"。散修路线的玩家可能在黑市上遇到泄露的超材料样品——性能惊人但来源不明。

### 8.6 "仿生灵膜与人体改造"伦理争议

西极仿生研究所宣布成功将仿生灵膜植入实验动物体内，实现了"人工经脉"的概念验证。消息传出后，修仙界炸锅——古法派认为这是"逆天改命"的禁忌，学院派认为这是修炼民主化的希望，伦理委员会紧急叫停了人体实验申请。但地下组织已经开始秘密进行人体植入实验。

> **游戏场景**：这是一个没有简单答案的伦理困境。玩家可能需要在"技术进步"和"伦理底线"之间做出选择——支持研究可能拯救无数灵根不佳的普通人，但也可能打开潘多拉魔盒。

## §9 一句话结论

> **人造灵材不是天然灵材的替代品，而是修仙文明从"靠天吃饭"走向"工业化"的标志。七大核心产品已经改变了中低阶修炼的面貌，六种前沿概念材料正在酝酿下一次技术革命——但天然 S 级单晶仍然是不可撼动的顶级资源，因为物理规律不会因为人类的工程能力而改变。**
