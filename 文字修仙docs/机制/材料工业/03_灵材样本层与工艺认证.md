# 03_灵材样本层与工艺认证

> **一句话版**：矿物层只负责判断“这是什么矿”，第二层样本层负责判断“你把它做成了什么”。同一种现实矿物在 `原矿 / 精粉 / 定向多晶 / 单晶样片 / 生体陶瓷 / 定向晶胚` 等不同工艺形态下，会得到完全不同的灵材认证结论。
> **关联文档**：[[02_物理公式与参数手册]] · [[01_现实矿物灵材数据库首轮筛查]] · [[02_灵材命名体系与批量转译规则]]

---

## 1. 为什么必须有第二层

如果只有“矿物物种层”，系统会出现一个很大的失真：

- `Quartz` 很强，但天然石英矿和工业单晶石英显然不是一回事
- `Hydroxylapatite` 能修复生体组织，但粉末、陶瓷和定向晶胚的表现完全不同
- `Sphalerite` 具备雷系路线潜力，但真正能上高阶器件的一定不是粗矿砂

所以第二层必须存在，而且它回答的问题与第一层不同：

| 层级 | 问题 | 例子 |
|------|------|------|
| 矿物层 | 这个现实矿种有没有蓄灵潜力？ | `Quartz` 能不能算灵材原型 |
| 样本层 | 这个矿被做成什么形态后，能不能真正稳定使用？ | `Quartz` 单晶样片能不能正式 `S` |

这层一接上，世界观就从“材料分类”变成了“材料工程”。

## 2. 第二层的数据产物

当前已经实际生成两类产物：

- CSV：`data/processed/rruff_mineral_sample_profiles.csv`
- SQLite 表：`sample_profiles`（位于 `data/processed/rruff_mineral_screening.sqlite`）

每条样本记录都会保存：

- 基础矿物信息：`mineral_name / formula / primary_affinity / role_label / risk_label`
- 矿物层结论：`mineral_grade / mineral_grade_raw / mineral_s_cert_tier`
- 样本工艺信息：`sample_form / sample_form_short / sample_purity / sample_form_factor / sample_process_factor`
- 第二层计算结果：`sample_match / sample_grade_raw / sample_grade / sample_cert_tier / sample_cert_score`
- 命名结果：`sample_engineering_name / sample_common_name`

也就是说，第二层已经不是“口头设想”，而是数据库里真实存在的一张表。

## 4. 第二层判定公式

### 4.1 样本中间量

当前实现里，样本层先从矿物层派生出五个中间量：

```text
sample_confidence = 0.65 * base_confidence + 0.20 * purity + 0.15 * process_factor
sample_stability = 0.72 * base_stability + 0.18 * process_factor + 0.10 * form_factor - raw_ore_penalty
sample_leak = base_leak * (1.08 - 0.36 * form_factor)
sample_rho_q = sigma_q * purity * form_factor * (0.85 + 0.15 * process_factor)
sample_eta_cond = eta_cond * (0.78 + 0.22 * form_factor) - 0.10 * (1.0 - process_factor)
```

这里的含义非常直白：

- 纯度越高，储灵密度越稳定
- 形态越有序，漏损越低
- 工艺越成熟，可重复性越高
- 原矿天然吃亏，因为它本来就带杂质、缺陷和缺乏定向

### 4.2 样本总分

```text
sample_match = 100 * sigmoid(
    -1.70
    + 2.05 * sigma_q
    + 1.25 * sample_rho_q
    + 0.75 * sample_eta_cond
    + 0.60 * sample_stability
    - 1.55 * toxicity
    - 1.10 * sample_leak
    + 0.50 * sample_confidence
)
```

这个公式的设计意图是：

- 第一层的“矿物潜力”仍然保留权重
- 但第二层真正决定能否冲到顶级的是纯度、定向性、工艺质量
- 毒性和漏损继续作为硬惩罚存在

### 4.3 证据加成

第二层还会统计 `evidence_bonus`：

- `jarvis_matched = 1`：`+1`
- `piezo_class = 1` 或 `polar_class = 1`：`+1`
- `magnetic_score >= 0.55`：`+1`

这样做的原因是：

> **第二层不是只看“做得精不精”，还要看它有没有足够硬的现实物理证据支撑。**

### 4.4 第二层认证门槛

#### 正式 `certified`

必须同时满足：

- `sample_match >= 84.0`
- `purity >= 0.90`
- `form_factor >= 0.90`
- `process_factor >= 0.84`
- `sample_leak < 0.14`
- `toxicity < 0.12`
- `evidence_bonus >= 2`

#### `precert`

必须同时满足：

- `sample_match >= 78.0`
- `purity >= 0.82`
- `form_factor >= 0.76`
- `process_factor >= 0.72`

#### `qualified`

- `sample_match >= 68.0`

#### 其余情况

- 安全但不过线：`raw`
- 危险候选：`restricted`
- 非安全候选：`blocked`

### 4.5 第二层等级修正

第二层还有一个很关键的判定：

```text
只有 sample_cert_tier = certified 时，sample_grade 才能真正记为 S。
否则即使 sample_grade_raw = S，也会被压回 A。
```

这条规则就是为了避免“高分但未认证样本”把正式顶材名额冲烂。

## 6. 关键原型案例

### 6.1 Quartz：第二层直接证明“灵石不是随便一块石英”

| 样本形态 | 样本俗名 | 等级 | 认证 | 认证分 |
|----------|----------|------|------|--------|
| `单晶样片` | `真灵石·单晶` | `S` | `certified` | `96.728133` |
| `定向多晶` | `上品灵石·定向体` | `A` | `precert` | `88.941736` |
| `精炼粉体` | `精制灵石` | `A` | `qualified` | `76.370610` |
| `天然原矿` | `精制灵石` | `A` | `qualified` | `58.423483` |

这意味着：

- `灵石` 作为文化原型成立
- 但真正顶级的“灵石”，其实更像工业化单晶器件材料

### 6.2 Sphalerite：雷灵砂路线接近成熟工业标准

| 样本形态 | 样本俗名 | 等级 | 认证 | 认证分 |
|----------|----------|------|------|--------|
| `单晶样片` | `上品雷灵砂·单晶` | `A` | `precert` | `93.941022` |
| `定向多晶` | `上品雷灵砂·定向体` | `A` | `precert` | `86.013224` |
| `精炼粉体` | `精制雷灵砂` | `A` | `qualified` | `73.134019` |

它已经足够作为高阶雷系法器原料，但还没达到正式 `S`。

### 6.3 Hydroxylapatite：水路线的“生体工艺化”特别明显

| 样本形态 | 样本俗名 | 等级 | 认证 | 认证分 |
|----------|----------|------|------|--------|
| `定向晶胚` | `上品水灵珠·晶胚` | `A` | `precert` | `87.580415` |
| `生体陶瓷` | `上品水灵珠·生陶` | `A` | `precert` | `82.475600` |
| `精制珠胚` | `精制水灵珠` | `A` | `qualified` | `72.010318` |

它的第二层结果非常有味道，因为它天然就更像：

- 生体修复材料
- 筑基期替换骨料
- 丹药载体与修复性器官材料

### 6.4 Galena：第二层不会替危险路线洗白

`Galena` 在矿物层就没有进入可安全修炼候选，因此当前没有向下生成常规样本画像。

这条设计很重要，因为它阻止了下面这种不合理情况：

- 只要工艺做得够精，所有毒矿都能强行白洗成大众修炼资源

当前系统不允许这种事情发生。

## 7. 第二层给世界观带来的真正变化

### 7.1 从“矿物目录”进入“材料工业”

第二层接上之后，世界观会自然出现：

- 灵材纯化厂
- 定向晶胚实验室
- 生体陶瓷生产线
- 单晶样片专利与监管体系
- 大宗原矿和高端样片之间的巨大产业差价

这比单纯写“某矿可修炼”更真实得多。

### 7.2 从“硬编码物品”进入“开放输入”

以后你在现实互联网搜到一个新材料，只要能拿到近似字段：

- 成分
- 晶型/点群
- 极化/压电/磁性/带隙信息
- 样本纯度与工艺形态

它就能直接接进第二层。

也就是说，你不需要再手工给它编一套数值，只需要验证它是否匹配世界观规则。

### 7.3 为什么这对游戏特别重要

因为文字修仙游戏最怕“材料系统写死后无法扩容”。

第二层一旦稳定，后面可以自然长出：

- 炼器配方树
- 样本制备路线图
- 灵材交易品级体系
- 阵法核心适配要求
- 不同境界对样本纯度和形态的门槛

这会让“修仙”越来越像一个真的工业文明，而不是一堆散装名词。

## 8. 一句话结论

> **第二层已经把“现实矿物是否有仙骨”推进成了“现实材料如何被工业化地做成仙骨”。正式 `S` 样本现在主要集中在单晶样片和定向晶胚，这说明顶级修仙材料并不是天然神物，而是高纯度、高定向、高工艺的现实材料工程产物。**
