# 05_AI 道侣与周报

> **一句话版**：AI 道侣不只是聊天机器人，而是你的「LEVSS Monitor APP」——周报汇总你的修炼数据，帮你发现趋势和风险。
> **关联文档**：[[02_UI与信息架构]] · [[03_叙事节奏控制]] · [[14_成就系统]]
> **代码对照**：`services/weekly_report.py` · `core/day_manager.py`

---

## §1 设计定位

AI 道侣的语气定位：**赛博修仙的 LEVSS Monitor APP**，不是古风仙侠的「道友」。

| 场景 | 语气 | 示例 |
|------|------|------|
| 数据卡 | 冷静专业 | 「LEVSS Monitor 提示：InputRate 提升 12.3%」 |
| 周报 | 客观分析 | 「📰 本周状态：练气期，疲劳 32%，趋势平稳」 |
| 濒死警告 | 紧急直白 | 「⚠️ 生命体征危急，建议立即休息」 |
| 成就解锁 | 简洁鼓励 | 「🏆 安全窗口 — 你学会了控制 Risk」 |

## §2 周报系统

每 7 tick 自动生成，包含 6 个板块：

### 1. 个人状态总结
```
🟢 本周状态：练气期，疲劳 15%，污染 8%，心性 72%，资金 1340 SQU
```
状态灯：🟢（良好）/ 🟡（注意）/ 🔴（危险）

### 2. 修炼进度
```
📈 熟练度：修炼 52%，炼丹 28%，采集 15%
```

### 3. 世界动态
```
🕊️ 局势平稳，适合安心修炼
☁️ 全局灵气污染偏高，注意净化（当 global_pollution > 0.3）
⚔️ 势力紧张局势加剧，外出需谨慎（当 hot_war_index > 0.5）
```

### 4. 势力关系
```
🤝 与长安灵大关系最佳（45）
```

### 5. 任务提醒
```
⏰ 2 个任务即将到期！
```

### 6. 建议
```
💡 建议：污染累积较多，考虑净化或换个地方
```
