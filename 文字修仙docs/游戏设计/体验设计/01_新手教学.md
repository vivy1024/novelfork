# 新手教学设计 · 从零到练气的完整上手体验

> **一句话版**：新手教学由三层系统组成——TutorialHints（就地气泡提示）+ TeachingWheels（胎息六轮任务）+ Story Volume 1（主线叙事驱动）。三层互补不重叠，老手全部自动跳过。
>
> **关联文档**：[[02_玩家交互范式]] · [[01_序章脚本：灵气回归之日]]
>
> **实现文件**：
> - `client/src/components/panels/TutorialHints.vue`
> - `runtime/data/teaching_wheels.yaml`
> - `runtime/data/events/story.yaml`

## 一、三层教学系统概览

```
┌─────────────────────────────────────────────────────────┐
│  Layer 1 · TutorialHints（就地气泡）                      │
│  触发：状态条件自动检测 | 关闭：localStorage 永久记住     │
│  特点：非打断式，贴近游戏元素，老手不会看到               │
├─────────────────────────────────────────────────────────┤
│  Layer 2 · TeachingWheels（胎息六轮）                     │
│  触发：进入游戏自动启动 | 关闭：六轮全部完成               │
│  特点：导师对话+实操任务+数据回顾，主动学习公式            │
├─────────────────────────────────────────────────────────┤
│  Layer 3 · Story Volume 1（主线叙事）                     │
│  触发：天数/行为/境界条件依次解锁 | 关闭：突破练气后卷末  │
│  特点：在游戏的裂缝里揭示世界观谜团，叙事驱动探索欲        │
└─────────────────────────────────────────────────────────┘
```

三层分工：

| 层 | 回答的问题 | 不回答的问题 |
|---|-----------|------------|
| TutorialHints | "这个按钮是什么" / "现在该干什么" | 世界观、系统深度 |
| TeachingWheels | "这个公式是什么意思" / "如何选灵材" | 剧情、势力、世界 |
| Story Volume 1 | "这个世界是什么" / "为什么灵气回归" | 操作技巧、公式 |

## 二、Layer 1 · TutorialHints（就地气泡）

### 2.1 设计原则

- **不用弹窗**，不中断游戏流程
- 气泡贴近相关 UI 元素（ScenePanel 中，位于地点信息下方）
- 最多同时显示 2 条
- 关闭后 `localStorage` 永久记住，不再复现
- **老手自动识别**：`tick > 20` 或已到练气境界时，整个组件不渲染

### 2.2 8条提示的触发逻辑

| ID | 触发条件 | 标题 | 核心引导 |
|----|---------|------|---------|
| `first_morning` | tick≤2 + 晨报阶段 + 消息为空 | 🌅 早安，新修士 | 引导点"查看晨报" |
| `first_action` | tick≤3 + 行动阶段 + 消息≤3 | ⚡ 选择第一个行动 | 教会点修炼卡 |
| `fatigue_warning` | fatigue≥50% + 新手 | 😮‍💨 疲劳影响效率 | 优先"休息/就寝" |
| `risk_hint` | risk≥15% + tick≤15 | ⚠️ 风险四分量 | 悬停状态条看详情 |
| `market_hint` | tick 3-8 + 背包空 | 🏪 灵材提升效率 | 去市场买灵材 |
| `teaching_hint` | 胎息 + tick 2-20 | 📚 教学六轮进行中 | 沈霖在等你完成任务 |
| `pollution_hint` | pollution≥25% + 胎息 | 🌫️ 污染需净化 | 影响突破成功率 |
| `breakthrough_ready` | lFit≥48 + 胎息 + 低污染 | ✨ 接近突破门槛 | sticky提示，注意时机 |

## 三、Layer 2 · TeachingWheels（胎息六轮）

### 3.1 导师角色：沈霖

**沈霖 · 应用灵学系副研究员**

- 练气七层，修了九年，科班出身
- 不信玄学，相信一切可以被测量
- 说话直接，偶尔透出对"修仙变成工程"的某种惆怅
- 自己的入学 L_fit 只有 0.31

沈霖是整个教学叙事的锚点——她在六轮里从"评估你"变成"信任你"最后"放手让你走"。

### 3.2 六轮结构

| 轮 | 名称 | 公式 | 任务 | 沈霖的基调 |
|----|------|------|------|-----------|
| 1 | 玄景轮 | 灵气浓度 Q + InputRate | 住处&学院各修炼1次，对比差异 | "告诉我你感觉到的区别" |
| 2 | 承明轮 | 风险四分量 Risk(t) | 加速修炼到黄色预警 | "我当年也做过这个测试" |
| 3 | 周行轮 | 经脉阻抗与导灵损耗 | 采集灵材 + 修炼对比 | "你的经脉比我当年通畅" |
| 4 | 青元轮 | 五行频谱 Spec | 购买/采集灵材 + 修炼 | "你的灵根图已经定型了" |
| 5 | 玉京轮 | 灵根适配度 L_fit | 研读功法 + 修炼 | 罕见地在白板写公式 |
| 6 | 灵初轮 | HardGate/SoftGate | 探索 + 净化 | 没有石板，只有冷茶 |

### 3.3 三段式节奏（每轮）

```
intro（导师对话）
  ↓ 老手可跳，新手建议阅读
task（实操任务）
  ↓ 不可跳，必须实际操作
review（数据回顾 + 知识解锁）
  ↓ 公式永久写入知识面板
```

### 3.4 六轮叙事弧线

```
第1轮：沈霖像在评估你 → 第3轮：开始信任 → 第6轮：放手
```

**第六轮 灵初轮 review 的最后一句**：

> "九年前我突破练气的那天，我导师也是这么和我说的。我当时觉得这句话是废话。现在我也说给你听，因为你以后会明白它的意思。"

## 四、Layer 3 · Story Volume 1（主线叙事）

### 4.1 第一卷：「灵气回归之年」

**主线谜团**：灵气为什么在上升？不是自然现象——是有人打开了门。

**核心揭示路径**（11节点，Day1~练气突破后）：

```
幕一（入场）  → 幕二（初次异常）→ 幕三（第一个选择）
Day 1-7           Day 8-15            Day 16-25
扩招谜题         QESA 布监测点         是否偷看限制文件
                 市场老张的暗示        ↗看 / ↘不看

幕四（世界活了）→ 幕五（真相边缘）→ 幕六（跃迁）
Day 26-40         Day 41-55           Day 56+
目击突破           沈霖说一半真相       突破感受描写
灰巷陌生人         突破前夜             沈霖说出完整真相
```

**卷末揭示**：

> "三年前，一个没有任何官方认可的民间小组，在南瞻地下打开了稳定的 LEVSS 节点。然后 QESA 来了，把那扇门接管了。然后学院拿到了灵气监测权。然后今年开始扩招。你是其中一个被扩招进来的人——明白为什么我说你不是偶然的了吗？"

### 4.2 节点类型

| narrative_type | 作用 |
|---------------|------|
| `world_reveal` | 揭示世界观机制（LEVSS/QESA/灵气来源） |
| `character_reveal` | 揭示人物（沈霖的困惑和立场） |
| `faction_reveal` | 揭示势力关系（QESA的管控方式） |
| `choice_point` | 分支选择（影响信息获取和关系） |
| `milestone` | 关键里程碑（突破、卷末） |
| `volume_end` | 卷末总结 + 下一卷预告 |

### 4.3 与 TeachingWheels 的关系

两者并行不重叠——故事节点不讲公式，教学任务不讲世界谜团：

| Day | TeachingWheels | Story |
|-----|---------------|-------|
| 1-3 | 玄景轮（学 Q） | `day1_arrival` + `tutor_first_meet` |
| 8-12 | 承明轮（学 Risk） | `qesa_visit` + `market_rumor` |
| 16-25 | 周行轮（学阻抗） | `forbidden_data`（选择节点） |
| 26-35 | 青元轮（学五行） | `npc_breakthrough_witnessed` |
| 40-50 | 玉京轮/灵初轮 | `shen_lin_truth` + `pre_breakthrough_night` |
| 突破后 | — | `breakthrough_moment` + `shen_lin_after_breakthrough` |

## 五、老手识别机制

### 读档进入（老玩家）

- TutorialHints：`tick > 20` → 组件不渲染
- TeachingWheels：后端检查 `wheels_completed` 列表，已完成的不再触发
- Story：`story_flags` 持久化，读档后自动跳过已触发节点

### 跳过选项

- `teaching_wheels.yaml` 中每轮 `intro.skip_allowed: true`
- 老手可跳叙事，不可跳实操任务
