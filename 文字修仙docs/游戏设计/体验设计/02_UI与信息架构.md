# 02_UI 与信息架构

> **一句话版**：信息是修仙世界的第一生产力——UI 的核心任务是「让玩家看懂公式、做出决策」，而非展示华丽特效。

## §1 布局架构

采用 Card Survival 四区布局，适配文字修仙的信息密度需求：

```
┌──────────────────────────────────────┐
│          TopBar（时间/境界）           │
├──────────┬───────────────────────────┤
│          │                           │
│ SideRail │     主消息区              │
│（系统入口）│   （消息流 + 状态卡）     │
│          │                           │
├──────────┴───────────────────────────┤
│  ActionRow（当前地点可用行动）         │
├──────────────────────────────────────┤
│  LocationRow（地点切换 + 移动）        │
└──────────────────────────────────────┘
```

## §2 面板体系（16 个 Dialog 弹窗）

所有面板为 Dialog 弹窗，统一模式：`props: { open }` + `emit: 'update:open'`，打开时懒加载数据。

### 核心面板

| 面板 | 内容 | 触发 |
|------|------|------|
| DantianPanel | 丹田/经脉/灵力状态 | SideRail 丹田按钮 |
| KnowledgePanel | 知识百科条目 | SideRail 知识按钮 |
| FactionPanel | 7 组织好感度 | SideRail 势力按钮 |
| WorldPanel | 月报/世界状态 | SideRail 世界按钮 |
| QuestPanel | 任务列表 | SideRail 任务按钮 |

### 工具面板

| 面板 | 内容 |
|------|------|
| AlchemyWorkbench | 炼丹界面（三区域 UI） |
| ArtifactPanel | 法器/装备 |
| MapView | 节点-边地图 |
| TimelinePanel | 时间轴 |
| MonthReportPanel | 月度推演报告 |

### 新增面板

| 面板 | 对应系统 |
|------|---------|
| GongfaPanel | 功法组件化 |
| AcademicPanel | 学术系统 |
| MingcongPanel | 命丛 |
| FinancePanel | 经济/借贷 |
| CompetitionPanel | 竞技比赛 |
| AchievementPanel | 成就（12 章分类） |
| NpcPanel | NPC 交互/社交 |

还有：SettingsPanel、ExperimentLogPanel、RecipeBookPanel、HelpPanel。

## §3 深色主题

- 全局深色主题（符合「赛博修仙」定位）
- 主色调：暗紫 + 青蓝 + 金色点缀
- 正文字号极小化（text-xs），适配高信息密度
- UI 组件库：shadcn-vue（Badge / Button / Card / Dialog / ScrollArea / Separator / Tooltip）
- CSS：Tailwind CSS

## §4 信息层级设计

### 消息语气分层

| 类型 | 语气 | 示例 |
|------|------|------|
| 数据卡 | AI 道侣语气 | 「LEVSS Monitor 提示：InputRate 提升 12.3%」 |
| 功法信息 | 按来源分 | 学院功法用学术语气，古法用传统语气 |
| 经济信息 | 冷数据 | 「余额 → 340 SQU」 |
| 叙事事件 | 正常文风 | 「你在灵泉洞深处发现了一株异常灵植...」 |
| 系统提示 | 简洁 | 「⚠️ 时间不足」 |

### FormulaTrace 展示

修炼/炼丹/走火后，公式追溯以可折叠面板呈现：

```
▸ 公式追溯
  InputRate = L_fit × S × E × qi_purity = 0.72 × ...
  Risk = O(t) + P(t) + M(t) + Ψ(t) = 0.31 × ...
  白话翻译：你的灵气适配度不错，但污染积累需要注意
```

## §5 渐进式 UI 解锁

UI 不一次全部展示，随教学进度解锁：

| 教学阶段 | 解锁 UI |
|---------|---------|
| 初始 | 消息区 + 基础行动（休息/研读） |
| 六轮教学完成 | 全部行动 + DantianPanel + KnowledgePanel |
| 练气突破 | FinancePanel + GongfaPanel + MapView |
| 首次炼丹 | AlchemyWorkbench + RecipeBookPanel |
| 首次月推演 | WorldPanel + MonthReportPanel + FactionPanel |
