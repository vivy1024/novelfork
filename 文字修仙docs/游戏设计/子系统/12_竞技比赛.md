## §1 设计哲学

- **参考《没钱修什么仙》**：法赛/武道竞赛/十大联赛是张羽逆袭的关键转折点
- **赛事 = 社交 + 经济 + 声望的综合体**：不只是打架，是展示、赌博、社交的综合场
- **AI 模拟**：玩家和 NPC 共同参赛，战力模型决定胜负，不是回合制战斗

## §3 战力模型

比赛不用回合制，而是 sigmoid 胜率模型：

```python
combat_power = f(
    realm_index,         # 境界基础
    cultivation_progress, # 修炼进度
    hp_state,            # 身体状态
    mind_score,          # 心性得分
    equipment_bonus,     # 装备加成
    talent_bonus,        # 天赋加成
)

win_probability = sigmoid(
    (my_power - opponent_power) / scale_factor
)
```

**设计原则**：高一个境界 ≈ 碾压（胜率 > 95%），同境界看综合参数。

## §4 赛制

- **单淘汰制**：首轮随机分组，逐轮淘汰
- **切磋模式**：学院月赛为切磋模式（伤害 ×0.5），不会致死
- **正式模式**：季赛和邀请赛为正式模式（有受伤风险）
- **按境界分组**：避免碾压，同境界玩家和 NPC 一起比

## §5 观战与下注

### 观战

- 消耗 1 小时，可观看当前进行的赛事
- 获得信息：参赛者战力估算、赛况实时播报
- **社交效果**：观战时可与同场观众社交

### 下注

```
bet(competition_id, target_npc_id, amount)
# 赔率由系统根据参赛者战力评估自动计算
# 冷门选手赔率高，热门选手赔率低
# 结算：赢了获得 amount × odds，输了扣 amount
```

## §7 世界集成

- **月度推演**：30 tick 自动检查是否该触发月赛/季赛
- **NPC 自动报名**：同境界 NPC 按概率参赛，不需要全是玩家
- **赛后社交**：比赛结束后，与对手的关系值变化（赢了 +3，输了 +1，被碾压 -1）
