## 二、左区：配方列表

### 2.1 数据来源

- API：`GET /artifacts/meta` 返回 `recipes`（炼器）
- API：`GET /state/recipes` 返回炼丹配方
- 配方按"已解锁/未解锁"分组

### 2.2 展示

```
配方名称        品质期望    难度
─────────────────────────────────
[✓] 静心散      B-C级      ★★☆
[✓] 清肺丹      C级        ★☆☆
[?] 聚灵丸      A级        ★★★ ← 需练气才能解锁
[—] 筑基丹      S级              ← 未解锁（灰显）
```

点击配方后，**投料区自动显示该配方所需材料槽**（主材 + 辅材 slots）。

## 三、中区：投料 + 工序

### 3.1 材料槽

每个槽显示：
- 材料名 + 五行频谱（条形图）
- 实际 P_actual vs 配方要求的 P_target（绿/黄/红）
- 可替换：点击槽 → 背包弹出，根据五行频谱过滤推荐

```
主材：酸枣仁    ████░░ 0.45g
              [实际 P=0.58] [目标 P≥0.55] ✓

辅材：茯苓多糖  ███░░░ 0.35g
              [实际 P=0.52] [目标 P≥0.55] ⚠️ 略低，影响 UsefulPeak

辅材：[添加第三种材料]  +
```

### 3.2 工艺参数（炼制步骤可见）

```
工序一：水提酸枣仁
  温度窗口：60-80°C
  当前设定：[  72  ]°C  ✓ 在窗口内
  说明：偏高 2°C → 提取效率 +3% 但杂质 +1%

工序二：研磨茯苓
  时长：20分钟（固定，无调参空间）

工序三：混合炼制
  时长：[ 40 ]分钟（可调 30-60）
  说明：延长时间 → ProcessVariance 降低，品质更稳定
```

## 五、底部操作区

| 按钮 | 说明 |
|------|------|
| **预览模式** | 不消耗材料，只看预算结果（ALC02 已实现 `/action/alchemy/preview`） |
| **展开 FormulaTrace** | 展开完整公式推导步骤（Layer 1/2） |
| **开始炼制** | 消耗材料，触发后端 `/action/alchemy`，进入中断式工序流程 |
