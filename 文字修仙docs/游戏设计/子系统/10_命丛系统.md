## §1 物理基础

> 命丛 = 生物体在 LEVSS 场中演化出的特化感应/操控器官。
> 类比：眼睛感应可见光，命丛感应特定自旋态粒子场。

关键特性：
- **获取即可用**：没有等级、没有熟练度、没有冷却——长出来就能用
- **无承载上限**：理论上可以拥有无限多命丛（但获取概率极低）
- **无排斥反应**：不同命丛之间不冲突
- **道统影响效率**：唯一的工程化要素——道统匹配度影响命丛发挥效率

## §2 命丛体系结构

```
16 大功能域 × 8 作用模式 × 8 变体 = 1024 种普通命丛
                                    + 16 种神命丛（每域 1 个）
```

## §3 获取途径

| 来源 | 概率 | 条件 |
|------|------|------|
| 高灵区探索 | 10% | 灵压 ≥ 0.5 的地点 |
| 普通探索 | 5% | 任何地点 |
| 交易购买 | — | NPC/市场有售（稀有） |
| 突破奖励 | 30% | 境界突破时随机获得 |

### 神命丛

- 每功能域 1 个，一局游戏中位置随机
- 只在高灵区出现
- 只有玩家能获得（NPC 无法觉醒神命丛）
- 效果远超普通命丛
- **不做唯物解释**——世界观允许的唯一超自然要素

## §4 命图系统

命图 = 特定命丛组合的集齐解锁效果。

### 经典命图（9 种）

| 命图 | 所需命丛 | 解锁效果 |
|------|---------|---------|
| 4 种无上命图 | 各需 4 个特定神命丛 | 全域效率 ×2 |
| 5 种入门命图 | 各需 3 个特定普通命丛 | 单域效率 ×1.5 |

**激活规则**：集齐所需命丛后自动激活，无需操作。

### EA 阶段规划

自由组合探索——玩家可以尝试未记录的命丛组合，可能发现隐藏命图。

## §5 效率计算

```python
mingcong_efficiency = base_effect × daotong_modifier

daotong_modifier = calc_mingcong_daotong_match(
    mingcong_daotong,    # 命丛的道统属性
    player_daotong,      # 玩家当前道基的道统
)
# → modifier ∈ [0.5, 1.5]
```

**设计意图**：道统是唯一影响命丛效率的工程因素。玩家可以通过调整功法组合来间接优化命丛效率。

## §6 玩家体验设计

- **早期**：探索时偶然获得，感受「惊喜」
- **中期**：有意识地在高灵区刷探索，追求特定功能域
- **后期**：集齐命图，体验能力质变
- **全程**：命丛面板展示已有命丛 + 命图进度条
