# 05_地图与探索

> **一句话版**：地图不是开放世界，而是「节点-边」网络——每个地点有独特灵压和资源，移动是有成本的决策，探索是发现世界的主要途径。
> **关联文档**：[[06_经济与交易]] · [[04_势力声望与任务]] · [[10_命丛系统]]
> **代码对照**：`map_service.py` · `data/locations.yaml` · `action_service.py → travel / gather / explore`

## §1 地图结构

### 节点-边模型

- **节点** = 地点（有灵压、资源、势力归属、五行频谱）
- **边** = 路径（有移动耗时，单位：分钟）

玩家不能自由移动，只能沿边移动到相邻节点。

### 地点属性

| 属性 | 说明 | 游戏影响 |
|------|------|---------|
| spirit_pressure | 灵压（0-1） | 修炼效率、风险系数 |
| wuxing_spectrum | 五行频谱 | 炼丹/修炼的环境加成 |
| actions | 可用行动列表 | 该地点能做什么 |
| icon | 地点图标 | 前端显示 |
| is_start | 是否起始点 | 新游戏出生地 |

## §3 解锁机制

### 三种解锁类型

| 类型 | 示例 | 设计目的 |
|------|------|---------|
| 境界门槛 | 练气后进入灵泉洞 | 自然进度门控 |
| 势力好感 | 玄清好感 ≥ 15 进入古法静室 | 社会关系门控 |
| 探索次数 | 城郊探索 5 次发现废弃矿洞 | 投入时间门控 |

### 渐进式开放

```
胎息期：初始 6 个 + 探索可解锁 5 个 = 最多 11 个
练气期：+6 个高阶地点 = 最多 17 个
筑基期：+更多高灵区（EA 扩展）
```

## §4 探索系统

### 探索行动

消耗 2-4 小时，在当前地点搜索发现：

| 发现类型 | 概率 | 内容 |
|---------|------|------|
| 资源 | 40% | 灵材/矿石（品质受灵觉影响） |
| 线索 | 25% | 知识条目/配方碎片 |
| NPC 相遇 | 15% | 游方修士/前辈/商人 |
| 命丛觉醒 | 5-10% | 触发命丛获取（高灵区 10%） |
| 奇遇 | 5% | 稀有事件/隐藏地点入口 |
| 危险 | 5% | 遭遇战/环境伤害 |

### 采集行动

- 产出由地点资源列表 + 随机品质决定
- 品质受 L_fit 和灵觉影响
- 高灵压地点产出更好但 Risk 更高

## §5 移动系统

```
POST /action/do { action: "travel", target_id: "academy_lab" }
# 消耗 travel_time 分钟（20-60分钟）
# 只能移动到相邻节点
```

移动时间由边权决定，在 `locations.yaml → adjacency` 中定义。
