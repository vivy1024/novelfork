## §1 设计哲学

功法组件化是「修仙 = 工程」的核心体现：

- **可拆解**：一套功法 = 吐纳法 + 采气诀 + 天地灵气 + 秘法修改器
- **可量化**：品级由道统匹配度、干扰结、组件质量三者计算得出
- **可优化**：玩家通过实验发现最优组合，而非抽卡碰运气
- **不存在「最强功法」**：只有「最适合你当前材料和道统的功法」

## §2 四组件模型

| 组件 | 作用 | 数据源 | 可替换 |
|------|------|--------|--------|
| **吐纳法** | 控制灵气与体内循环的呼吸节律 | `breathing_methods.yaml`（6种） | ✅ |
| **采气诀** | 决定从环境中摄取灵气的方式 | `gathering_methods.yaml`（5种） | ✅ |
| **天地灵气** | 修炼时使用的灵气类型 | `qi_types.yaml`（10种） | ✅ |
| **秘法修改器** | 附加效果（增效/增险/偏移道统） | `secret_modifiers.yaml`（4种） | ✅ |

### 组件来源

| 来源 | 说明 | 示例 |
|------|------|------|
| 公开教学 | 基础组件，游戏开始即可获取 | 基础吐纳法、自然感应 |
| 势力传承 | 需达到好感度阈值 | 古法静吐、学院标准采气 |
| 交易购买 | NPC/市场出售的组件片段 | 稀有秘法修改器 |
| NPC 传授 | 需达到特定关系等级 | 至交 NPC 分享秘法 |

### 道统匹配度

```
calc_daotong_match(player_daotong, material_daotong, qi_daotong) → match_score

match_score ∈ [0, 100]
  100 = 三方完美共振
  50-99 = 部分匹配，有效率损失
  < 50 = 失谐，可能产生干扰结
```

## §4 品级九品制

功法品级 = 组件匹配质量的综合评分。

| 品级 | 分值区间 | 含义 |
|------|----------|------|
| 一品 | 90-100 | 天人合一，极稀有 |
| 二品 | 80-89 | 大师水准 |
| 三品 | 70-79 | 优秀 |
| 四品 | 60-69 | 良好 |
| 五品 | 50-59 | 普通 |
| 六品 | 40-49 | 勉强可用 |
| 七品 | 30-39 | 粗糙 |
| 八品 | 20-29 | 低劣 |
| 九品 | 0-19 | 几乎无效 |

### 品级计算

```python
grade = calc_gongfa_grade(

    interference_knot,    # 干扰结强度 (权重 30%, 负面)
    component_quality,    # 组件质量均值 (权重 30%)
)
```

## §5 干扰结

当组件之间的道统不兼容时，产生干扰结（interference knot）。

```
interference = calc_interference_knot(
    breathing_daotong, gathering_daotong, qi_daotong
)

interference ∈ [-1, 1]
  < 0 = 相生增效（组件共振）
  = 0 = 中性
  > 0 = 相克干扰（效率下降 + Risk 上升）
```

**玩家体验**：干扰结是最关键的调参目标。就像调音响的阻抗匹配——最好的扬声器配错了功放，还不如普通组合。

## §6 道基

道基 = 长期修炼积累后形成的个人化特质。由功法组合×灵气历史×品级×修炼数据综合生成。

```python
daoji = generate_daoji(
    gongfa_history,   # 使用过的功法记录
    qi_history,       # 使用过的灵气类型
    grade_history,    # 历史品级
    cultivation_data, # 总修炼时长/成功率/走火次数
)
# → 返回独特的道基名称 + 属性加成
```

**设计原则**：道基不可选择，只能通过修炼行为「养」出来。玩家的选择决定结果，但无法直接控制。

## §7 秘法修改器

四种附加效果，高风险高回报：

| 秘法 | 效果 | 代价 |
|------|------|------|
| 风险增量型 | 修炼效率 +20% | Risk 系数 +0.15 |
| 道统偏移型 | 强制更换道统 | 干扰结可能暴涨 |
| 效率损失型 | 降低某组件效率 | 专注其他组件 |
| 干扰增量型 | 增加干扰结 | 可以反向利用 |

## §8 玩家认知路径

```
胎息期：用默认功法 → 看公式 → 理解组件存在
练气期：尝试替换组件 → 对比品级变化 → 理解道统匹配
筑基期：自选最优组合 → 研究干扰结 → 培养道基
```
