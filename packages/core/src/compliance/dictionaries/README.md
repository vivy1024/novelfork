# 合规词库来源说明

本目录词库是 **种子词库**，不是任何平台的官方完整敏感词库。

## 来源原则

- `common.json`：基于项目已有 `agents/sensitive-words.ts` 的内置规则迁移，并按平台合规常见类别结构化。
- 平台额外词库：只放公开资料中能确认的平台规则类别、AI 使用监管信号、导流/广告等通用审核风险的少量种子词。
- 不声称覆盖平台完整审核规则。
- 产品必须支持外部导入词库，用于接入用户自有词库、第三方开源词库或团队维护词库。
