# 第11章 门里那声咳嗽

周谨川回到家时，楼道里一股潮湿的米浆味还没散。隔壁张婶家的收音机沙沙响着，里面播着防非典后的复工新闻，声音忽远忽近。

许淑芬把纱门掀开一道缝，看见是他，先往他手上看。

“收条呢？”

周谨川把那张折了三道的纸递过去。纸面被汗水浸过，边角发软，朱广顺那几个字歪歪扭扭，红指印糊了一圈，像随手按上去的血痂。

周建国接过去，低头看了半天，喉结滚了一下。

“这两千，买了七天话头。”

“买了个嘴上的话头。”许淑芬把收条压到搪瓷盘底下，语气很平，“真要翻脸，这张纸也就是个留底。真要认，它就值七天；真不认，连七天都未必值。”

她话说得冷静，手却还是在盘沿上多按了一下，像怕那张轻飘飘的纸被风吹跑。

屋里没开电扇，南窗半掩着，外头的热气卷着灰尘往里灌。墙上那本薄得发黄的挂历停在六月，最下面一行小字印着“2004年”。周谨川盯了一眼，目光又往前挪，落到上个月被撕掉的那一角残痕上，心口像给什么东西轻轻刮了一下。

不是后天。

是今天。

差的不是一句早晚，是真真切切的两天。

这两天，足够让一个人拿错主意，也足够让一整个家措手不及。

周建国把收条翻过来又看了看，像想从纸背后再看出点什么，半晌才抬头：“茶摊那边，朱广顺就这么写了？邹国林也在？”

“在。”周谨川把门关严，走到桌边坐下，嗓子里还带着白天来回奔的干涩，“他没伸手写，但人坐那儿没反对。褚老三的名字也是他们自己露的。还有老王，白面包那边跟的还是一拨人。今天这两千，不是把事情平了，是让他们觉得咱家没躲、没跑、还能掏得出一点，先把最急的一脚踹慢。”

许淑芬看着他：“踹慢，不是停了。”

“我知道。”

他说得很快，几乎没留缝。

许淑芬听见这三个字，反而没再接。她转身把灶边的凉白开放到桌上，杯子磕在木桌面，发出一声脆响。

周谨川把半杯水一口喝下去，喉头的热才往下压了压。可那股从下午就攒着的烦闷并没散，反而因为到家、因为看见这张收条、因为看见父母都盯着自己，越发清楚起来。

他记错了。

不是记错人，不是记错地方，是记错了最要命的那个点。

前一世那些事在他脑子里像一条断断续续的旧磁带，能听见大概旋律，却总有噪声和空白。他原本记得很死，债主堵门是在后天，最急的一拨也是后天中午前后才会逼到门口，结果今天一早，门已经被拍响了，连要钱的话头都砸到了脸上。若不是早上顶住那一阵，若不是中午在茶摊把话头生生抠下来，今天这一关就不是“花两千买七天”，而是被人当场掀开锅底，看你家到底还剩多少。

周建国伸手摸了摸裤兜，像是下意识要摸烟，摸了个空，手又垂下来。

“你上午说的时候，我还想着，是不是他们提前来试探。”他声音发闷，“可看今天这架势，不是试探。他们就是照着今天来的。”

屋里一下静了。

外头楼道里有人提着水桶上楼，铁桶边碰着栏杆，铛、铛两声。张婶家收音机换了节目，主持人拖着江州市本地口音读一串商场促销，隔着墙，字音碎成一片。

许淑芬先开口：“你那天说后天，我信了。”

她没责怪，语气甚至近乎平常。可就是这种平常，让周谨川肩背更紧了。

“我也信了。”周建国把视线落在桌角，“所以今天早上第一下砸门，我脑子是空的。”

周谨川没有躲，直直迎着两人的目光：“是我记错了。”

这句话说出来，比下午在茶摊前咬着牙把两千块推出去还难。他原以为自己重来一回，哪怕钱没有、势没有、关系也没有，至少还能靠那点提前知道的事抢个半步。可真正撞上来才知道，所谓提前知道，也会有误差，也会掉链子，也会在最关键的地方少两天。

两天。

他在心里又念了一遍。

就像用钝刀子把那点侥幸硬刮掉。

“不是全错。”他把手掌压在桌边，慢慢说，“人、线、地方，大差不差。马会来、老罗、老贺、褚老三，蓝门第三间，白面包，红色摩托，这些都没偏。偏的是日子。不是事情没来，是比我记的早了两天。”

许淑芬眼睛一动不动地看着他：“那以后呢？你怎么拿主意？”

这话比“你怎么记错了”更重。

周谨川默了两秒，才道：“不能再盯死某一天。以后不盯日子，盯人，盯动静，盯谁先露头。人到哪儿，事就到哪儿。只要他们还在走那条线，日子早两天晚两天，咱都得按最早的来防。”

许淑芬这才轻轻点了下头。

“这句像话。”

周建国把收条重新摊平，用手掌抹了两下，像在抹平纸上的皱褶，也像在给自己顺气：“那这七天，怎么过？”

周谨川把今天从江北到茶摊前后的情形，一段一段讲给他们听。钱大河鱼摊上那句证词，蓝门第三间门里那一下闷响，朱广顺看钱时眼神怎么飘，邹国林怎么接话、怎么拿“旧账”“跑腿”这些字眼压人，褚老三和白面包又怎么串到了一处。他讲得比白天更细，把自己能记住的脸、衣服、方位、时间都尽量讲清。

他说到蓝门第三间的时候，屋里又安静了一阵。

周建国皱着眉：“你确定门里有人？”

“不能说死，只能说八成。”周谨川道，“门缝太窄，里头没看见人脸，只听见闷响。旁边住的女人说最近常有人出入，那地方不像空着。还有断麻绳。”

许淑芬抿了抿嘴：“这事别乱碰。”

“我知道。”周谨川看向她，“没证，没把握，不能硬闯，也不能只凭一句猜。先把看见的、听见的记下来。”

他说着站起来，从柜子最上层取下那本已经起毛边的旧作业本。作业本前面几页还留着他中学时的钢笔字，后面空了不少。他翻到干净的一页，把今天的日期写上去，又按顺序写：

清晨，朱广顺上门；  
中午前，茶摊见面，邹国林在场；  
两千元，七天；  
江北钱大河证词；  
废粮站后排蓝门第三间；  
白面包、褚老三、“十点”“老王那边”。

字写得不快，一笔一画都压得很实。写到“蓝门第三间”时，周谨川笔尖顿了顿，墨水在纸上洇出个小点。

许淑芬站到他身侧，看了一会儿：“别只写名字。谁说的、什么时候说的、边上有什么人，都写上。”

“嗯。”

“还有今天谁看见你回来，谁知道你去过哪儿，也写。”

“好。”

她说这些的时候，声音还是平平的，不高不低，像在算家里米面油盐。可正是这种细，才让周谨川心里那股乱劲一点点往下沉。他忽然想起前些日子，她把上门试探的人、楼下停过的车、哪天谁从窗下往上看，都记在废药盒背面。许多事真到了临头，撑住这个家的，从来不是什么漂亮话。

周建国挪了挪椅子：“那两千从哪儿补？”

“先算眼前。”许淑芬回头看他，“林姐那四千八是急账，不能装没这回事。家里剩下多少，我知道。你口袋里还有多少，我也知道。先把活命的钱、买菜的钱、来回跑的路费留出来。剩下的，哪笔最先追到门口，就先应哪笔。”

她说完去橱柜里把钱一份一份拿出来。

碎票、整票、扎过皮筋的、用报纸角包着的，都摆到桌上。旧屋昏黄，钱色也旧，数起来却更让人心里发紧。周建国把兜里几张皱票一并掏出来，放在边角。周谨川也把剩下的零钱摸出来，和父母的一起堆在桌面。

一家三口谁都没说“太少了”这句废话，只把钱摊开，按用途分。

电费水费，先压后；  
日常菜钱，留最少；  
路费，必须留；  
上门应急的钱，单独包一份；  
林姐那笔，另记一页，不能混。

许淑芬从针线篓里翻出一截红线，把几小叠钱捆住，捆的时候手稳得很。
