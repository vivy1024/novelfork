# 第12章 静默名单

院区地面上阳光很足。

几辆深色军车停在台阶下，车身反着白光，远处树上的蝉声断断续续，和院墙外模糊的车流声混在一起。城还在照常运转，天也蓝，谁从这里路过，都不会想到地下刚摊开过一份名单。

汪淼站在台阶边，手机还亮着。

那条陌生号码发来的短信很短。申玉菲要见他。

史强看了一眼：“手脚够快。你这边刚出门，那边就闻着味儿了。”

汪淼按灭手机：“也可能说明，他们早知道我会来。”

史强眯眼：“老常说先按你的意思去找丁仪。可我看你这脸色，像还想回去把材料再看一遍。”

“我要再看一遍。”

几分钟后，两人重新穿过安检，沿着温度偏低的走廊往里走。会议室门开着，常伟思还没走，正同那名戴眼镜的中年男人低声说话。桌上散着文件夹，投影幕布还没收。

常伟思抬头看见汪淼：“回来得正好。有些东西你该看完整。”

汪淼坐下。常伟思把一只深蓝色文件夹推到他面前：“更新后的汇总。过去六十三天，全球前沿基础科学领域，确认或高度怀疑的非正常死亡四十六起，其中自杀二十七起。剩下的有坠楼、车祸、溺亡、药物过量，还有几起没有明确结论。”

汪淼翻开第一页。照片下面跟着机构、研究方向、死亡时间。苏黎世、芝加哥、莫斯科、东京、慕尼黑、北京、日内瓦。

“你们怎么确认这些人彼此有关联？”

戴眼镜的中年男人说：“先从学术活动轨迹查起。会议、通信、联合课题、基金评审、私人沙龙、远程讨论。设备和实验室没有稳定共性，接触的人和材料却出现交叉。交叉点最多的，就是这个组织。”

另一份材料摊开，封面只有四个字：科学边界。

“它不是正式注册机构，没有统一办公地点，也没有公开会章。”中年男人说，“更像一个跨国邀请制社群。门槛高，成员流动性强，活动形式散。可以是学术晚宴、封闭讨论会、基金资助，或者线上研讨群。”

史强道：“你以为是俱乐部，转头它又像基金会，再一转眼，成了几个人的小圈子。”

“有核心人物吗？”汪淼问。

“有活动密度更高的人。”常伟思说，“申玉菲算一个。还有几个身份在核验中。问题在于，这个组织和死亡之间有关联，不等于它就是全部原因。”

他点了点统计表：“四十六人里，十三人与‘科学边界’有紧密联系，九人处于外围接触状态，另一些只在某个时间段收到过相似材料，或短暂参加过讨论。它是传播链上的重要节点，但后面还有什么，还没摸到头。”

投影亮起，画面上出现几段遗书摘录。英文、俄文、德文、日文，最后一列是中文整理稿。

“实验结果没有意义。”
“世界在欺骗我们。”
“继续做下去只是重复错误。”
“物理学从根上就是空的。”

汪淼盯着最后一句，喉咙发紧。

杨冬留下那句话时，他只觉得荒诞。可现在，这句话被摆进一排不同国家、不同年龄、不同研究方向的遗书里，不再只属于杨冬。

“你们认为，他们为什么会得出同一种结论？”

“还不能说完全同一种。”中年男人扶了扶眼镜，“表述相似，触发过程未必一致。有人在死前两周停止实验，有人疯狂重复验证，有人把多年数据全部烧掉，还有人去找神学人士谈话。可他们都经历了一件事：认知基础被撬开了。”

史强低声道：“说人话，就是脑子里的地基让人刨了。”

汪淼继续翻资料，看到一张时间线。几名死者在死亡前最后二十天，轨迹里都出现过封闭讨论、私人会面、匿名邮件、访问邀请之类的标记。没有一步对所有人都完全重合，但“接触某种内容”越来越清晰。

“内容是什么？”汪淼问。

“部分是文字，部分是演示材料，还有些可能是面对面讨论。”常伟思说，“完整样本不多。很多接触者在崩溃前把东西清掉了。硬盘格式化，笔记焚毁，邮箱删除，像是不愿让别人再看到。”

汪淼手指停在一份口供摘要上。那名北美青年数学物理学家的同事说，死者曾反复说，自己过去二十年的工作像在给一个不存在的目标写墓志铭。

汪淼想起杨冬。这样的人，如果有一天怀疑整个结构是假的，会发生什么，几乎不用想。

“还有一组材料，你必须看。”常伟思说。

屏幕上换成一张关系图。申玉菲的照片在中央，周围分出几条线，连向不同名字。杨冬的名字在右侧，线是红色。

“四次得到证实。”中年男人说，“地点两次在北京，一次线上，一次在城郊一处封闭讨论会。时间主要集中在杨冬死亡前一个月。”

“谈了什么？”

“没有录音。”常伟思说，“只拿到外围证词。杨冬在那之后取消了一场原定报告，退出了一个小型讨论组，研究状态明显下滑。最后几天，她和外界联系很少。”

史强道：“这个申玉菲，不满街拉人，就盯着有资格的人递话。”

汪淼看着屏幕上的照片。照片里的女人神情平静，穿着普通，像大学里随处可见的讲师或学者家属，可她站在线条中心。

“你们对她了解多少？”

“职业经历并不突出，学术履历有断层，经济来源复杂但干净，社会关系克制。”常伟思说，“真正有价值的，是她在几个小圈层里的口碑。她擅长组织封闭交流，让互不相识的人坐到一张桌上，再把一场普通讨论慢慢引到某个方向。”

“哪个方向？”

“怀疑。先怀疑某个公式，再怀疑某个实验，再怀疑某种理论方法，最后怀疑支撑这些东西的根。”

“这不算罪名。”汪淼说。

“当然。科学本来就靠怀疑。”常伟思点头，“但如果有人有办法把这种怀疑推进到自毁，那就是另一回事了。”

“有没有活下来的接触者？”

中年男人抽出一页名单：“有。九人出现严重崩溃、失踪或长期停研，暂未死亡。我们只接触到其中三个。一个拒绝交流，只肯重复一句‘别再算了’；一个接受治疗后把全部研究转去做中学科普；还有一个，昨天失联。”

常伟思说：“邮件、会面、讨论、资料传播、心理崩解，没有哪个环节超出人类行为范围。问题在于，普通劝诱做不到这么稳定的结果。”

汪淼合上文件夹，掌心压在封面上。

杨冬自杀，遗言指向物理学本身。全球多名基础科学家死亡，遗书呈现同构语言。科学边界是高重合区域，申玉菲处于中心。接触链条指向某种材料或讨论，它能让不同国家、不同学科的人在短时间内出现相似的认知崩塌。

“你们为什么找我？”他抬头，“因为我和杨冬认识，因为我是搞应用研究，离理论物理有一点距离，还是因为申玉菲已经盯上我？”

“都有。”常伟思说，“你在学界有分量，方向又不完全重合，对他们来说值得邀请；对我们来说，你更容易分辨哪些只是故弄玄虚，哪些真的会动到根子。还有一点，你目前还没被打断。”

汪淼听明白了。名单上的许多人，出事前也许都和自己一样，以为不过是参加一次讨论，读几份材料，跟同行争几个晚上。

“如果我进去，”他说，“你们能保证我家人的安全？”

“你家人不接触这件事。”常伟思说，“外围会做布置，但不会把她们拖进来。你和申玉菲接触，只限于你本人。这个底线，我们守。”

史强插话：“老汪，我这边也盯着。”

汪淼重新点开那条短信。时间写得很清楚：下午三点，地点在城西一处会所，落款没有名字，只有一句“申女士期待与您讨论一些真正重要的问题”。

“在去见她之前，我还是要先见丁仪。”汪淼说。

常伟思点头：“车已经安排了。你和他谈完，直接去赴约，中间不再回这里。她既然主动发来短信，说明你已经进入她的视线。你接下来每一步，都要像你自己会走的路。”

汪淼又问：“你刚才说，这是对基础科学的系统打击。依据除了死人名单，还有什么？”

中年男人抽出一份简报。第一页列的是几家国际实验机构近期异常：项目延误、数据争议、关键人员离岗、审稿突撤。第二页是几个国家秘密共享来的内部评估，措辞谨慎，却都指向同一个判断：基础理论研究正在出现非正常停滞，幅度还在扩大。

“这还没到社会面能感觉出来的程度。”常伟思说，“城市照常转，工厂照常开，股市照常起伏。可如果最底层那批人集体停下来，或者被引到错误方向，后果会在几年后一起结账。”

汪淼终于明白了，这间地下会议室里的“战争”为什么会被说得那样平静。没有爆炸，没有前线，没有公开敌军。伤口在实验室，在邮件附件里，在几场看上去体面的讨论会上。故事发生在当代中国，却已迅速延展到全球层面，而地面上的世界依旧平静正常。

门外有人敲了两下，年轻军官探头进来：“车备好了。”

常伟思站起身，把文件夹收拢：“汪教授，从现在开始，你看到的每一份材料，听到的每一句话，都先别急着信，也别急着反驳。你只做一件事，记住它们。”

“然后呢？”

“看它们会不会在你脑子里互相对上。”

汪淼也站了起来。

走出会议室时，走廊里的冷气迎面扑来。史强跟在旁边，小声问：“现在什么感觉？”

汪淼想了想：“像有人告诉你，脚下这层地板是空的，但你还得继续往前走。”

“那就别低头。”史强说。

两人走到电梯口，金属门上映出模糊的人影。汪淼看着自己发白的脸，忽然意识到，名单上的那些人，也许都曾在某个清晨这样照过镜子，穿好衣服出门，以为自己只是去参加一次普通会面。

电梯门开了。汪淼走进去，手机在掌心轻轻震了一下。

又是一条新短信。

还是那个号码。

内容只有一行：

“丁先生有些话说不清，您不妨直接来问我。”

汪淼盯着屏幕，眼神一点点沉下去。

申玉菲知道他下一步要去见谁。
