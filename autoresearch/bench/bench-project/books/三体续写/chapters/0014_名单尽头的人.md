# 第14章 名单尽头的人

电梯门在地下二层缓缓合拢，像一张金属嘴唇，把走廊上最后一点自然光咬断。

汪淼站在里面，手里捏着那张折过两次的名单。上面的名字已经压进脑子里：不同国家，不同实验室，不同语言，最后却落到同一种结局。

史强靠着轿厢壁看他：“还撑得住吧，汪教授？”

“我没事。”

声音有点空。地下会议室里的空气太干，开水味、纸张味和设备散出的热味混在一起，像消毒过度后的病房。

电梯停在负三层。门开后，冷气更低，走廊灯管亮得发白。尽头厚门半开，里面有人说英语，还有人低声报数字。

史强抬手示意他先进去。

汪淼跨过门槛，先看到墙上的世界地图。整面墙的电子投影上，北美、欧洲、东亚亮得最密，红点、黄点、白线交错铺开。城市本身却平静如常，没有起火，没有警报，摄像画面里行人照样过马路、上班、买早餐。秩序稳定，甚至显得有些嘲讽。

常伟思站在长桌尽头，军装一丝不乱。桌上散着不同语种的复印件，俄文、英文、法文、日文。几名军官和便装分析员分坐两侧，最靠近投影幕的是一名戴眼镜的中年男人。

“汪教授，坐。”

汪淼坐下。有人把一杯热水推到他手边，他没喝。常伟思直接点开第一张照片。

“匈牙利，布达佩斯。理论物理学家，四十九岁，上月十五日，自缢。现场没有搏斗痕迹。”

第二张是浴缸，旁边镜框里是一家三口的合照。

“加拿大，多伦多，粒子天体物理方向，三十六岁，割腕。死前两周退出合作项目，理由只有一句：继续下去没有意义。”

接着是第三张、第四张、第五张。照片都很克制，像普通档案，可正因为普通，拼到一起才更可怕。每一张单独看都像小概率意外，放到同一张桌子上，规律就露出来了。

戴眼镜的分析员把一页统计表推到汪淼面前。

“过去两个月，确认和高度怀疑的相关非正常死亡共四十六起，其中二十七起归为自杀，九起看似意外，剩余十起还在核验。分布在十四个国家。共同特征有三类：第一，研究方向高度集中，几乎都在基础理论前沿；第二，死前行为模式收缩，停会、停课、停实验、切断社交；第三，接触链有重合区域。”

“什么重合区域？”汪淼问。

“‘科学边界’。”常伟思说，“或者与它相近的闭门讨论圈。”

汪淼盯着那行字，想起杨冬，想起她那句遗书。

“你们确定它是一个组织，不是松散圈子？”

“这正是麻烦的地方。”中年分析员说，“它有名字，有邀请机制，有固定话题，有内部材料，也有不同层级。可真去找注册信息、资金来源、办公地点，什么都抓不牢。像一团雾，却能留下脚印。”

史强哼了一声：“雾能把人弄死，这雾就挺有本事。”

常伟思翻开另一份文件。那是几封遗书的对照表，原文和译文并排放着。字句长短不同，语气也不完全一样，但几个意思反复出现：世界是假的，规律在退场，实验不再可信，继续研究毫无意义。

“心理暗示？群体感染？某种设计过的话术模型？”汪淼问。

“先查的就是这个。”常伟思说，“请了精神科、社会学、舆情分析和语言学的人。有人确实原本就有压力，但还有一批人，没有抑郁病史，没有经济困境，甚至刚拿到突破性结果。”

他指向一个名字。

“瑞士，四十一岁。死前四十八小时刚完成一组数据复核，结果和理论预期高度吻合，当晚还给学生发邮件安排明年课题。第二天凌晨跳楼。”

“邮件里没异常？”

“没有，语法完整，措辞平稳，连附件命名习惯都和平时一致。”

常伟思继续说：“问题还在扩大。不是所有人都死了，还有些人活着，但已经不能工作。有的拒绝再看实验结果，有的一进实验室就吐，有的把自己关在屋里，一遍遍写同一句话。”

“哪一句？”

分析员低头翻档案：“‘尺子弯了。’”

汪淼沉默了。这话像疯话，可他听得懂。搞实验的人都知道，结论靠的是那把看不见的尺子：标准、重复、误差、参照。要是连尺子都不可信，楼就不是从楼顶塌，而是从地基开始陷。

“你们让我来，是因为我和他们属于同一类人。”他说。

“这是一层原因。”常伟思说。

灯光被调暗，屏幕上切出一张关系图。中央是申玉菲的名字，向外连着杨冬、几位国外学者、几个未核实代号，再往外是会所、沙龙、论坛、加密频道，以及一个被圈起来的词：游戏。

“申玉菲在找你。”常伟思说，“时间比我们正式接触你更早。她看过你的项目资料，打听过你的作息，也知道你和杨冬的旧关系。她没有贸然接近，说明她在挑入口。你身上有她需要的条件。”

“什么条件？”

“名望，专业方向，性格，还有可信度。”分析员说，“你不是容易被煽动的人。你愿意进去，对他们来说说明很多事。”

“对你们来说也一样。”汪淼说。

史强笑了一下：“这话算你说对了。”

常伟思没有回避：“对。你进得去，我们也能多看见一层。但这件事上，我们和你的目的并不冲突。你要找杨冬为什么会走到那一步，我们也要找。”

汪淼问：“你们凭什么认定这不是普通刑案，不是单纯心理问题，也不是某个邪教在筛选脆弱人群？”

“凭三件东西。”常伟思竖起手指，“第一，全球同步。不同国家、不同机构、不同文化背景，在短时间内出现同构案例，传统模仿效应解释不了。第二，集中打在基础科学前沿。应用技术、工程开发、产业实验室几乎没受影响，刀口很准。第三，幸存者提供的碎片信息里，有些内容在彼此隔绝的情况下反复出现：‘射手’、‘农场’、‘尺子弯了’、‘实验在嘲笑你’。这些人互不认识，有的连语言都不通。”

“有没有材料源头？”

“有几处入口，没一处是源头。”分析员说，“沙龙只是表层，邮件只是一种投送方式，那个游戏目前看更像筛子。能进去的，看见更多；看见更多的人，出事更快。”

汪淼抬头望向电子地图。红点沉默地亮着。他忽然想到地面上的北京，此刻大概接近中午，红绿灯照常切换，写字楼电梯照常上上下下，小贩把油条从锅里捞出来，孩子背着书包过街。表面世界平静正常，可在这座城市脚下，却有人把全球最聪明的一批人做成了一份名单。

“你们说这是战争。”他低声道。

“对。”常伟思平静回答，“只是还没到街上。”

史强说：“汪教授，你们这帮搞理论的，一个个跟宝贝似的，现在有人专挑宝贝下手。你说这是为啥？”

汪淼想了想：“如果有人想让整个人类失去对世界的解释能力，先打掉最前沿的理论，是最快的。”

没人接话，但谁都知道这话的分量。

常伟思点头：“所以我们需要你。你去看，去听，去分辨。凡是能验证的，立刻验证；不能验证的，也记下来。你只做你擅长的事。”

“如果他们让我看一些东西，或者参与讨论，内容本身可能有风险呢？”

“你有权中断。”常伟思说，“你不是我们的兵。联络方式已经给你了，任何时候都可以撤出来。”

史强补了一句：“撤不出来的时候，就打那个频段。你只管说话，剩下的我们来。”

中年分析员又递来一份薄册子：“这里面是你可以问、也适合问的问题。别太像审讯，也别表现得过分好奇。你第一次进去，只是被观察的对象。”

汪淼翻开，第一页就是杨冬的名字，下面列着她四次接触申玉菲后的时间变化，冷得像体温计刻度。他看了很久，合上册子。

“她们最后一次见面在哪儿？”

“城西一处私人会所。”常伟思说，“和你今天下午去的是同一处。”

汪淼眉心轻轻一动。线路从一开始就铺好了，杨冬走过，现在轮到他。

“那你们怎么判断申玉菲这一层，不是最里面？”

“因为她太容易被看见了。”分析员说，“真正的核心人物，不会频繁出面拉人，不会让名字在这么多接触链里反复出现。她更像筛子前面的引导者。”

H001在汪淼脑子里终于从散雾聚成轮廓。科学边界不是单一俱乐部，也不是满街贴标语的组织。它有门，有台阶，有外圈，有筛选。申玉菲站在门口，挑人，把一些人领进去。进去的人里，有些继续往下走，有些先倒了。

“我明白了。”汪淼说，“你们要我进去，不是为了立刻抓住谁，是为了确认门后面到底有几层，每一层在干什么。”

“差不多。”常伟思说。

“我还有一个条件。我的家人不能被动卷进去。无论你们安排什么监控和保护，都不要贴到他们能察觉的程度。我要他们照常生活。”

常伟思点头：“可以。”

屏幕上又滚动起部分国外机构的内部通报：暂停合作、数据复核延期、理论组重组、项目方向调整。没有哪个国家宣布灾难，所有措辞都很克制，像在修补局部故障。可这些故障连到一起，已经不是补丁能盖住的了。

“为什么新闻上几乎没有？”汪淼问。

分析员说：“因为单个看都像个案。有人猝死，有人抑郁，有人坠楼，有人服药过量。每个国家都能自己解释。只有把资料合起来，图案才出来。”

“也有人不愿让它出来。”常伟思补了一句。

汪淼没再追问。他站起来时，腿有些发僵。史强把薄册子、联络纸和一只新手机一并塞给他。

“走吧，先去找丁仪。”

“现在？”

“现在。你要验证，就别拖到下午。申玉菲那边三点，这边还有空。”

汪淼把东西收进包里，跟着史强往外走。厚门在身后合拢，会议室的灯光很快消失。

电梯上行时，他盯着不断变化的数字。每跳一格，都更清楚地想起那张地图。红点铺在世界上，城市却仍然安静。地面上的树还在风里动，公交车照样进站，早餐铺子的油烟照样升起。温暖还在，可这份温暖下面，已经有人开始把理性本身从人类手里往外抽。

电梯门打开，白天的光一下涌进来。院子里树影很淡，远处传来汽车喇叭，空气带着初夏的热气和草地修剪后的青涩味。世界看上去一切如常。

史强走到车边，拉开车门，回头看他：“汪教授，最后问你一遍，后悔还来得及。”

汪淼站在阳光里，手按在车门边缘，指节绷紧了一瞬，又慢慢松开。

“去找丁仪。”

史强点头，上车，发动。

黑色轿车驶出院门时，岗哨抬杆，动作标准利落。外面的北京正安静运转，街边广告牌反着太阳光，行人提着公文包穿过人行道，路口电子屏在播午间新闻。新闻里没有那些名字，没有那些遗书，也没有那张被红点扎满的世界地图。

车汇入车流后，汪淼从包里摸出那只新手机。

屏幕亮起的一刻，上面已经静静躺着一条新信息，发信人没有名字，只有一串空白号码。

内容很短。

“丁先生今天心情不好，别提杨冬。”
