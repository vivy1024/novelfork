## 📝 归档原则

文档归档的条件：

1. **内容已过时** - 描述的功能已实现或计划已变更
2. **已被替代** - 有更新、更完整的文档替代
3. **历史参考** - 仅作为历史记录保留

归档文档的处理：

- ✅ 保留原文件（不删除）
- ✅ 移动到 `archive/` 目录
- ✅ 在文档首页标注「已归档」状态
- ✅ 提供替代文档链接
