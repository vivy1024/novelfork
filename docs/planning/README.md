## 📅 里程碑

### 短期（1-2 周）

**v0.0.1 发布准备**：
1. 补充 `data-testid` 属性
2. 提高测试通过率到 80%+
3. 添加功能性测试
4. 修复已知 bug
5. 创建 GitHub Release

### 中期（1-2 个月）

**v0.1.0 - 测试与性能**：
1. 单元测试覆盖率 80%+
2. E2E 测试覆盖核心流程
3. 性能优化（虚拟滚动、上下文管理）
4. 基础用户体验优化

**v0.2.0 - 用户体验**：
1. 主题切换（Light/Dark/Auto）
2. 快捷键支持
3. 国际化支持（i18n）
4. 加载状态优化
5. 错误提示优化

### 长期（3-6 个月）

**v1.0.0 - 正式版**：
1. Phase 5 功能（交互叙事、局部干预）
2. 协作功能（多人实时协作）
3. 云同步
4. 插件系统
5. 完整文档和教程
