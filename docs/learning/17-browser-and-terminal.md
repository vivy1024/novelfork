# 浏览器与终端

NovelFork 内置浏览器和终端工具，让 Agent 能够与网页交互、截取页面截图，以及管理长期运行的进程。

## Browser 工具

Browser 工具使用你系统中已安装的 Chrome 或 Edge 浏览器，以 headless（无界面）模式运行。无需额外安装任何软件。

### 支持的操作

- **launch**：启动浏览器并打开指定 URL
- **navigate**：导航到新页面
- **click**：点击页面上的元素
- **fill**：在输入框中填写文本
- **screenshot**：截取当前页面截图
- **scroll**：滚动页面
- **evaluate**：执行 JavaScript 代码
- **dom**：获取页面 DOM 结构

### 典型用途

- 前端 QA 验证：截图确认 UI 渲染是否正确
- 表单测试：模拟用户填写和提交
- 页面内容提取：获取动态渲染的页面数据

## Terminal 工具

Terminal 工具用于创建和管理持久化的终端进程，适合需要长期运行的任务。

### 支持的操作

- **create**：创建一个新的终端进程（如启动 dev server）
- **write**：向终端发送命令或输入
- **read**：读取终端的最新输出
- **list**：列出所有活跃的终端进程

### 典型用途

- 启动开发服务器（`bun run dev`）并保持运行
- 监控构建输出或日志
- 在后台运行测试套件
