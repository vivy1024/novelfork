# 外观设置

> 主题切换、字体配置、终端外观。

## 核心概念

**主题**：亮色 / 暗色 / 跟随系统。暗色模式下支持 OLED 纯黑背景。

**字体**：编辑器字体和大小可自定义。支持系统字体和 Web 字体。

**终端外观**：终端面板的字体、字号、配色方案独立配置。

## 推荐使用流程

1. 设置 → 外观 → 选择主题
2. 调整编辑器字体和大小
3. 如需 OLED 模式，在暗色主题下开启

## 最佳实践

- 长时间写作建议暗色模式减少眼疲劳
- 编辑器字号建议 14-16px，兼顾阅读舒适和信息密度
- 中文写作推荐等宽字体或专用中文字体

## 常见坑

- **字体不生效** → 系统未安装该字体，使用系统已有字体或 Web 字体
- **OLED 模式下文字看不清** → 部分低对比度文字在纯黑背景下不明显，调高对比度

## Agent 查阅提示

- 外观配置存储在 settings store 的 appearance 字段
- 主题切换通过 CSS 变量实现，无需重载页面
- 编辑器字体通过 `--editor-font-family` CSS 变量注入
