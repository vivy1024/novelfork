# 外观与个性化

> 在「设置 → 外观与界面」中自定义应用的视觉风格和交互行为。

---

## 主题模式

三种主题可选：

| 模式 | 说明 |
|------|------|
| 浅色 | 白色背景，适合白天使用 |
| 深色 | 深灰背景，减少眼睛疲劳 |
| 跟随系统 | 自动匹配操作系统的深浅模式 |

### OLED 纯黑

深色模式下使用纯黑背景（#000000），适合 AMOLED 屏幕：
- 节省电量（OLED 黑色像素不发光）
- 减少拖影
- 更沉浸的写作体验

---

## 字体设置

### 界面字体大小

滑块调节（12-20px），影响所有界面文字。推荐：
- 14px：标准（默认）
- 16px：大屏或视力不佳
- 12px：小屏幕最大化内容区域

### 字体族

- 系统默认：跟随操作系统
- Segoe UI：Windows 原生风格
- Helvetica：macOS 风格
- 苹方/微软雅黑：中文优化

## 终端设置

终端用于 AI Agent 执行命令时的输出展示。

### 终端主题

- 自动（跟随系统）：与主题模式一致
- 深色：始终深色背景
- 浅色：始终浅色背景

### 终端字体大小

滑块调节（8-32px），默认 14px。影响终端输出的可读性。

---

## 显示选项

### 高级动画

为消息和卡片启用 blur-in 动画效果。关闭可提升低配设备的流畅度。

### 自动换行

分别控制三种内容的自动换行：
- **Markdown**：长段落是否自动换行
- **代码**：代码块是否自动换行（关闭则水平滚动）
- **Diff**：差异对比是否自动换行

---

## 语言

- 简体中文（默认）
- English

切换后界面文字立即更新。AI 回复语言由「AI 代理 → 要求使用用户语言回复」控制。

---

## 个人资料

在「设置 → 个人资料」中管理：

### 头像

- 点击「上传头像」选择图片文件
- 支持 JPG/PNG/GIF，最大 2MB
- 头像显示在会话界面和设置页面

### Git 配置

- **Git 用户名**：用于 Git 提交的作者名
- **Git 邮箱**：用于 Git 提交的邮箱

这些信息用于 NovelFork 的版本管理功能（Checkpoint/Rewind）。

---

## 最佳实践

- 长时间写作用深色模式 + OLED 纯黑，减少眼睛疲劳
- 终端字体设大一点（16-18px），方便查看 Agent 执行日志
- 代码自动换行建议关闭（保持代码格式完整）
- Markdown 自动换行建议开启（长段落更易阅读）
