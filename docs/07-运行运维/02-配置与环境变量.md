## 当前配置入口

- Studio 运行端口：`NOVELFORK_STUDIO_PORT`，默认 `4567`。
- 开发运行：`bun run main.ts --root=. --port=4567`。
- 编译运行：`pnpm bun:compile` 后运行 `dist/novelfork.exe --root=. --port=4567`。

## 模型配置

模型供应商配置以 Studio 设置页和 runtime provider store 为当前主线。旧 CLI 配置示例仅可作为过渡参考，不能作为唯一当前入口。

## 安全规则

- API Key、Token、密码不得写入仓库。
- 文档示例只能使用脱敏占位值。
- 配置变更必须说明是用户级、项目级还是运行时派生。
