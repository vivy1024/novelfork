# InkOS 未实现功能路线图

**更新日期**: 2026-04-17  
**产品定位**: AI 长篇小说生产工作台（PWA + IDE）  
**参考**: 类似 narrafork 的 PWA 应用，具有 IDE 功能

---

## 一、核心架构升级 🚧

### 1.1 PWA 应用形态
- ⏳ Service Worker 离线缓存（参考 `vite-plugin-pwa`）
- ⏳ 安装到桌面（替代 Node SEA 的重量级方案）
- ⏳ 渐进式 Web 应用 manifest.json
- ⏳ 离线模式下的基础编辑能力

### 1.2 IDE 三栏布局
- ⏳ 左侧项目导航（按故事结构组织，非文件树）
  - 章节列表（带状态标识：✓ 已批准 / ● 草稿 / ○ 未写）
  - 设定集分簇（人物/势力/规则/地理）
  - 结构卡片（黄金三章/救猫咪节拍）
- ⏳ 中央多标签编辑器
  - 富文本编辑器（章节）
  - Markdown 编辑器（设定文件）
  - 结构化视图（状态文件）
  - Ctrl+S 保存 + dirty 标记
- ⏳ 底部动态上下文面板（Lorebook）
  - Tab 1: 人物与势力（实体识别 + 关键参数）
  - Tab 2: 伏笔追踪（倒计时 UI）
  - Tab 3: 节奏波形图（3+1 分析）
  - Tab 4: 世界观词条（维度浏览）

## 二、世界观引擎（核心竞争力）🔥

### 2.1 SQLite 知识图谱
- ⏳ 扩展 `memory.db` 为通用世界观容器
- ⏳ 9 个可插拔维度：
  - ✅ 人物状态库（已有 facts 表）
  - ✅ 伏笔追踪器（已有 hooks 表）
  - ⏳ 物品账本（materials 表）
  - ⏳ 时间线引擎（timeline_events 表）
  - ⏳ 阵营博弈网（factions 表）
  - ⏳ 物理/数值约束（world_rules 表）
  - ⏳ 经济系统（world_rules 表）
  - ⏳ 地理志（geography 表）
  - ⏳ 角色采访台（独立会话模块）

### 2.2 Lorebook 上下文注入
- ⏳ 关键词触发机制（替代全量 Markdown 注入）
- ⏳ NER 实体抽取（从 prompt 提取人名/地名/术语）
- ⏳ RAG 精确检索（按 token 预算裁剪）
- ⏳ Token 消耗降低 60%-80%

### 2.3 设定防膨胀机制
- ⏳ 冗余设定检测（未使用的废案警告）
- ⏳ Sanderson 三定律校验（代价/限制/一致性）
- ⏳ 设定健康度监控面板

## 三、网文行业标准集成 📚

### 3.1 节奏控制系统
- ⏳ 3+1 节奏公式（3 章紧张 + 1 章过渡）
- ⏳ 节奏波形图可视化（张弛度曲线）
- ⏳ 自动提醒（连续高压章节警告）
- ⏳ 10 章小高潮 / 100-150 章大高潮标记

### 3.2 黄金三章检测器
- ⏳ 前 3 章节奏/冲突/悬念密度分析
- ⏳ 留人点检测（6000 字内的关键卡点）
- ⏳ 毒点预警（7 条确定性规则）

### 3.3 伏笔倒计时系统
- ⏳ 伏笔埋设时自动设置回收章节
- ⏳ 倒计时 UI（还剩 N 章需回收）
- ⏳ 过期伏笔警告（超过预期未回收）

### 3.4 毒点检测器
- ⏳ 7 条网文毒点规则：
  - 主角憋屈无后手
  - 设定崩坏
  - 金手指失效/被夺
  - 感情强扭
  - 水字数
  - 降智
  - 断崖节奏

## 四、AI Agent 工具链 🤖

### 4.1 内嵌终端
- ⏳ 基于 `@xterm/xterm` + `node-pty`
- ⏳ AI Agent 可执行 shell 命令
  - `grep`/`rg` 搜索参考资料
  - `curl` 抓取学术 API
  - `python` 跑数值计算脚本
- ⏳ 用户可直接操作文件/运行脚本

### 4.2 数据源接入
- ⏳ 本地文件解析（PDF/EPUB/TXT/CSV/YAML）
- ⏳ 网页抓取（`puppeteer-core` + `readability`）
  - 论文摘要
  - 百科词条
  - 排行数据
- ⏳ MCP 工具服务器扩展

### 4.3 计算沙箱
- ⏳ Agent 生成并执行 Python/JS 脚本
- ⏳ 数值一致性验证（LEVSS 参数推导）
- ⏳ 经济模型平衡性计算
- ⏳ 执行结果自动注入知识库

## 技术栈参考（来自 narrafork）

| 技术 | 用途 | InkOS 应用 |
|------|------|-----------|
| `vite-plugin-pwa` | PWA 离线缓存 | 安装到桌面，替代 Node SEA |
| `@xyflow/react` + `dagre` | DAG 流程图 | Pipeline 可视化 |
| `@dnd-kit/core` | 拖拽排序 | 章节/大纲拖拽 |
| `@xterm/xterm` + `node-pty` | 内嵌终端 | AI Agent 执行命令 |
| `puppeteer-core` + `readability` | 网页抓取 | 雷达数据采集 |
| `drizzle-orm` + `better-sqlite3` | 类型安全 ORM | 世界观引擎 5 张新表 |
