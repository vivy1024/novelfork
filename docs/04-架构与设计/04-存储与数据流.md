## 当前定位

存储层包括本地文件、SQLite、runtime store 和由 Studio API 暴露的数据契约。

## 数据流原则

- UI 不直接绕过 API 写业务数据。
- 当前进程临时状态必须明确标注为 `process-memory`，不保证重启后保留。
- 迁移、repair、schema 兼容规则必须写入开发者指南和 API 契约。
