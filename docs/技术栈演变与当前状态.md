## 一、项目概述

**InkOS** 是一个 AI 长篇小说生产工作台，定位为 **Scrivener 项目管理 + Cursor AI 能力 + 网文行业领域智能**。

### 核心特性
- 10 Agent 写作管线（Radar → Planner → Composer → Architect → Writer → Observer → Reflector → Normalizer → Auditor → Reviser）
- SQLite 时序记忆系统（facts, summaries, hooks, world_entries）
- 33 维连续性审计 + 去 AI 味
- 多模型路由 + 守护进程
- Web 工作台 + CLI 工具

## 二、技术栈演变历史

### 阶段 1：CLI 工具时代（v0.x - v1.0）

**技术栈**：
- Node.js 22+ (内置 SQLite)
- TypeScript 5.8
- 纯命令行界面

**特点**：
- 专注核心写作管线
- 本地文件系统存储
- 适合开发者使用

### 阶段 2：Tauri 桌面端尝试（v1.0 - v1.1）

**技术栈**：
- Tauri 2.0 (Rust + WebView)
- React 19
- Vite 6
- Hono 4.7 (HTTP Server)

**提交记录**：
- `056e38c` - feat(desktop): add tiptap dependencies and successfully build Tauri desktop
- `b6b5204` - feat(studio): complete Tauri bridge — all features functional offline
- `f61ea8d` - feat(studio): local daemon auto-writer + desktop doctor diagnostics

**问题**：

**决策**：
