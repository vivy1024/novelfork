## 资源树对象

| 对象 | 说明 | 当前状态 |
|---|---|---|
| 作品 | 一本小说的根对象 | current |
| 卷 | 章节分组 | planning |
| 已有章节 | 正式章节正文 | current |
| 生成章节 | AI 候选稿或生成结果 | current |
| 草稿 | 未合并正文的草稿 | planning |
| 大纲 | 作品和章节规划 | planning |
| 故事经纬 | 人物、地点、势力、物品、伏笔、世界规则等资料 | planning |
| 素材 | 外部参考、片段和设定 | planning |
| 发布报告 | 发布前检查结果 | planning |

## 维护原则

资源管理器不是通用 IDE 文件树，而是小说创作资源树。
