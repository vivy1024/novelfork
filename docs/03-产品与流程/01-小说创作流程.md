## 主流程

```text
作品 / 卷 / 章节
  -> 章节编辑
  -> AI 生成候选稿或草稿
  -> 用户确认
  -> 合并到正式正文或保留为草稿
  -> 写作工具、故事经纬、审校与发布检查辅助迭代
```

## 核心原则

- 正文是用户确认后的结果。
- AI 不得直接覆盖正式正文。
- 故事经纬是辅助创作的资料与连续性入口。
- 候选稿、草稿和正式章节必须在资源管理器中清楚区分。
