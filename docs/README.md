## 文档状态说明

| 文档类型 | 状态 | 说明 |
|---|---|---|
| `current` | ✅ 当前有效 | 当前事实或当前推荐使用方式 |
| `planning` | 📋 规划中 | 已批准或正在执行的计划，不能冒充已完成 |
| `reference` | 📚 参考资料 | 外部资料或调研，不代表当前实现承诺 |
| `archived` | 🗄️ 历史归档 | 历史记录，仅供追溯 |
| `deprecated` | ⚠️ 已废弃 | 已被替代，必须给替代入口 |

## 维护规则

- 每篇文档必须有标准 header 和文档类型。
- 根 README 的当前事实入口只能指向 `current` 或明确标注的 `planning` 文档。
- 参考资料和历史归档不得冒充当前产品路线。
- 修改文档后运行 `bun run docs:verify`。
