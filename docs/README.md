## 文档类型约定

| 类型 | 含义 |
|------|------|
| `current` | 当前事实描述，与代码同步 |
| `planning` | 规划中，尚未实现 |
| `reference` | 参考资料，非本项目产出 |
| `archived` | 已归档，仅保留历史价值 |
| `deprecated` | 已废弃，不应参考 |
