## 当前定位

## 数据库

- 位置：`~/.novelfork/novelfork.db`
- 驱动：better-sqlite3（Node）/ bun:sqlite（Bun）
- ORM fallback：drizzle-orm
- WAL 模式：启动时启用 `PRAGMA journal_mode=WAL` + `synchronous=NORMAL`
