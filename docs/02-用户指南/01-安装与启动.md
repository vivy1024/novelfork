## 当前推荐路径

### 开发运行

```bash
bun run main.ts --root=. --port=4567
```

启动后访问：`http://localhost:4567`。

### 编译运行

```bash
pnpm bun:compile
./dist/novelfork.exe --root=. --port=4567
```

编译产物使用 embedded assets 启动本地 Studio。
