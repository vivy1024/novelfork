## 快速开始

1. 安装依赖：`bun install`
2. 启动工作台：`bun run dev`（在 `packages/studio` 目录）
3. 打开浏览器：`http://localhost:4567`
4. 配置 AI 模型：设置 → AI 供应商 → 添加供应商
5. 创建第一本书：仪表盘 → 创建新书
6. 开始写作！

---

## 工作台布局

打开一本书后，工作台分为三栏：

```
┌──────────┬──────────────┬─────────────┐
│ 左侧      │ 中间          │ 右侧         │
│ 资源管理器 │ 编辑器        │ 驾驶舱/经纬/写作│
└──────────┴──────────────┴─────────────┘
```

### 左侧：资源管理器

树形结构展示当前书籍的所有资源：

- **已有章节** — 正式定稿的章节（可创建、编辑、保存、删除）
- **生成章节** — AI 生成的候选稿（可查看、合并、替换、另存草稿、放弃）
- **草稿** — 暂存的写作片段
- **大纲** — 卷大纲编辑器
- **经纬/资料库** — 人物、地点、势力、物品、伏笔、世界规则
- **故事文件** — pending_hooks、章节摘要等
- **真相文件** — book_rules、current_focus 等
- **发布报告** — 发布就绪检查结果

### 中间：编辑器

- TipTap 富文本编辑器，支持 Markdown
- 章节标题、字数、保存状态显示
- 候选稿编辑器：查看 AI 生成内容 + 合并/替换/另存/放弃

### 右侧：面板（三个 Tab）

**驾驶舱（默认）**：

- **总览** — 日更进度、当前焦点、最近章节摘要、风险提示
- **伏笔** — 聚合所有来源的伏笔列表
- **设定** — 世界规则 + 资源账本
- **AI** — 模型状态 + 最近候选稿

**经纬** — 人物/事件/设定/摘要管理

**写作**：
- AI 动作按钮：生成下一章、续写段落、审校、改写、去 AI 味、连续性检查
- Agent 写作入口：输入意图，Agent 自动探索→规划→写作→审计
- 写作模式：续写/扩写/补写/对话生成/多版本/大纲分支
- 写作工具：节奏分析/对话分析/钩子生成/日更进度/健康仪表盘

## 写作流程

### 手动写作

1. 点击顶部「新建章节」或资源树的空态按钮
2. 在编辑器中写正文
3. 点击「保存」

### AI 辅助写作

**使用 AI 动作按钮**：

1. 选中章节
2. 右侧面板 → 写作 Tab → 点击「生成下一章」或「续写当前段落」
3. AI 结果进入候选区
4. 在候选区查看、编辑
5. 点击「合并」确认写入正式章节

**使用写作模式**：

1. 右侧面板 → 写作 Tab → 选择写作模式（续写/扩写/对话等）
2. 点击「生成」
3. 查看结果 → 选择目标（候选稿/草稿）→ 确认

**使用 Agent 写作**：

1. 右侧面板 → 写作 Tab → Agent 写作输入框
2. 输入意图：「写下一章，回收玉佩伏笔，保持林月冷峻性格」
3. 点击「启动 Agent 写作」
4. Agent 自动探索当前状态 → 规划大纲 → 生成正文 → 审计质量
5. 结果在 ChatWindow 中查看

---

## 管理世界设定（经纬）

1. 右侧面板 → 经纬 Tab
2. 添加人物、事件、设定
3. 每条记录可关联章节，支持三种可见性
4. AI 写作时会自动注入相关经纬信息

---

## 发布前检查

1. 顶部「发布就绪」按钮
2. 选择目标平台（起点/晋江/番茄等）
3. 运行检查：敏感词、AI 比例、格式规范、连续性
4. 查看报告，按建议修改

---

## 导出

1. 顶部「导出」按钮
2. 选择格式（Markdown / TXT）：全书或单章范围
3. 下载文件

## Agent 会话

ChatWindow 提供 5 种 Agent 角色：

| 角色 | 适合场景 |
|------|---------|
| **Writer**（写作） | 日常续写、章节生成 |
| **Planner**（规划） | 制定大纲、情节点设计 |
| **Auditor**（审计） | 连续性检查、质量审校 |
| **Architect**（架构） | 世界观构建、规则体系 |
| **Explorer**（探索） | 只读分析状态、给出建议 |
