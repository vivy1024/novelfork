## 当前主线

模型配置以 Studio 设置页和 runtime provider store 为当前主线。用户可在 AI 供应商相关入口维护平台集成和 API key 接入。

## 配置原则

- 未配置模型时，本地书籍创建、章节编辑和故事经纬维护仍可继续。
- 触发 AI 续写、评点、深度检测或工作台 Agent 时，系统应提示补充模型配置。
- 凭据缺失或模型不可用时，系统必须返回真实错误或 AI gate，不生成假正文。
