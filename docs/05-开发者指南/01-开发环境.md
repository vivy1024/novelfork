## 常用命令

```bash
pnpm install
bun run main.ts --root=. --port=4567
pnpm -r typecheck
pnpm -r test
pnpm --dir packages/studio compile
bun run docs:verify
```

## 运行时

- 生产运行时：Bun
- 开发兼容：Node
- 存储层：better-sqlite3（Node）/ bun:sqlite（Bun），drizzle-orm fallback
- 数据库位置：`~/.novelfork/novelfork.db`
- 测试框架：Vitest
- 编译：`pnpm --dir packages/studio compile`（内部调用 `packages/studio/scripts/compile.ts`，bun compile 生成单文件 exe）

## 开发约束

- 文档改动必须通过 `bun run docs:verify`。
- UI 功能不得恢复 mock、fake、noop 假成功。
- 未接入能力必须显示未接入、只读或透明过渡。
- 小说领域代码必须且只能存在于 `packages/novel-plugin/` 中（插件化边界）。
